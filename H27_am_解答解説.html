大問1：複素数と複素積分

問1）$z^4 = \dfrac{\sqrt{3}}{4} - \dfrac{1}{4}j$ の解

右辺を極形式に変換： $$\left|\frac{\sqrt{3}}{4}-\frac{j}{4}\right| = \frac{1}{4}\sqrt{(\sqrt{3})^2+1^2} = \frac{1}{4}\cdot 2 = \frac{1}{2}$$ 偏角（$0\le\theta<2\pi$）：実部正・虚部負（第4象限）なので $$\arg = 2\pi - \arctan\!\left(\frac{1/4}{\sqrt{3}/4}\right) = 2\pi - \frac{\pi}{6} = \frac{11\pi}{6}$$ よって $z^4 = \dfrac{1}{2}e^{j\cdot 11\pi/6}$。
4乗根（$k=0,1,2,3$）： $$z_k = \left(\frac{1}{2}\right)^{1/4}\!\exp\!\left(j\cdot\frac{11\pi/6 + 2\pi k}{4}\right) = 2^{-1/4}\exp\!\left(j\cdot\frac{(11+12k)\pi}{24}\right)$$
$k$偏角 $\theta_k$$z_k$
0$11\pi/24$$2^{-1/4}e^{j\cdot 11\pi/24}$
1$23\pi/24$$2^{-1/4}e^{j\cdot 23\pi/24}$
2$35\pi/24$$2^{-1/4}e^{j\cdot 35\pi/24}$
3$47\pi/24$$2^{-1/4}e^{j\cdot 47\pi/24}$
4点は複素平面上で半径 $2^{-1/4}\approx0.841$ の円上に $90°$ 等間隔で配置される（起点 $\theta_0 = 11\pi/24 = 82.5°$）。

問2）複素積分（留数定理）

（ア）$\displaystyle\int_C\dfrac{1}{z-5}dz$，$C:|z|=3$
特異点 $z=5$：$|5|=5>3$ → $C$ の外部。コーシーの定理より被積分関数は $C$ 内で正則。
$$\int_{|z|=3}\frac{dz}{z-5} = 0$$
（イ）$\displaystyle\int_C\dfrac{1}{z-5}dz$，$C:|z|=6$
特異点 $z=5$：$|5|=5<6$ → $C$ の内部。留数 $= 1$。
$$\int_{|z|=6}\frac{dz}{z-5} = 2\pi j\cdot 1 = 2\pi j$$
（ウ）$\displaystyle\int_C\dfrac{1}{(z-5)^2}dz$，$C:|z|=6$
特異点 $z=5$（2位の極）は $C$ 内部。$\dfrac{1}{(z-5)^2}$ の留数： $$\text{Res}_{z=5}\frac{1}{(z-5)^2} = \lim_{z\to5}\frac{d}{dz}[(z-5)^2\cdot\frac{1}{(z-5)^2}] = \frac{d}{dz}(1)\bigg|_{z=5} = 0$$
$$\int_{|z|=6}\frac{dz}{(z-5)^2} = 2\pi j\cdot 0 = 0$$
（ウ）は $\dfrac{1}{(z-5)^2}$ の原始関数 $-\dfrac{1}{z-5}$ が $|z|=6$ 内で一価であることから直接 $0$ と判断することもできる（$n\ge2$ の極の周回積分はゼロ）。

大問2：デルタ関数のフーリエ級数とサンプリング定理

問1）インパルス列のフーリエ級数展開

$\delta_T(t) = \sum_{n=-\infty}^\infty \delta(t-nT_s)$ は周期 $T_s$ の周期関数。フーリエ級数： $$\delta_T(t) = \sum_{n=-\infty}^\infty c_n \exp\!\left(j\frac{2\pi n}{T_s}t\right)$$ フーリエ係数（$-T_s/2 \le t < T_s/2$ の1周期で積分）： $$c_n = \frac{1}{T_s}\int_{-T_s/2}^{T_s/2}\delta_T(t)\,e^{-j(2\pi n/T_s)t}\,dt$$ 区間内にあるデルタ関数は $t=0$ の $\delta(t)$ のみ： $$c_n = \frac{1}{T_s}\int_{-T_s/2}^{T_s/2}\delta(t)\,e^{-j(2\pi n/T_s)t}\,dt = \frac{1}{T_s}\cdot e^0 = \frac{1}{T_s}$$ （すべての $n$ に対して同じ値）
$$\delta_T(t) = \frac{1}{T_s}\sum_{n=-\infty}^\infty \exp\!\left(j\frac{2n\pi}{T_s}t\right) \qquad\blacksquare$$

問2）サンプリング後の関数 $f_s(t)$ のフーリエ変換

$f_s(t) = \displaystyle\sum_{n=-\infty}^\infty T_s f(t)\delta(t-nT_s) = T_s f(t)\cdot\delta_T(t)$ 問1）の結果を代入： $$f_s(t) = T_s f(t)\cdot\frac{1}{T_s}\sum_{n=-\infty}^\infty e^{j(2\pi n/T_s)t} = f(t)\sum_{n=-\infty}^\infty e^{j(2\pi n/T_s)t}$$ フーリエ変換： $$F_s(\omega) = \int_{-\infty}^\infty f_s(t)e^{-j\omega t}\,dt = \sum_{n=-\infty}^\infty\int_{-\infty}^\infty f(t)\,e^{-j(\omega - 2\pi n/T_s)t}\,dt$$
$$F_s(\omega) = \sum_{n=-\infty}^\infty F\!\left(\omega - \frac{2\pi n}{T_s}\right)$$ （$F(\omega)$ を角周波数 $2\pi/T_s$ 間隔で複製した周期関数）
これはサンプリング定理の基礎：$f(t)$ の最高周波数 $\omega_{\max}$ に対して $2\pi/T_s > 2\omega_{\max}$ が成り立てばエイリアシングが生じず，$F(\omega)$ を完全に復元できる（ナイキスト–シャノンのサンプリング定理）。

大問3：拡散方程式（熱方程式）

$$\frac{\partial^2}{\partial x^2}u(x,t) = \frac{1}{D}\frac{\partial}{\partial t}u(x,t) \qquad (D > 0)$$

問1）変数分離 → 2つの常微分方程式

$u(x,t) = X(x)T(t)$ を代入： $$X''(x)T(t) = \frac{1}{D}X(x)T'(t)$$ $X(x)T(t)$ で割ると： $$\frac{X''}{X} = \frac{T'}{DT}$$ 左辺は $x$ のみ，右辺は $t$ のみの関数 → 両辺は定数 $-\lambda^2$ に等しい（周期解のため負の実数）：
$$\boxed{X'' + \lambda^2 X = 0} \qquad \boxed{T' + D\lambda^2 T = 0}$$

問2）一般解

$X$ の方程式：$X'' + \lambda^2 X = 0$（周期解）→ $X = A\sin(\lambda x) + B\cos(\lambda x)$
$T$ の方程式：$T' + D\lambda^2 T = 0$ → $T = Ce^{-D\lambda^2 t}$
重ね合わせ： $$u(x,t) = \sum_\lambda \left[A_\lambda\sin(\lambda x) + B_\lambda\cos(\lambda x)\right]e^{-D\lambda^2 t}$$ （$A_\lambda$，$B_\lambda$ は境界・初期条件で決まる定数）

問3）境界値問題の解

境界条件 $u(0,t) = 0$： $A_\lambda\sin(0)+B_\lambda\cos(0) = B_\lambda = 0$ → $\cos$ 項消える。 境界条件 $u(L,t) = 0$： $A_\lambda\sin(\lambda L) = 0$ → $\lambda L = n\pi$ → $\lambda_n = \dfrac{n\pi}{L}$（$n = 1,2,3,\ldots$） 解の形： $$u(x,t) = \sum_{n=1}^\infty A_n \sin\!\left(\frac{n\pi x}{L}\right)e^{-D(n\pi/L)^2 t}$$
初期条件 $u(x,0) = f(x)$： $$f(x) = \sum_{n=1}^\infty A_n\sin\!\left(\frac{n\pi x}{L}\right)$$ 正弦フーリエ級数の直交性（$\int_0^L\sin(n\pi x/L)\sin(m\pi x/L)dx = (L/2)\delta_{nm}$）より： $$A_n = \frac{2}{L}\int_0^L f(x)\sin\!\left(\frac{n\pi x}{L}\right)dx$$
$$u(x,t) = \sum_{n=1}^\infty A_n \sin\!\left(\frac{n\pi x}{L}\right)\exp\!\left(-\frac{Dn^2\pi^2}{L^2}t\right)$$ $$A_n = \frac{2}{L}\int_0^L f(x)\sin\!\left(\frac{n\pi x}{L}\right)dx$$
物理的解釈：各モード $n$ の振幅は時定数 $\tau_n = L^2/(Dn^2\pi^2)$ で指数的に減衰する。高次モード（大きい $n$）ほど速く減衰し，十分時間が経つと基本モード（$n=1$）のみが残る。
