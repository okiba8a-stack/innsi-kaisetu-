大問1【電気数学】1次元非斉次ヘルムホルツ方程式

方程式：$\dfrac{d^2}{dx^2}\phi(x) + k^2\phi(x) = -\delta(x - x_0)$，ただし $k = k_r - jk_i$（$k_r, k_i > 0$）

問1）斉次方程式の独立な解

斉次方程式 $\psi'' + k^2\psi = 0$ の特性方程式：$\lambda^2 + k^2 = 0 \Rightarrow \lambda = \pm jk$
$$\psi_1(x) = e^{jkx}, \quad \psi_2(x) = e^{-jkx}$$

問2）境界条件による係数決定

$k = k_r - jk_i$ より $jk = k_i + jk_r$ なので：

$|e^{jkx}| = e^{k_i x}$：$x \to -\infty$ で $\to 0$，$x \to +\infty$ で $\to \infty$
$|e^{-jkx}| = e^{-k_i x}$：$x \to +\infty$ で $\to 0$，$x \to -\infty$ で $\to \infty$

a) 有界条件

$x \to -\infty$ で有界 $\Rightarrow$ $x < x_0$ において $e^{-jkx}$ が発散 $\Rightarrow$ $\boldsymbol{c_1 = 0}$
$x \to +\infty$ で有界 $\Rightarrow$ $x > x_0$ において $e^{jkx}$ が発散 $\Rightarrow$ $\boldsymbol{c_4 = 0}$

b) $x = x_0$ での連続条件

$$c_2 e^{jkx_0} = c_3 e^{-jkx_0}$$

c) 区間積分による跳び条件

式(1)を $[x_0 - \varepsilon, x_0 + \varepsilon]$ で積分し $\varepsilon \to 0$：
$\int k^2\phi\,dx \to 0$（$\phi$ は有界），$\int \delta(x-x_0)\,dx = 1$ より
$$\phi'(x_0^+) - \phi'(x_0^-) = -1$$ すなわち：$c_3(-jk)e^{-jkx_0} - c_2(jk)e^{jkx_0} = -1$

d) 係数の決定

b) より $c_2 e^{jkx_0} = c_3 e^{-jkx_0} \equiv A$ とおくと，c) は： $-jkA - jkA = -1 \Rightarrow A = \dfrac{1}{2jk}$
$$c_1 = 0,\quad c_4 = 0,\quad c_2 = \frac{e^{-jkx_0}}{2jk},\quad c_3 = \frac{e^{jkx_0}}{2jk}$$ $$\phi(x) = \begin{cases} \dfrac{e^{jk(x-x_0)}}{2jk} & (x < x_0) \\[8pt] \dfrac{e^{-jk(x-x_0)}}{2jk} & (x > x_0) \end{cases} = \frac{e^{-jk|x-x_0|}}{2jk}$$

問3）フーリエ変換による別解法

a) $\Phi(\beta)$ に関する方程式

フーリエ変換の微分則：$\mathcal{F}[\phi''] = (j\beta)^2\Phi(\beta) = -\beta^2\Phi(\beta)$
$\mathcal{F}[\delta(x-x_0)] = e^{-j\beta x_0}$
式(1)に適用：$-\beta^2\Phi(\beta) + k^2\Phi(\beta) = -e^{-j\beta x_0}$
$$(k^2 - \beta^2)\Phi(\beta) = -e^{-j\beta x_0} \quad \Rightarrow \quad \Phi(\beta) = \frac{e^{-j\beta x_0}}{\beta^2 - k^2}$$

b) 逆フーリエ変換

$\phi(x) = \dfrac{1}{2\pi}\int_{-\infty}^{\infty}\Phi(\beta)e^{j\beta x}\,d\beta = \dfrac{1}{2\pi}\int_{-\infty}^{\infty}\frac{e^{-j\beta x_0}}{\beta^2 - k^2}e^{j\beta x}\,d\beta$
$$\phi(x) = \frac{1}{2\pi}\int_{-\infty}^{\infty}\frac{e^{j\beta(x-x_0)}}{\beta^2 - k^2}\,d\beta \qquad \text{（示された）}$$

問4）ジョルダンの補助定理と留数による一致確認

a) $f(\beta) = e^{j\beta(x-x_0)}/(\beta^2 - k^2)$ の特異点・位数・留数

$\beta^2 - k^2 = (\beta - k)(\beta + k) = 0$ より特異点 $\beta = \pm k$（1位の極）
特異点位置（$k = k_r - jk_i$）位数留数
$\beta = k$下半平面（$\text{Im}(k) = -k_i < 0$）1$\dfrac{e^{jk(x-x_0)}}{2k}$
$\beta = -k$上半平面（$\text{Im}(-k) = k_i > 0$）1$\dfrac{e^{-jk(x-x_0)}}{-2k}$

b) 積分の評価

$x > x_0$ の場合：$x - x_0 > 0$ なので，$e^{j\beta(x-x_0)}$ は上半平面で $\to 0$。
上半平面で閉曲線（反時計回り）を組む。内部の極：$\beta = -k$（上半平面）。
$$\int = 2\pi j \cdot \text{Res}(f, -k) = 2\pi j \cdot \frac{e^{-jk(x-x_0)}}{-2k} = -\frac{\pi j}{k}e^{-jk(x-x_0)}$$ $x < x_0$ の場合：$x - x_0 < 0$ なので，下半平面で閉曲線（時計回り，$-2\pi j$）。
内部の極：$\beta = k$（下半平面）。 $$\int = -2\pi j \cdot \text{Res}(f, k) = -2\pi j \cdot \frac{e^{jk(x-x_0)}}{2k} = -\frac{\pi j}{k}e^{jk(x-x_0)}$$
$$\int_{-\infty}^{\infty}\frac{e^{j\beta(x-x_0)}}{\beta^2-k^2}d\beta = \begin{cases} -\dfrac{\pi j}{k}e^{jk(x-x_0)} & (x < x_0) \\[8pt] -\dfrac{\pi j}{k}e^{-jk(x-x_0)} & (x > x_0) \end{cases}$$

c) 解の一致確認

問3)b) より $\phi(x) = \dfrac{1}{2\pi}\times\left(-\dfrac{\pi j}{k}\right)e^{\mp jk(x-x_0)} = -\dfrac{j}{2k}e^{\mp jk(x-x_0)}$

問2)d) より $\phi(x) = \dfrac{e^{\pm jk(x-x_0)}}{2jk}$。ここで $\dfrac{1}{2jk} = \dfrac{-j}{2k}$ なので両解は完全に一致。
$$\frac{1}{2jk} = \frac{1}{2jk}\cdot\frac{-j}{-j} = \frac{-j}{2k} \quad \Rightarrow \quad \text{解は一致（示された）}$$

大問2【電気回路】電源回路と共振回路

問1）電源回路（最大電力伝送）

a) 負荷電力と有能電力

電流 $I = \dfrac{e}{R_g + R_L}$，負荷電力 $P = I^2 R_L$：
$$P = \frac{e^2 R_L}{(R_g + R_L)^2}$$ 最大化：$\dfrac{dP}{dR_L} = e^2\dfrac{R_g - R_L}{(R_g + R_L)^3} = 0 \Rightarrow R_L = R_g$（整合条件）
有能電力（最大電力）：$$P_{\max} = \frac{e^2}{4R_g}$$

b) 整合時の $R_L$，電圧 $V$

$$R_L = R_g, \quad V = \frac{e}{2}$$

問2）共振回路の周波数伝達特性

a) 直列 $L,C,r$ のインピーダンス

$$Z = r + j\!\left(\omega L - \frac{1}{\omega C}\right)$$

b) $|T|^2$ の意味（$T = 2v_2/e$）

ポート#1の電源有能電力：$P_{\rm av} = \dfrac{|e|^2}{4R_g}$（複素振幅表記 $\times \frac{1}{2}$ のピーク値で）
正確には：$P_{\rm av} = \dfrac{|e|^2}{8R_g}$（ピーク値），$P_2 = \dfrac{|v_2|^2}{2R_g}$
$$|T|^2 = \left|\frac{2v_2}{e}\right|^2 = \frac{4|v_2|^2}{|e|^2} = \frac{|v_2|^2/(2R_g)}{|e|^2/(8R_g)} = \frac{P_2}{P_{\rm av}}$$ すなわち $|T|^2 =$ 「ポート#2の負荷 $R_g$ で消費される電力」÷「電源の有能電力」

c) 直列共振時 $|T|$ の周波数特性（概略）

2ポート内部が直列 $LCr$（図2.4）のとき，全回路は $e - R_g - L - C - r - R_g$ の直列回路。
$v_2 = \dfrac{eR_g}{2R_g + Z}$，$T = \dfrac{2R_g}{2R_g + r + j(\omega L - 1/\omega C)}$
$\omega_0 = 1/\sqrt{LC}$ のとき $|T|$ は最大（バンドパス特性）

d) $|T|_{\max}$ と $\omega_0$

$|T| = \dfrac{2R_g}{\sqrt{(2R_g+r)^2+(\omega L - 1/\omega C)^2}}$ は虚部 $= 0$ で最大。
$$\omega_0 = \frac{1}{\sqrt{LC}}, \quad |T|_{\max} = \frac{2R_g}{2R_g + r}$$

e) $\Delta\omega$ の計算

$|T| = |T|_{\max}/\sqrt{2}$ となる条件：$(\omega L - 1/\omega C)^2 = (2R_g + r)^2$
$\omega_1,\omega_2$ はそれぞれ $\omega_1\omega_2 = \omega_0^2$（積公式）を満たし， $\omega_{1,2} = \dfrac{\pm(2R_g+r)+\sqrt{(2R_g+r)^2+4L/C}}{2L}$
$$\Delta\omega = \omega_2 - \omega_1 = \frac{2R_g + r}{L}$$

f) 負荷Q値・無負荷Q値と $|T|_{\max}$ の関係

$Q_L = \dfrac{\omega_0}{\Delta\omega} = \dfrac{\omega_0 L}{2R_g+r}$，$Q_0 = \dfrac{\omega_0 L}{r}$ より $\dfrac{Q_L}{Q_0} = \dfrac{r}{2R_g+r}$
$$|T|_{\max} = \frac{2R_g}{2R_g+r} = 1 - \frac{Q_L}{Q_0}$$ よって $\dfrac{Q_L}{Q_0} = 1 - |T|_{\max}$

g) 並列共振回路の $|T|$ 周波数特性（概略）

図2.5：並列 $LCr$ を分路（シャント）素子として使用。
共振時 $\omega = \omega_0$ で並列インピーダンス $\to \infty$（高インピーダンス）→ 信号が通過 → $|T|$ は最大。
$\omega \to 0, \infty$ では $C$ または $L$ が短絡的に働き $|T| \to 0$。
直列共振と同様に $\omega_0$ 付近で帯域通過。ただし選択特性（帯域幅・$Q$値）は回路定数に依存。

大問3【電磁気学】電気力線とガウスの法則

問1）電気力線の性質

a) 開始点と終端点

電気力線は正電荷から出発し，負電荷で終端する（または無限遠へ延びる）。 孤立した正電荷では無限遠へ，孤立した負電荷では無限遠から来る。

b) 電気力線の数

電荷 $Q\,[\text{C}]$ から出る電気力線の総本数は $Q/\varepsilon_0$ に比例する（ガウスの法則）。 電荷量が大きいほど電気力線は多い。

c) 電気力線の方向

電気力線の接線方向が各点での電界ベクトル $\mathbf{E}$ の方向を示す。 正電荷から負電荷への向きが電界の向き（正の試験電荷が受ける力の方向）。

d) $+q$ と $-q$ の2導線（図④）

全ての電気力線が $+q$ の導線から出て $-q$ の導線へ終端する。
電気力線は2本の導線を結ぶ曲線群として，両導線間を弧状に横切る対称分布を描く。 各導線に8本と定義したので，$+q$ から8本が出て全て $-q$ へ。

e) $+q$ と $+q$ の2導線（図⑤）

両導線とも正電荷なので，電気力線は各導線から外側へ放射状に広がる。
2導線間の中央では電界が打ち消し合うため，電気力線は中央付近を避ける（電界ゼロ点が存在）。
電気力線は中央より外側に広がり，無限遠（または仮想的な負電荷）へ向かう。

問2）電界ベクトルと電荷密度・電荷総量

$$\mathbf{E}(x,y,z) = \frac{1}{\varepsilon_0}\Bigl\{A_0(xz-2yz)\,\mathbf{i} + B_0(xy^2-2yz^2)\,\mathbf{j} + C_0(2x^2y-xyz)\,\mathbf{k}\Bigr\}$$

a) 関連する基本法則

ガウスの法則（電磁界第一方程式） $$\nabla\cdot\mathbf{E} = \frac{\rho}{\varepsilon_0}$$ 電界の発散がその点の電荷密度を与える。

b) 電荷密度 $\rho(x,y,z)$

$\rho = \varepsilon_0\nabla\cdot\mathbf{E} = \dfrac{\partial(\varepsilon_0 E_x)}{\partial x} + \dfrac{\partial(\varepsilon_0 E_y)}{\partial y} + \dfrac{\partial(\varepsilon_0 E_z)}{\partial z}$ $$\frac{\partial(\varepsilon_0 E_x)}{\partial x} = A_0\frac{\partial(xz - 2yz)}{\partial x} = A_0 z$$ $$\frac{\partial(\varepsilon_0 E_y)}{\partial y} = B_0\frac{\partial(xy^2 - 2yz^2)}{\partial y} = B_0(2xy - 2z^2)$$ $$\frac{\partial(\varepsilon_0 E_z)}{\partial z} = C_0\frac{\partial(2x^2y - xyz)}{\partial z} = C_0(-xy)$$
$$\rho(x,y,z) = A_0 z + B_0(2xy - 2z^2) + C_0(-xy) = A_0 z + (2B_0 - C_0)xy - 2B_0 z^2$$

c) 電荷総量 $Q$

立方体 $0 \le x,y,z \le a$ で体積積分： $$Q = \int_0^a\!\!\int_0^a\!\!\int_0^a \rho\,dx\,dy\,dz$$ 項ごとに積分： $$\int A_0 z\,dx\,dy\,dz = A_0\cdot a\cdot a\cdot\frac{a^2}{2} = \frac{A_0 a^4}{2}$$ $$\int (2B_0-C_0)xy\,dx\,dy\,dz = (2B_0-C_0)\cdot\frac{a^2}{2}\cdot\frac{a^2}{2}\cdot a = \frac{(2B_0-C_0)a^5}{4}$$ $$\int(-2B_0 z^2)\,dx\,dy\,dz = -2B_0\cdot a\cdot a\cdot\frac{a^3}{3} = -\frac{2B_0 a^5}{3}$$
$$Q = \frac{A_0 a^4}{2} + \frac{(2B_0-C_0)a^5}{4} - \frac{2B_0 a^5}{3}$$
