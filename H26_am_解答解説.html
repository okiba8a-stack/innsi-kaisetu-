大問1：2階非斉次線形微分方程式 $y'' + y' = R(x)$

問1）$R(x)=0$ の一般解 $y_0$

特性方程式：$\lambda^2 + \lambda = 0 \implies \lambda(\lambda+1) = 0 \implies \lambda = 0,\,-1$
$$y_0 = C_1 + C_2 e^{-x} \quad (C_1,C_2 \text{ は任意定数})$$

問2）$R(x)=x^2$ の多項式特殊解 $y_P$

右辺 $x^2$ に対して通常 $y_P = ax^2+bx+c$ を試みるが，$\lambda=0$ が特性根に含まれるため $y=c$ が斉次解に属する。次数を1上げて $y_P = ax^3+bx^2+cx$ と設定： $$y_P' = 3ax^2+2bx+c, \quad y_P'' = 6ax+2b$$ 代入： $$y_P''+y_P' = (6ax+2b)+(3ax^2+2bx+c) = 3ax^2+(6a+2b)x+(2b+c) = x^2$$
係数比較： $x^2$：$3a = 1 \Rightarrow a = \dfrac{1}{3}$
$x^1$：$6a+2b = 0 \Rightarrow b = -1$
$x^0$：$2b+c = 0 \Rightarrow c = 2$
$$y_P = \frac{1}{3}x^3 - x^2 + 2x$$

問3）$R(x)=x^2$ の一般解と初期値問題

一般解 $y = y_0 + y_P$： $$y = C_1 + C_2 e^{-x} + \frac{1}{3}x^3 - x^2 + 2x$$ 導関数：$y' = -C_2 e^{-x} + x^2 - 2x + 2$
初期条件 $y(0)=0$，$y'(0)=1$： $$y(0) = C_1 + C_2 = 0 \quad\cdots(1)$$ $$y'(0) = -C_2 + 2 = 1 \implies C_2 = 1 \quad\cdots(2)$$ (1)より $C_1 = -1$
$$\boxed{y = -1 + e^{-x} + \frac{1}{3}x^3 - x^2 + 2x}$$

大問2：複素関数 $f(z) = \dfrac{3+j}{z^2-(1+j)z-(2+j)}$

問1）特異点

分母 $z^2-(1+j)z-(2+j) = 0$ の根を求める。$z=-1$ を試すと： $$(-1)^2-(1+j)(-1)-(2+j) = 1+1+j-2-j = 0 \quad \checkmark$$ 因数分解：$z^2-(1+j)z-(2+j) = (z+1)(z-(2+j))$ 確認：$(z+1)(z-(2+j)) = z^2-(2+j)z+z-(2+j) = z^2-(1+j)z-(2+j)$ ✓
特異点（1位の極）：$z = -1$ および $z = 2+j$

問2）点 $P_3$ の複素数 $z_3$

$z_2 - z_1 = (5+3j)-(1+j) = 4+2j$（$P_1$ から $P_2$ へのベクトル）
時計まわりに $\pi/3$ 回転 $\Leftrightarrow$ 乗算 $e^{-j\pi/3} = \cos(-60°)+j\sin(-60°) = \dfrac{1}{2}-j\dfrac{\sqrt{3}}{2}$： $$(4+2j)\!\cdot\!\left(\frac{1}{2}-j\frac{\sqrt{3}}{2}\right) = 4\cdot\frac{1}{2}+4\cdot\left(-j\frac{\sqrt{3}}{2}\right)+2j\cdot\frac{1}{2}+2j\cdot\left(-j\frac{\sqrt{3}}{2}\right)$$ $$= 2 - 2\sqrt{3}j + j + \sqrt{3} = (2+\sqrt{3}) + j(1-2\sqrt{3})$$ $$z_3 = z_1 + (2+\sqrt{3}) + j(1-2\sqrt{3}) = (1+j) + (2+\sqrt{3}) + j(1-2\sqrt{3})$$
$$z_3 = (3+\sqrt{3}) + j(2-2\sqrt{3}) = (3+\sqrt{3}) + 2(1-\sqrt{3})j$$ 数値近似：$\approx 4.73 - 1.46j$

問3）三角形 $C$（反時計まわり）の周回積分

三角形の頂点：$P_1=(1,1)$，$P_2=(5,3)$，$P_3\approx(4.73,-1.46)$。
特異点 $z=-1$（$= (-1,0)$）：三角形のx範囲（1〜5）の外 → 三角形の外部。
特異点 $z=2+j$（$= (2,1)$）：バリセントリック座標で確認（$s>0$, $t>0$, $s+t<1$が成立） → 三角形の内部。
留数（$z=2+j$ における）： $$\text{Res}_{z=2+j} f(z) = \frac{3+j}{(2+j)+1} = \frac{3+j}{3+j} = 1$$ 留数定理（反時計まわり）： $$\oint_C f(z)\,dz = 2\pi j \sum_{\text{内部}} \text{Res} = 2\pi j \cdot 1$$
$$\oint_C f(z)\,dz = 2\pi j$$

大問3：畳み込み積分

$f_1(t) = \begin{cases}1 & 0\le t\le 1\\ 0 & \text{otherwise}\end{cases}$， $f_2(t) = \begin{cases}\sin(2\pi t) & 0\le t\le 2\\ 0 & \text{otherwise}\end{cases}$

問1）畳み込み $f(t) = \int_0^\infty f_1(\tau)f_2(t-\tau)d\tau$

$f_1(\tau)\ne0$ の条件：$0\le\tau\le 1$。$f_2(t-\tau)\ne0$ の条件：$0\le t-\tau\le 2$，すなわち $t-2\le\tau\le t$。 有効積分区間：$\max(0,t-2)\le\tau\le\min(1,t)$
Region 1：$0\le t\le 1$（$\tau$ 区間：$[0,t]$）
$$f(t) = \int_0^t \sin(2\pi(t-\tau))\,d\tau$$ $u = t-\tau$ とおくと： $$= \int_0^t \sin(2\pi u)\,du = \left[-\frac{\cos(2\pi u)}{2\pi}\right]_0^t = \frac{1-\cos(2\pi t)}{2\pi}$$
Region 2：$1< t\le 2$（$\tau$ 区間：$[0,1]$）
$$f(t) = \int_0^1 \sin(2\pi(t-\tau))\,d\tau = \left[-\frac{\cos(2\pi(t-\tau))}{2\pi}\right]_0^1$$ $$= \frac{-\cos(2\pi t-2\pi)+\cos(2\pi t)}{2\pi} = \frac{-\cos(2\pi t)+\cos(2\pi t)}{2\pi} = 0$$ （$\sin(2\pi\tau)$のちょうど1周期分の積分のため）
Region 3：$2< t\le 3$（$\tau$ 区間：$[t-2,1]$）
$$f(t) = \int_{t-2}^1 \sin(2\pi(t-\tau))\,d\tau = \left[-\frac{\cos(2\pi(t-\tau))}{2\pi}\right]_{t-2}^1$$ $$= \frac{-\cos(2\pi(t-1))+\cos(2\pi(t-2))}{2\pi} = \frac{-\cos(2\pi t)+\cos(2\pi t)}{2\pi}$$ 待って：$\cos(2\pi(t-1)) = \cos(2\pi t - 2\pi) = \cos(2\pi t)$，$\cos(2\pi(t-2)) = \cos(2\pi t)$... 正確に計算： $$= \frac{1}{2\pi}\left[-\cos(2\pi(t-1)) + \cos(2\pi(t-2))\right] = \frac{-\cos(2\pi t) + \cos(2\pi t)}{2\pi}$$ 誤り：$t-\tau$ を正しく追跡する。$\tau=t-2$ → $t-\tau = 2$，$\tau=1$ → $t-\tau = t-1$ $$f(t) = \left[-\frac{\cos(2\pi u)}{2\pi}\right]_{t-1}^{2} = \frac{-\cos(4\pi)+\cos(2\pi(t-1))}{2\pi}$$ $$= \frac{-1+\cos(2\pi t-2\pi)}{2\pi} = \frac{\cos(2\pi t)-1}{2\pi}$$
$$f(t) = \begin{cases} 0 & (t < 0) \\ \dfrac{1-\cos(2\pi t)}{2\pi} & (0\le t\le 1) \\[6pt] 0 & (1 < t \le 2) \\[6pt] \dfrac{\cos(2\pi t)-1}{2\pi} & (2 < t \le 3) \\[6pt] 0 & (t > 3) \end{cases}$$

問2）$f(t)$ のグラフ（$-1\le t\le4$）

各区間の値：
$t$$f(t)$備考
$t < 0$$0$—
$t = 0$$0$$(1-\cos0)/(2\pi)=0$
$t = 0.5$$1/\pi \approx 0.318$$(1-\cos\pi)/(2\pi)=2/(2\pi)$
$t = 1$$0$$(1-\cos(2\pi))/(2\pi)=0$
$1 < t \le 2$$0$—
$t = 2.5$$-1/\pi \approx -0.318$$(\cos\pi-1)/(2\pi)=-1/\pi$
$t = 3$$0$$(\cos(6\pi)-1)/(2\pi)=0$
$t > 3$$0$—
グラフの特徴：
・$0\le t\le 1$ では $f(t) = (1-\cos(2\pi t))/(2\pi)$ — 正の半波形（$t=0.5$ で最大 $1/\pi$）
・$1 < t \le 2$ では $f(t) = 0$（平坦）
・$2 < t \le 3$ では $f(t) = (\cos(2\pi t)-1)/(2\pi)$ — 負の半波形（$t=2.5$ で最小 $-1/\pi$）
・それ以外はゼロ

大問4：ベクトル場の回転と面積分

$\boldsymbol{A} = -x^2y\,\boldsymbol{i} + xy^2\,\boldsymbol{j} + z^2\,\boldsymbol{k}$

問1）$\text{rot}\,\boldsymbol{A}$

$$\text{rot}\,\boldsymbol{A} = \nabla\times\boldsymbol{A} = \begin{vmatrix}\boldsymbol{i}&\boldsymbol{j}&\boldsymbol{k}\\\partial_x&\partial_y&\partial_z\\-x^2y&xy^2&z^2\end{vmatrix}$$ $\boldsymbol{i}$成分：$\partial_y(z^2) - \partial_z(xy^2) = 0 - 0 = 0$
$\boldsymbol{j}$成分：$-[\partial_x(z^2) - \partial_z(-x^2y)] = -[0 - 0] = 0$
$\boldsymbol{k}$成分：$\partial_x(xy^2) - \partial_y(-x^2y) = y^2 - (-x^2) = x^2+y^2$
$$\text{rot}\,\boldsymbol{A} = (x^2+y^2)\,\boldsymbol{k}$$

問2）半球面 $S$（$x^2+y^2+z^2=1$，$z\ge0$）の面積分

ストークスの定理を利用：$S$ の境界 $C$ は単位円 $x^2+y^2=1$，$z=0$（法線が $+z$ 方向のとき，$C$ は $xy$ 平面で反時計まわり）。 $$\iint_S \text{rot}\,\boldsymbol{A}\cdot\boldsymbol{n}\,dS = \oint_C \boldsymbol{A}\cdot d\boldsymbol{r}$$ $C$ 上：$x=\cos\theta$，$y=\sin\theta$，$z=0$，$d\boldsymbol{r} = (-\sin\theta, \cos\theta, 0)d\theta$
$\boldsymbol{A}|_{z=0} = (-\cos^2\theta\sin\theta,\;\cos\theta\sin^2\theta,\;0)$： $$\boldsymbol{A}\cdot d\boldsymbol{r} = \cos^2\theta\sin^2\theta + \cos^2\theta\sin^2\theta = 2\cos^2\theta\sin^2\theta = \frac{\sin^2(2\theta)}{2}$$ $$\oint_C \boldsymbol{A}\cdot d\boldsymbol{r} = \int_0^{2\pi}\frac{\sin^2(2\theta)}{2}\,d\theta = \frac{1}{2}\cdot\pi = \frac{\pi}{2}$$ （$\int_0^{2\pi}\sin^2(2\theta)d\theta = \pi$ を使用）
直接計算で確認：$\text{rot}\,\boldsymbol{A}\cdot\boldsymbol{n} = (x^2+y^2)\cdot z$（単位球なので $\boldsymbol{n}=(x,y,z)$，$z$成分）。
球座標（$x=\sin\phi\cos\theta$，$y=\sin\phi\sin\theta$，$z=\cos\phi$，$dS=\sin\phi\,d\phi\,d\theta$）： $$\iint_S (x^2+y^2)z\,dS = \int_0^{2\pi}\int_0^{\pi/2}\sin^2\phi\cos\phi\cdot\sin\phi\,d\phi\,d\theta = 2\pi\int_0^{\pi/2}\sin^3\phi\cos\phi\,d\phi$$ $= 2\pi\left[\dfrac{\sin^4\phi}{4}\right]_0^{\pi/2} = 2\pi\cdot\dfrac{1}{4} = \dfrac{\pi}{2}$ ✓
$$\iint_S \text{rot}\,\boldsymbol{A}\cdot\boldsymbol{n}\,dS = \frac{\pi}{2}$$
