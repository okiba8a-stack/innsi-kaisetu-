H21年度 午前 電気回路 解答解説

平成20年8月19日実施 ／ 専門科目：電気回路（午前）

大問1：最大電力伝送・F行列・整合回路

問1-1) ノートン → テブナン変換

電流源 \(I_S\)・コンダクタンス \(G_S\) のノートン等価回路を テブナン等価に変換。
\[R_S = \frac{1}{G_S}, \qquad V_S = \frac{I_S}{G_S}\]

問1-2) 最大電力伝送（\(Z_L\) を最適化）

テブナン等価（\(V_S,\,R_S\) は純抵抗）において、負荷 \(Z_L\) で消費される電力： \[P = \frac{1}{2}\frac{|V_S|^2\,\mathrm{Re}(Z_L)}{|R_S+Z_L|^2}\] \(Z_L=R_S^*=R_S\)（純抵抗）のとき最大（共役整合）。
\[Z_L = R_S = \frac{1}{G_S}, \qquad P_{\max} = \frac{V_S^2}{4R_S} = \frac{I_S^2}{4G_S}\] （\(V_S\) は振幅の場合 \(\frac{V_S^2}{8R_S}\)）

問1-3) 整合回路の F 行列（図1.3：シャント \(Z_1\) → 直列 \(Z_2\)）

シャント \(Z_1\) の F 行列：\(\begin{bmatrix}1&0\\1/Z_1&1\end{bmatrix}\)
直列 \(Z_2\) の F 行列：\(\begin{bmatrix}1&Z_2\\0&1\end{bmatrix}\)
カスケード（シャント先・直列後）： \[F = \begin{bmatrix}1&0\\1/Z_1&1\end{bmatrix}\begin{bmatrix}1&Z_2\\0&1\end{bmatrix} = \begin{bmatrix}1&Z_2\\1/Z_1&1+Z_2/Z_1\end{bmatrix}\]
\[\begin{bmatrix}A&B\\C&D\end{bmatrix} = \begin{bmatrix}1&Z_2\\1/Z_1&1+Z_2/Z_1\end{bmatrix}\]

問1-4) F 行列 \([A,B;C,D]\) による \(Z_{in}\) と \(Z_{out}\)

F 行列の定義 \(V_1=AV_2+BI_2,\ I_1=CV_2+DI_2\) と、負荷 \(R_L\) 条件 \(V_2=R_LI_2\)： \[Z_{in} = \frac{V_1}{I_1} = \frac{AR_L+B}{CR_L+D}\] 出力インピーダンス（\(V_S=0\) で短絡）：\(V_1=-R_SI_1\)，\(V_2/(-I_2)\)： \[Z_{out} = \frac{B+DR_S}{A+CR_S}\]
\[Z_{in} = \frac{AR_L+B}{CR_L+D}, \qquad Z_{out} = \frac{B+DR_S}{A+CR_S}\]

問1-5) \(P_{out}/P_{\max}\) 最大化の条件と \(Z_1,Z_2\)

無損失（純リアクタンス）整合回路 \(Z_1=jX_1,\,Z_2=jX_2\) で \(P_{out}/P_{\max}=1\) にするには：
・電源側整合：\(Z_{in}=R_S\)（入力端に最大電力が入る）
・負荷側整合：\(Z_{out}=R_L\)（\(R_L\) に最大電力が出る）

\(Z_{in}=jX_1(R_L+jX_2)/(R_L+j(X_1+X_2))=R_S\) が実数になる条件：
分子の虚部 \(=0\)：\(X_2^2+X_1X_2+R_L^2=0\)
実部条件：\(X_1^2R_L/(R_L^2+(X_1+X_2)^2)=R_S\)

連立解（\(R_S>R_L\)）： \[X_2^2 = R_L(R_S-R_L), \qquad X_1 = -\frac{R_SR_L}{X_2}\]
\[Z_2 = \pm j\sqrt{R_L(R_S-R_L)}, \qquad Z_1 = \mp j\,R_S\!\sqrt{\frac{R_L}{R_S-R_L}}\] （\(Z_1,Z_2\) は符号が逆の純虚数：一方がインダクタ、他方がキャパシタ）
このとき \(P_{out}/P_{\max}=1\)（無損失整合ネットワークで全電力が \(R_L\) に供給される）

大問2：直列 RLC 共振回路

実効値 \(V\)・角周波数 \(\omega\) の交流電源、抵抗 \(R\)、インダクタンス \(L\)、可変キャパシタンス \(C\) の直列接続（\(\omega L > R\)）。

問2-1) 共振時のキャパシタンス \(C_0\)

共振条件 \(\mathrm{Im}(Z)=0\)：\(\omega L = 1/(\omega C_0)\)
\[C_0 = \frac{1}{\omega^2 L}\]

問2-2) 共振時の電流 \(I_0\)

共振時 \(Z=R\)（純抵抗最小）。
\[I_0 = \frac{V}{R}\]

問2-3) \(|I/I_0|=1/\sqrt{2}\) となる \(C_1,C_2\)

\(|Z|=\sqrt{2}R\)の条件：\(R^2+(ωL-1/(\omega C))^2=2R^2\)
\[ωL - \frac{1}{\omega C} = \pm R \quad \Rightarrow \quad C = \frac{1}{\omega(\omega L \mp R)}\] \(C_2 > C_1\) より（\(\omega L - R < \omega L + R\)）：
\[C_1 = \frac{1}{\omega(\omega L + R)}, \qquad C_2 = \frac{1}{\omega(\omega L - R)}\] （\(C_1 < C_0 < C_2\)）

問2-4) コイルの Q 値（\(C_0,C_1,C_2\) のみ使用）

\(Q = \omega L/R\)。\(C_0=1/(\omega^2L)\) より \(\omega L = 1/(\omega C_0)\)。
\(C_2 - C_1 = \dfrac{2R}{\omega(\omega^2L^2-R^2)} \approx \dfrac{2R}{\omega^3L^2} = 2R\omega C_0^2\)（\(\omega L \gg R\)）
→ \(R \approx (C_2-C_1)/(2\omega C_0^2)\)、\(Q = 1/(\omega RC_0) = 2C_0/(C_2-C_1)\)
\[Q = \frac{2C_0}{C_2 - C_1}\] （\(\omega L \gg R\) の近似。厳密には \(Q=(Q^2-1)/Q\) 形を含む）

大問3：RL 並列回路の交流特性

図3.1：\(R\) と \(L\) の並列回路。振幅 100 V・周波数 50 Hz の交流電圧を印加 → 振幅 10 A の電流（図3.2）。

問3-1) 合成インピーダンス

\[Z = R \parallel j\omega L = \frac{j\omega RL}{R + j\omega L}\] \[|Z| = \frac{\omega RL}{\sqrt{R^2+\omega^2L^2}}, \qquad \angle Z = \arctan\frac{R}{\omega L}\]

問3-2) 位相関係（図3.2より）

図3.2：電流ゼロクロス（上昇方向）が \(t_0 = 1/600\) s。
電圧 \(V=100\sin(\omega t)\)（\(\omega=2\pi\times50=100\pi\) rad/s）のゼロクロス \(t=0\) より遅れている。
時間差 \(1/600\) s → 位相差 \(= \omega\times(1/600) = 100\pi/600 = \pi/6\) rad
並列 RL は電流がアドミタンス \(Y=1/R - j/(\omega L)\) に比例 → 電流は電圧より位相が遅れる。
（ア）\(\dfrac{\pi}{6}\) rad ／ （イ）b. 遅れている

問3-3) R の値

\(|Z| = |V|/|I| = 100/10 = 10\ \Omega\)
\(\angle Z = \pi/6\)（電流が電圧より \(\pi/6\) 遅れる）：\(\dfrac{R}{\omega L} = \tan(\pi/6) = \dfrac{1}{\sqrt{3}}\) → \(R = \dfrac{\omega L}{\sqrt{3}}\)
\(|Z| = \dfrac{\omega RL}{\sqrt{R^2+\omega^2L^2}} = \dfrac{\omega L}{\sqrt{3}\cdot\sqrt{1/3+1}} = \dfrac{\omega L}{\sqrt{3}\cdot\frac{2}{\sqrt{3}}} = \dfrac{\omega L}{2} = 10\)
→ \(\omega L = 20\ \Omega\)（\(L = 20/(100\pi) = 1/(5\pi)\) H）
\[R = \frac{\omega L}{\sqrt{3}} = \frac{20}{\sqrt{3}} = \frac{20\sqrt{3}}{3}\ \Omega \approx 11.5\ \Omega\]

問3-4) 実効電力と無効電力の絶対値

\(V_{rms}=100/\sqrt{2}\) V、\(I_{rms}=10/\sqrt{2}\) A、皮相電力 \(|S|=V_{rms}\times I_{rms}=500\) VA
\(\cos(\pi/6)=\sqrt{3}/2,\quad \sin(\pi/6)=1/2\)
実効電力（有効電力）：\(P = 500\cos(\pi/6) = 250\sqrt{3} \approx 433\) W
無効電力：\(|Q| = 500\sin(\pi/6) = 250\) var

解答まとめ

問主な結果
1-1\(R_S=1/G_S\)、\(V_S=I_S/G_S\)
1-2\(Z_L=R_S\)、\(P_{\max}=V_S^2/(4R_S)\)
1-3F行列：\([A,B;C,D]=[1,Z_2;1/Z_1,1+Z_2/Z_1]\)
1-4\(Z_{in}=(AR_L+B)/(CR_L+D)\)、\(Z_{out}=(B+DR_S)/(A+CR_S)\)
1-5\(Z_2=\pm j\sqrt{R_L(R_S-R_L)}\)、\(Z_1=\mp jR_S\sqrt{R_L/(R_S-R_L)}\)
2-1\(C_0=1/(\omega^2L)\)
2-2\(I_0=V/R\)
2-3\(C_1=1/(\omega(\omega L+R))\)、\(C_2=1/(\omega(\omega L-R))\)
2-4\(Q\approx 2C_0/(C_2-C_1)\)
3-1\(Z=j\omega RL/(R+j\omega L)\)
3-2位相遅れ \(\pi/6\) rad（遅れている）
3-3\(R=20\sqrt{3}/3\ \Omega\)
3-4\(P=250\sqrt{3}\) W、\(|Q|=250\) var
