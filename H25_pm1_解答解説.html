大問1　アドミタンス解析・並列共振回路

問1)　RL 直列回路（図1.1）の合成アドミタンス

インピーダンス $Z = R + j\omega L$
$$\boxed{Y = \frac{1}{R+j\omega L} = \frac{R-j\omega L}{R^2+\omega^2 L^2}}$$ 実部：$G = \dfrac{R}{R^2+\omega^2 L^2}$，虚部：$B = \dfrac{-\omega L}{R^2+\omega^2 L^2}$

問2)　アドミタンス軌跡（複素平面上）

$G = \dfrac{R}{R^2+\omega^2 L^2}$，$B = \dfrac{-\omega L}{R^2+\omega^2 L^2}$ から $\omega$ を消去する。 $B/G = -\omega L/R \;\Rightarrow\; \omega = -RB/(GL)$（$B<0$ のとき $\omega>0$） $G^2+B^2 = \dfrac{R^2+\omega^2 L^2}{(R^2+\omega^2 L^2)^2} = \dfrac{1}{R^2+\omega^2 L^2} = \dfrac{G}{R}$ $(G - \tfrac{1}{2R})^2 + B^2 = \left(\tfrac{1}{2R}\right)^2$
半径 $\dfrac{1}{2R}$，中心 $\left(\dfrac{1}{2R},\,0\right)$ の円。
$\omega=0$ では $Y=1/R$（実軸上）、$\omega\to\infty$ では $Y\to 0$ なので、実軸下半分（$B\le 0$）の半円軌跡を描く。

問3)　図1.2：$I_1$, $I_2$ と「$\omega$ によらず $I$ と $V$ が同位相」になる条件

電源電圧（実効値 $V_e$，角周波数 $\omega$）に対し： $$I_1 = \frac{V_e}{R+j\omega L}, \quad I_2 = \frac{V_e}{R+\frac{1}{j\omega C}}$$ 合成アドミタンス $Y = Y_1 + Y_2$ の虚部が 0 のとき $I$ と $V$ が同位相。 $\text{Im}(Y_1) = \dfrac{-\omega L}{R^2+\omega^2 L^2}$，$\text{Im}(Y_2) = \dfrac{\omega C}{1+\omega^2 R^2 C^2}$ $\text{Im}(Y) = 0$ の条件： $$\frac{-\omega L}{R^2+\omega^2 L^2} + \frac{\omega C}{1+\omega^2 R^2 C^2} = 0$$ $\omega \neq 0$ で除し整理（$L-CR^2 = $ 共通因子で括る）： $$L(1+\omega^2 R^2 C^2) = C(R^2+\omega^2 L^2)$$ $$(L-CR^2) - \omega^2 CL(L-CR^2) = 0$$ $$(L-CR^2)(1-\omega^2 CL) = 0$$ すべての $\omega$ で成立するには $(L-CR^2)=0$ かつ $(1-\omega^2 CL)=0$ の両方、または恒等的に $=0$： $L = CR^2$ のとき $(L-CR^2)=0$ → 括弧全体が $0 \times (\cdots) = 0$ ∀$\omega$ ✓
$$I_1 = \frac{V_e}{R+j\omega L}, \quad I_2 = \frac{j\omega C \cdot V_e}{1+j\omega RC}$$ 同位相条件：$\boxed{L = CR^2}$（$\omega$ によらず成立）

問4)　条件 $L=CR^2$ のとき電流 $I$ の実効値

$L=CR^2$ のとき $Y_1 = \dfrac{1}{R(1+j\omega CR)} = \dfrac{1}{R}\cdot\dfrac{1}{1+j\omega CR}$，$Y_2 = \dfrac{j\omega C}{1+j\omega RC}$ $$Y = Y_1 + Y_2 = \frac{1/R + j\omega C}{1+j\omega CR} = \frac{(1/R)(1+j\omega CR)}{1+j\omega CR} = \frac{1}{R}$$
$$\boxed{I = \frac{V_e}{R}}$$ $Y = 1/R$（実数・$\omega$ 非依存）なので、電流は純抵抗 $R$ と同等。実効値 $I = V_e/R$。

大問2　RL 回路の過渡応答（図2.1・2.2）

問1)　図2.1：$t=0$ でスイッチ $S_1$ 投入後の微分方程式

$$\boxed{L\frac{di}{dt} + Ri = E}$$ （KVL：$E = Ri + L\dfrac{di}{dt}$）

問2)　$i(0)=0$ の初期条件で解く（ラプラス変換を利用）

ラプラス変換（$i(0)=0$）：$(Ls+R)I(s) = E/s$ $$I(s) = \frac{E}{s(Ls+R)} = \frac{E/R}{s} - \frac{E/R}{s+R/L}$$
$$\boxed{i(t) = \frac{E}{R}\left(1-e^{-\frac{R}{L}t}\right), \quad t \ge 0}$$ 時定数 $\tau = L/R$。$t\to\infty$ で $i\to E/R$。

問3)　$t=0$ から $t=T$ の間の仕事・電力量

電源がする仕事（$W_E$）： $$W_E = \int_0^T E\,i(t)\,dt = \frac{E^2}{R}\int_0^T\!\left(1-e^{-\frac{R}{L}t}\right)dt = \frac{E^2}{R}\!\left[T - \frac{L}{R}\left(1-e^{-\frac{RT}{L}}\right)\right]$$ 抵抗で消費される電力量（$W_R$）： $$W_R = \int_0^T Ri^2\,dt = \frac{E^2}{R}\!\left[T - \frac{3L}{2R} + \frac{2L}{R}e^{-\frac{RT}{L}} - \frac{L}{2R}e^{-\frac{2RT}{L}}\right]$$ 検算：$W_E - W_R = \dfrac{1}{2}Li(T)^2 = \dfrac{L}{2}\!\left(\dfrac{E}{R}\right)^2\!\left(1-e^{-\frac{RT}{L}}\right)^2$（インダクタンスへの蓄積エネルギー）✓
$$\boxed{W_E = \frac{E^2}{R}\!\left[T - \frac{L}{R}\!\left(1-e^{-\frac{RT}{L}}\right)\right]}$$ $$\boxed{W_R = \frac{E^2}{R}\!\left[T - \frac{3L}{2R} + \frac{2L}{R}e^{-\frac{RT}{L}} - \frac{L}{2R}e^{-\frac{2RT}{L}}\right]}$$

問4)　図2.2：$S_2$ 投入後の独立な回路方程式 2 つ

図2.2 は $E$ ー $S_2$ ー $R_1$ （直列）の後に $R_2$ と $L$ が並列接続。$i_1(t)$ は $L$ に流れる電流、$i_2(t)$ は $R_2$ に流れる電流。

$R_2$ と $L$ の端子電圧が等しい：$L\dfrac{di_1}{dt} = R_2\,i_2$ ⋯(1) KVL（外ループ）：$E = R_1(i_1+i_2) + R_2\,i_2$ ⋯(2)
$$\text{(1)}: L\frac{di_1}{dt} = R_2\,i_2$$ $$\text{(2)}: E = R_1\,i_1 + (R_1+R_2)\,i_2$$

問5)　$i_1(0)=0$，$i_2(0^-)=0$ の初期条件で解く

(2) から $i_2 = (E-R_1 i_1)/(R_1+R_2)$ を(1)に代入： $$L\frac{di_1}{dt} = \frac{R_2(E-R_1 i_1)}{R_1+R_2}$$ ラプラス変換（$i_1(0)=0$）：$\;L\cdot s I_1 = \dfrac{R_2(E/s - R_1 I_1)}{R_1+R_2}$ $$I_1(s) = \frac{R_2 E}{L(R_1+R_2)} \cdot \frac{1}{s(s+\alpha)}, \quad \alpha = \frac{R_1 R_2}{L(R_1+R_2)}$$ 部分分数展開：$\dfrac{1}{s(s+\alpha)} = \dfrac{1}{\alpha}\!\left(\dfrac{1}{s}-\dfrac{1}{s+\alpha}\right)$ を使うと： $$i_1(t) = \frac{E}{R_1}\left(1-e^{-\alpha t}\right)$$ $i_2(t)$ は (2) から： $$i_2(t) = \frac{E - R_1 i_1}{R_1+R_2} = \frac{E\,e^{-\alpha t}}{R_1+R_2}$$
$$\alpha = \frac{R_1 R_2}{L(R_1+R_2)} \quad\left(\text{時定数 }\tau = \frac{L(R_1+R_2)}{R_1 R_2}\right)$$ $$\boxed{i_1(t) = \frac{E}{R_1}\left(1-e^{-\alpha t}\right), \quad i_2(t) = \frac{E}{R_1+R_2}e^{-\alpha t}, \quad t \ge 0}$$ 初期値：$i_1(0)=0$, $i_2(0^+) = E/(R_1+R_2)$（$R_2$ は抵抗なので瞬時変化可）
定常値：$i_1(\infty)=E/R_1$, $i_2(\infty)=0$（インダクタ短絡・$R_2$ 開放に相当）

問6)　$i_1(t)$, $i_2(t)$ のグラフの概形

$i_1(t)$：$t=0$ で 0 から始まり、時定数 $\tau$ で漸近的に $E/R_1$ へ増加（指数的立ち上がり）。
$i_2(t)$：$t=0^+$ で $E/(R_1+R_2)$ から始まり、時定数 $\tau$ で 0 へ指数的減衰。
（$R_1+R_2>R_1$ なので $i_2(0^+) < i_1(\infty)$）

問7)　$S_2$ 閉成後十分時間が経過したときの等価回路

十分時間経過後：$i_1(\infty)=E/R_1$（$L$ を電流 $E/R_1$ が流れる）、$i_2(\infty)=0$（$R_2$ に電流なし）。
インダクタ $L$ は DC 短絡のため $R_2$ と並列でも事実上 $R_2$ は不活性。
等価回路：点線内は電流 $E/R_1$ を流す短絡路（インダクタ $L$）のみ。
（$R_2$ は両端電圧 0 で開路等価）

大問3　MOSFET 増幅器（小信号解析）

MOSFET 小信号等価回路：電圧制御電流源 $g_m v_{gs}$（ドレインからソースへ流れる）。コンダクタンス $G_1$ は VDD 側の負荷、$G_2$ は直列素子、$G_3$ はグラウンド側負荷。

問1)　小信号等価回路

接続の確認（図3.1）：
  • VDD → $G_1$ → ドレイン（小信号では VDD = AC グラウンド）
  • ドレイン → $G_2$ → 出力ノード $v_{out}$
  • $v_{out}$ → $G_3$ → グラウンド
  • ゲート：$v_{in}$（ソース = GND なので $v_{gs}=v_{in}$）
  • MOSFET：ドレイン・ソース間に $g_m v_{gs}$ の電流源（ドレインから流出方向）
小信号等価回路：$G_1$ を AC-GND とドレイン間に接続、$G_2$ をドレインと $v_{out}$ の間に接続、$G_3$ を $v_{out}$-GND 間に接続し、$g_m v_{in}$ の電流源をドレイン-GND 間に接続。

問2)　電圧利得 $A_v = v_{out}/v_{in}$

ノード $v_d$（ドレイン）の KCL： $$G_1(0-v_d) + G_2(v_{out}-v_d) - g_m v_{in} = 0 \quad\cdots(A)$$ ノード $v_{out}$ の KCL： $$G_2(v_d-v_{out}) - G_3 v_{out} = 0 \;\Rightarrow\; v_d = \frac{G_2+G_3}{G_2}v_{out} \quad\cdots(B)$$ (B) を (A) に代入して整理： $$-\frac{(G_1+G_2)(G_2+G_3)}{G_2}v_{out} + G_2 v_{out} = g_m v_{in}$$ $$-\frac{G_1(G_2+G_3)+G_2 G_3}{G_2}v_{out} = g_m v_{in}$$
$$\boxed{A_v = \frac{v_{out}}{v_{in}} = \frac{-g_m G_2}{G_1 G_2 + G_1 G_3 + G_2 G_3}}$$ （負：反転増幅）

大問4　MOSFET 増幅器の周波数特性

図4.1：VDD → $R_2$ → ドレイン（出力 $v_{out}$）、ソース → $R_1$ → GND（ソース接地抵抗）、ゲート：$v_{in}$。MOSFET 小信号等価回路：$g_m v_{gs}$ + $C_{gs}$（ゲート-ソース間容量）。

問1)　小信号等価回路

構成：
  • $R_2$：VDD（AC-GND）とドレイン（= $v_{out}$）を接続
  • $R_1$：ソースノード $v_s$ と GND を接続
  • $C_{gs}$：ゲート（$v_{in}$）とソース（$v_s$）の間
  • $g_m v_{gs}$：ドレイン-GND 間電流源（ドレインから流出）
  • $v_{gs} = v_{in} - v_s$

問2)　伝達関数 $G(j\omega) = v_{out}/v_{in}$

ソースノード $v_s$ の KCL（$C_{gs}$ の電流がソースに流入）： $$g_m v_{gs} + j\omega C_{gs}v_{gs} = \frac{v_s}{R_1}$$ $$(g_m + j\omega C_{gs})(v_{in}-v_s) = \frac{v_s}{R_1}$$ $$v_{gs} = \frac{v_{in}-v_s}{1} = \frac{v_{in}/R_1}{1/R_1 + g_m + j\omega C_{gs}} = \frac{v_{in}}{1+g_m R_1 + j\omega C_{gs}R_1}$$ ドレイン KCL（$R_2$ と電流源）： $$\frac{v_{out}-0}{R_2} + g_m v_{gs} = 0 \;\Rightarrow\; v_{out} = -g_m R_2\,v_{gs}$$
$$\boxed{G(j\omega) = \frac{v_{out}}{v_{in}} = \frac{-g_m R_2}{1+g_m R_1 + j\omega C_{gs}R_1} = \frac{A_0}{1+j\omega/\omega_0}}$$ ここで： $$A_0 = -\frac{g_m R_2}{1+g_m R_1}, \quad \omega_0 = \frac{1+g_m R_1}{C_{gs}R_1}$$

問3)　低域利得 $A_0$

$$\boxed{A_0 = G(j0) = -\frac{g_m R_2}{1+g_m R_1}}$$

問4)　遮断角周波数 $\omega_0$

$|G(j\omega_0)| = |A_0|/\sqrt{2}$ となる $\omega_0$： $$|1+j\omega_0/\omega_0| = \sqrt{2} \;\Rightarrow\; \omega_0 = \frac{1+g_m R_1}{C_{gs}R_1}$$
$$\boxed{\omega_0 = \frac{1+g_m R_1}{C_{gs}R_1}}$$

問5)　ボーデ線図（骨格）

利得特性（骨格 Bode 線図）：
  • $\omega \ll \omega_0$：$|G| = |A_0| = g_m R_2/(1+g_m R_1)$（定数、水平）
  • $\omega = \omega_0$：$|G| = |A_0|/\sqrt{2}$（$-3\,\text{dB}$ 点）
  • $\omega \gg \omega_0$：$-20\,\text{dB/decade}$ で減衰（折れ線近似：$\omega_0$ の点で折れる）
位相特性： $\angle G(j\omega) = 180° - \arctan(\omega/\omega_0)$（$A_0<0$ による反転で $180°$ のオフセット）
  • $\omega \to 0$：位相 $= 180°$
  • $\omega = \omega_0$：位相 $= 180° - 45° = 135°$
  • $\omega \to \infty$：位相 $= 90°$
