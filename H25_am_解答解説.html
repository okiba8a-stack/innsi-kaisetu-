大問1：フーリエ解析

1-1）方形波 \(v_1(t)\) のフーリエ級数展開

問題：図のように周期 \(2\pi\)、振幅 \(E\) の方形波 \[v_1(t)=\begin{cases}E & (0 < t < \pi)\\ -E & (-\pi < t < 0)\end{cases}\] のフーリエ級数展開を求めよ。

STEP 1：対称性の確認

\(v_1(t)\) は奇関数（\(v_1(-t) = -v_1(t)\)）なので、フーリエ係数 \(a_0 = 0,\; a_n = 0\)。

STEP 2：\(b_n\) の計算

$$b_n = \frac{1}{\pi}\int_{-\pi}^{\pi}v_1(t)\sin(nt)\,dt = \frac{2}{\pi}\int_0^{\pi}E\sin(nt)\,dt$$ $$= \frac{2E}{\pi}\!\left[-\frac{\cos(nt)}{n}\right]_0^{\pi} = \frac{2E}{n\pi}\bigl[1 - \cos(n\pi)\bigr] = \frac{2E}{n\pi}\bigl[1-(-1)^n\bigr]$$
  • \(n\) が偶数：\(b_n = 0\)
  • \(n\) が奇数：\(b_n = \dfrac{4E}{n\pi}\)
$$\boxed{v_1(t) = \frac{4E}{\pi}\sum_{n=1,3,5,\ldots}\frac{\sin(nt)}{n} = \frac{4E}{\pi}\!\left(\sin t + \frac{\sin 3t}{3} + \frac{\sin 5t}{5} + \cdots\right)}$$
奇数次の sin 成分のみからなる（偶数次はゼロ）。これは方形波の半波対称性 \(v_1(t+\pi) = -v_1(t)\) による。

1-2）フィルタ通過後の出力波形

問題：\(v_1(t)\) を「周期が \(\pi/2\) より長い正弦波成分のみを通過させる回路」に通した定常出力を求めよ。

STEP 1：通過条件の変換

周期 \(T > \pi/2 \;\Rightarrow\; \dfrac{2\pi}{\omega} > \dfrac{\pi}{2} \;\Rightarrow\; \omega < 4\)

STEP 2：各高調波を確認

次数 n角周波数 ω = n通過？
11 < 4✓ 通過
33 < 4✓ 通過
55 ≥ 4✗ 遮断
7以上7, 9, … > 4✗ 遮断
$$\boxed{v_{\rm out}(t) = \frac{4E}{\pi}\!\left(\sin t + \frac{\sin 3t}{3}\right)}$$

振幅が異なる基本波と第3高調波の重ね合わせ（台形波に近い波形）。

1-3）修正方形波 \(v_2(t)\) のフーリエ級数展開

問題：図のように周期 \(2\pi\)、振幅 \(E\) で、\(0 < \alpha < \pi/2\) のとき \[v_2(t)=\begin{cases}E & (\alpha < t < \pi-\alpha)\\ -E & (-\pi+\alpha < t < -\alpha)\\ 0 & (\text{その他})\end{cases}\] のフーリエ級数展開を求めよ。

STEP 1：対称性の確認

\(v_2(t)\) も奇関数 → \(a_0 = a_n = 0\)

STEP 2：\(b_n\) の計算

$$b_n = \frac{2}{\pi}\int_0^{\pi}v_2(t)\sin(nt)\,dt = \frac{2}{\pi}\int_{\alpha}^{\pi-\alpha}E\sin(nt)\,dt$$ $$= \frac{2E}{\pi}\!\left[-\frac{\cos(nt)}{n}\right]_{\alpha}^{\pi-\alpha} = \frac{2E}{n\pi}\bigl[\cos(n\alpha) - \cos(n(\pi-\alpha))\bigr]$$

ここで \(\cos(n(\pi-\alpha)) = \cos(n\pi)\cos(n\alpha) = (-1)^n\cos(n\alpha)\)（\(\sin(n\pi)=0\)）なので：

$$b_n = \frac{2E\cos(n\alpha)}{n\pi}\bigl[1 - (-1)^n\bigr]$$
  • \(n\) 偶数：\(b_n = 0\)
  • \(n\) 奇数：\(b_n = \dfrac{4E\cos(n\alpha)}{n\pi}\)
$$\boxed{v_2(t) = \frac{4E}{\pi}\sum_{n=1,3,5,\ldots}\frac{\cos(n\alpha)}{n}\sin(nt) = \frac{4E}{\pi}\!\left(\cos\alpha\,\sin t + \frac{\cos 3\alpha}{3}\sin 3t + \cdots\right)}$$
\(\alpha = 0\) とすると \(\cos(n\alpha) = 1\) となり、問1-1の \(v_1(t)\) のフーリエ級数に一致することを確認できる。

1-4）基本波成分のみとなる \(\alpha\) の決定

問題：\(v_2(t)\) を同じフィルタ（周期 \(>\pi/2\)、すなわち \(\omega < 4\) を通過）に通したとき、出力が周期 \(2\pi\) の基本波成分のみとなる \(\alpha\) を求め、出力波形を図示せよ。

STEP 1：フィルタ通過後の式

\(v_2(t)\) のうち \(\omega < 4\) を通過するのは \(n=1\)（\(\omega=1\)）と \(n=3\)（\(\omega=3\)）：

$$v_{\rm out}(t) = \frac{4E}{\pi}\!\left(\cos\alpha\,\sin t + \frac{\cos 3\alpha}{3}\sin 3t\right)$$

STEP 2：\(n=3\) 成分をゼロにする条件

$$\frac{\cos 3\alpha}{3} = 0 \;\Rightarrow\; \cos 3\alpha = 0$$ $$3\alpha = \frac{\pi}{2} \quad \left(0 < \alpha < \frac{\pi}{2} \text{ より } 0 < 3\alpha < \frac{3\pi}{2}\text{、最小解を取る}\right)$$ $$\therefore\; \alpha = \frac{\pi}{6}$$
$$\boxed{\alpha = \frac{\pi}{6}}$$

このとき出力：

$$v_{\rm out}(t) = \frac{4E}{\pi}\cos\frac{\pi}{6}\cdot\sin t = \frac{4E}{\pi}\cdot\frac{\sqrt{3}}{2}\cdot\sin t = \frac{2\sqrt{3}\,E}{\pi}\sin t$$

出力波形：

振幅 \(\dfrac{2\sqrt{3}\,E}{\pi} \approx 1.10\,\dfrac{E}{\pi/2} \approx 0.55\,E\)（元の振幅 \(E\) より小さい）の純正弦波。

大問2：確率・通信路

地点 X → Y → Z の通信路（図2.1）で、X が「1」を送る確率を \(p_0\)、「0」を送る確率を \(1-p_0\)、X→Y の誤り確率を \(p_1\)、Y→Z の誤り確率を \(p_2\) とする。

2-1）地点 Y で「1」を受け取る確率

全確率の法則で場合分け：

X が送る値確率Y が「1」を受け取る確率寄与
「1」\(p_0\)\(1-p_1\)（誤りなし）\(p_0(1-p_1)\)
「0」\(1-p_0\)\(p_1\)（誤りあり）\((1-p_0)p_1\)
$$P(Y=1) = p_0(1-p_1) + (1-p_0)p_1$$ $$\boxed{= p_0(1-2p_1) + p_1}$$

2-2）Z で「1」を受け取ったとき Y の受信が「1」であった確率

ベイズの定理を使う： \(P(Y{=}1 \mid Z{=}1) = \dfrac{P(Z{=}1 \mid Y{=}1)\,P(Y{=}1)}{P(Z{=}1)}\)

各条件付き確率：

  • \(P(Z{=}1 \mid Y{=}1) = 1-p_2\)　（Y→Z で誤りなし）
  • \(P(Z{=}1 \mid Y{=}0) = p_2\)　（Y→Z で誤りあり）

周辺確率 \(P(Z{=}1)\)：

\(q \equiv P(Y{=}1) = p_0(1-2p_1)+p_1\) とおくと、

$$P(Z{=}1) = (1-p_2)\,q + p_2\,(1-q) = q(1-2p_2) + p_2$$
$$\boxed{P(Y{=}1 \mid Z{=}1) = \frac{(1-p_2)\,q}{q(1-2p_2)+p_2}}$$ $$\text{ただし } q = p_0(1-2p_1)+p_1$$
ベイズの定理の手順まとめ：①事前確率 \(P(Y)\) を求める → ②尤度 \(P(Z|Y)\) を設定 → ③周辺確率 \(P(Z)\) を全確率の法則で計算 → ④ベイズ公式に代入。

2-3）白色雑音が加わるときの確率密度分布 \(p(y)\)

X から「1」を連続送信。Y の受信器では受信信号強度 \(y\) = 送信信号 + 白色雑音 となる。

白色雑音の性質：

  • タイミング・強度がランダム → 各サンプルは独立
  • 中心極限定理より、十分な量の信号を受信すると振幅分布は正規（ガウス）分布に収束する

送信信号「1」（固定値）+ ガウス雑音 \(n \sim \mathcal{N}(0,\,\sigma^2)\) より：

$$y = 1 + n \;\sim\; \mathcal{N}(1,\,\sigma^2)$$
$$\boxed{p(y) = \frac{1}{\sqrt{2\pi\sigma^2}}\exp\!\left(-\frac{(y-1)^2}{2\sigma^2}\right)}$$

平均 1、分散 \(\sigma^2\) のガウス分布（\(\sigma^2\) は雑音の電力）。

物理的意味：送信信号値（ここでは 1）を中心としたガウス分布。SNR が高い（\(\sigma\) が小さい）ほど分布が鋭く、判定誤り率が低い。実際の通信システム設計ではこの分布から誤り確率（BER）が計算される。

大問3：微分方程式・複素関数

3-1(ア）斉次微分方程式の基本解

$$\frac{d^2y}{dx^2} - 5\frac{dy}{dx} + 6y = 0 \tag{3.1}$$

特性方程式：

$$\lambda^2 - 5\lambda + 6 = 0 \;\Rightarrow\; (\lambda-2)(\lambda-3) = 0$$ $$\lambda = 2,\quad \lambda = 3$$

（相異なる実数根）

$$\boxed{y_1 = e^{2x},\quad y_2 = e^{3x}}$$

一般解：\(y = C_1 e^{2x} + C_2 e^{3x}\)（\(C_1, C_2\) は任意定数）

3-1(イ）非斉次微分方程式の一般解

$$\frac{d^2f}{dx^2} - 5\frac{df}{dx} + 6f = e^{2x} \tag{3.2}$$

STEP 1：斉次解（3.1の一般解）

$$f_h = C_1 e^{2x} + C_2 e^{3x}$$

STEP 2：特殊解を未定係数法で求める

右辺 \(e^{2x}\) に対応する特解候補は \(Ae^{2x}\) だが、\(\lambda=2\) は特性方程式の根なので、\(x\) をかけて修正：

$$f_p = Axe^{2x}$$

微分する：

$$f_p' = Ae^{2x} + 2Axe^{2x} = A(1+2x)e^{2x}$$ $$f_p'' = 2A(1+2x)e^{2x} + 2Ae^{2x} = A(4+4x)e^{2x}$$

方程式に代入：

$$A(4+4x)e^{2x} - 5A(1+2x)e^{2x} + 6Axe^{2x} = e^{2x}$$ $$A\bigl[(4+4x) - 5(1+2x) + 6x\bigr]e^{2x} = e^{2x}$$ $$A\bigl[4+4x-5-10x+6x\bigr]e^{2x} = e^{2x}$$ $$A\bigl[-1 + 0\cdot x\bigr]e^{2x} = e^{2x}$$ $$-A = 1 \;\Rightarrow\; A = -1$$

特殊解：\(f_p = -xe^{2x}\)

一般解（斉次解 + 特殊解）：

$$\boxed{f(x) = C_1 e^{2x} + C_2 e^{3x} - xe^{2x}}$$
なぜ \(Axe^{2x}\) とするか：右辺 \(e^{2x}\) の指数 2 が特性方程式の根（1 重）のため、通常の \(Ae^{2x}\) を代入すると斉次解と線形従属になりゼロになる。\(x\) を 1 回かけることで独立な特解が得られる。

3-2）複素関数の極と留数

$$f(z) = \frac{z}{z^2-7z+12} \tag{3.3}$$

STEP 1：分母を因数分解して極を求める

$$z^2 - 7z + 12 = (z-3)(z-4) = 0$$ $$\therefore \text{極：} z = 3 \text{（1位の極）},\quad z = 4 \text{（1位の極）}$$

STEP 2：各極での留数を計算

1位の極での留数公式：\(\mathrm{Res}[f, z_0] = \lim_{z \to z_0}(z-z_0)\,f(z)\)

極 \(z=3\) での留数：

$$\mathrm{Res}[f, 3] = \lim_{z \to 3}(z-3)\cdot\frac{z}{(z-3)(z-4)} = \frac{3}{3-4} = \frac{3}{-1} = -3$$

極 \(z=4\) での留数：

$$\mathrm{Res}[f, 4] = \lim_{z \to 4}(z-4)\cdot\frac{z}{(z-3)(z-4)} = \frac{4}{4-3} = \frac{4}{1} = 4$$
極位数留数
\(z = 3\)1位\(-3\)
\(z = 4\)1位\(4\)
確認（部分分数展開）： \(\dfrac{z}{(z-3)(z-4)} = \dfrac{A}{z-3}+\dfrac{B}{z-4}\) とすると、 \(A = \mathrm{Res}[f,3] = -3\)、\(B = \mathrm{Res}[f,4] = 4\) より \(f(z) = \dfrac{-3}{z-3}+\dfrac{4}{z-4}\)。 \(z \to \infty\) で \(f \to 0\) ✓
