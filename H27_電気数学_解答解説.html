大問1：複素数・複素積分

1-1）\(z^4 - \left(\dfrac{\sqrt{3}}{4} - \dfrac{1}{4}j\right) = 0\) の解

STEP 1：右辺を極形式に変換する

\(z^4 = \dfrac{\sqrt{3}}{4} - \dfrac{1}{4}j\) の絶対値と偏角を求める。

$$|z^4| = \sqrt{\left(\frac{\sqrt{3}}{4}\right)^2 + \left(\frac{1}{4}\right)^2} = \sqrt{\frac{3}{16} + \frac{1}{16}} = \sqrt{\frac{4}{16}} = \frac{1}{2}$$

実部 \(> 0\)、虚部 \(< 0\) より第4象限：

$$\arg(z^4) = -\arctan\!\left(\frac{1/4}{\sqrt{3}/4}\right) = -\arctan\!\left(\frac{1}{\sqrt{3}}\right) = -\frac{\pi}{6}$$

\(0 \le \theta < 2\pi\) の範囲で表すと：\(\arg(z^4) = 2\pi - \dfrac{\pi}{6} = \dfrac{11\pi}{6}\)

$$z^4 = \frac{1}{2}\,e^{\,j\cdot\frac{11\pi}{6}}$$

STEP 2：4乗根を求める

\(|z| = \left(\dfrac{1}{2}\right)^{1/4} = 2^{-1/4}\)、偏角は \(\dfrac{1}{4}\left(\dfrac{11\pi}{6} + 2\pi k\right)\)（\(k = 0,1,2,3\)）

4つの解（すべて \(|z| = 2^{-1/4}\)）：

$$z_0 = 2^{-1/4}\,e^{\,j\cdot\frac{11\pi}{24}}, \quad \theta_0 = \frac{11\pi}{24}$$ $$z_1 = 2^{-1/4}\,e^{\,j\cdot\frac{23\pi}{24}}, \quad \theta_1 = \frac{23\pi}{24}$$ $$z_2 = 2^{-1/4}\,e^{\,j\cdot\frac{35\pi}{24}}, \quad \theta_2 = \frac{35\pi}{24}$$ $$z_3 = 2^{-1/4}\,e^{\,j\cdot\frac{47\pi}{24}}, \quad \theta_3 = \frac{47\pi}{24}$$

（隣り合う解は \(\pi/2 = 90°\) ずつ離れている）

複素平面上の図示：

各解は \(2^{-1/4}\) を半径とする円上に 90° 間隔で並ぶ。最初の偏角が \(11\pi/24 = 82.5°\) なのは、\(z^4\) の偏角 \(11\pi/6\) を 4 で割ったため。

1-2）複素積分

被積分関数の特異点は \(z = 5\)（実軸上）。

（ア）\(\displaystyle\int_C \frac{1}{z-5}\,dz\)、\(C\colon |z|=3\)

特異点 \(z=5\) は \(|z|=3\) の外側（\(|5|=5>3\)）にある。
コーシーの定理より、正則関数の閉曲線積分は 0。
$$\boxed{\int_C \frac{1}{z-5}\,dz = 0}$$

（イ）\(\displaystyle\int_C \frac{1}{z-5}\,dz\)、\(C\colon |z|=6\)

特異点 \(z=5\) は \(|z|=6\) の内側（\(|5|=5<6\)）にある。
留数定理を適用：\(\mathrm{Res}\!\left[\dfrac{1}{z-5},\;z=5\right] = 1\)
$$\boxed{\int_C \frac{1}{z-5}\,dz = 2\pi j}$$

（ウ）\(\displaystyle\int_C \frac{1}{(z-5)^2}\,dz\)、\(C\colon |z|=6\)

特異点 \(z=5\)（2 位の極）は \(|z|=6\) の内側にある。
コーシーの積分公式（高階版）： $$\int_C \frac{f(z)}{(z-a)^{n+1}}\,dz = \frac{2\pi j}{n!}\,f^{(n)}(a)$$ ここで \(f(z)=1\)、\(a=5\)、\(n=1\)： $$\int_C \frac{1}{(z-5)^2}\,dz = \frac{2\pi j}{1!}\cdot f'(5) = 2\pi j \cdot 0 = 0$$

あるいは留数：\(\mathrm{Res}\!\left[\dfrac{1}{(z-5)^2},\;z=5\right] = \lim_{z\to 5}\dfrac{d}{dz}\!\left[(z-5)^2\cdot\dfrac{1}{(z-5)^2}\right] = \dfrac{d}{dz}[1] = 0\)

$$\boxed{\int_C \frac{1}{(z-5)^2}\,dz = 0}$$
判定の手順まとめ：①特異点の位置を確認 → ②積分路の内側か外側か判定 → ③コーシーの定理 or 留数定理を適用

大問2：デルタ関数列のフーリエ解析とサンプリング定理

2-1）\(\delta_T(t)\) のフーリエ級数展開

\(\delta_T(t) = \displaystyle\sum_{n=-\infty}^{\infty}\delta(t-nT_s)\) は周期 \(T_s\) の周期関数。

フーリエ級数の係数を計算する

フーリエ係数（インデックスを \(m\) とする）：

$$c_m = \frac{1}{T_s}\int_{-T_s/2}^{T_s/2}\delta_T(t)\,e^{-j\frac{2\pi m}{T_s}t}\,dt$$

一周期 \([-T_s/2,\,T_s/2]\) に含まれるデルタ関数は \(\delta(t)\) のみ：

$$c_m = \frac{1}{T_s}\int_{-T_s/2}^{T_s/2}\delta(t)\,e^{-j\frac{2\pi m}{T_s}t}\,dt = \frac{1}{T_s}\cdot e^{0} = \frac{1}{T_s}$$

すべての \(m\) に対して \(c_m\) が同じ値 \(1/T_s\)。

フーリエ係数 \(c_m = 1/T_s\) を代入：

$$\boxed{\delta_T(t) = \frac{1}{T_s}\sum_{n=-\infty}^{\infty}\exp\!\left(j\frac{2\pi n}{T_s}t\right)}$$
デルタ関数列のフーリエ係数がすべて等しい（\(1/T_s\) で一定）ことが重要。これはデルタ関数のサンプリング性質 \(\int f(t)\delta(t-t_0)\,dt = f(t_0)\) から直接導かれる。

2-2）サンプリング関数 \(f_s(t)\) のフーリエ変換

\(f_s(t) = \displaystyle\sum_{n=-\infty}^{\infty}T_s\,f(t)\,\delta(t-nT_s)\) のフーリエ変換を求める。

STEP 1：問 1) の結果を使って \(f_s(t)\) を書き換える

$$f_s(t) = T_s\,f(t)\cdot\delta_T(t) = T_s\,f(t)\cdot\frac{1}{T_s}\sum_n e^{\,j\frac{2\pi n}{T_s}t} = f(t)\sum_n e^{\,j\frac{2\pi n}{T_s}t}$$

STEP 2：フーリエ変換を計算する

$$F_s(\omega) = \int_{-\infty}^{\infty}f_s(t)\,e^{-j\omega t}\,dt = \int_{-\infty}^{\infty}f(t)\sum_n e^{\,j\frac{2\pi n}{T_s}t}\cdot e^{-j\omega t}\,dt$$ $$= \sum_{n=-\infty}^{\infty}\int_{-\infty}^{\infty}f(t)\,e^{-j\!\left(\omega - \frac{2\pi n}{T_s}\right)t}\,dt = \sum_{n=-\infty}^{\infty}F\!\left(\omega - \frac{2\pi n}{T_s}\right)$$
$$\boxed{F_s(\omega) = \sum_{n=-\infty}^{\infty}F\!\left(\omega - n\omega_s\right)}$$

ただし \(\omega_s = \dfrac{2\pi}{T_s}\)（サンプリング角周波数）

物理的意味（サンプリング定理）：
サンプリング後のスペクトルは、元のスペクトル \(F(\omega)\) がサンプリング周波数 \(\omega_s\) ごとに繰り返し並ぶ形になる。元信号の最大周波数を \(\omega_{\max}\) とすると、\(\omega_s > 2\omega_{\max}\)（ナイキスト条件）を満たせば、スペクトルの重なり（エイリアシング）が生じず元信号を復元できる。

大問3：拡散方程式（偏微分方程式）

対象方程式：

$$\frac{\partial^2 u}{\partial x^2} = \frac{1}{D}\frac{\partial u}{\partial t} \qquad (D\text{：正定数})\tag{1}$$

3-1）変数分離による常微分方程式の導出

\(u(x,t) = X(x)T(t)\) を式(1) に代入：

$$X''(x)\,T(t) = \frac{1}{D}\,X(x)\,T'(t)$$

両辺を \(X(x)T(t)\) で割る：

$$\frac{X''(x)}{X(x)} = \frac{1}{D}\cdot\frac{T'(t)}{T(t)}$$

左辺は \(x\) のみ、右辺は \(t\) のみの関数 → 両辺とも定数 \(\lambda\) に等しい。

2つの常微分方程式：

$$X'' - \lambda X = 0 \qquad \cdots (X\text{方程式})$$ $$T' - D\lambda T = 0 \qquad \cdots (T\text{方程式})$$

3-2）周期解の仮定による一般解

λ の符号の考察

  • \(\lambda > 0\) → \(X(x)\) が指数関数的に発散（周期解でない）→ 棄却
  • \(\lambda = 0\) → \(X(x) = ax+b\)（非自明な周期解なし）→ 棄却
  • \(\lambda = -k^2 < 0\) → \(X(x)\) が三角関数（周期解）→ 採用

\(\lambda = -k^2\)（\(k > 0\)）とおくと：

$$X'' + k^2 X = 0 \;\Rightarrow\; X(x) = A\cos kx + B\sin kx$$ $$T' + Dk^2 T = 0 \;\Rightarrow\; T(t) = C\,e^{-Dk^2 t}$$

重ね合わせにより一般解：

$$\boxed{u(x,t) = \sum_k \bigl[A_k\cos kx + B_k\sin kx\bigr]\,e^{-Dk^2 t}}$$

（任意定数 \(A_k,\,B_k\)、\(k\) は境界条件で決まる）

3-3）境界条件・初期条件を満たす解

境界条件 \(u(0,t) = 0\) より：

$$X(0) = 0 \;\Rightarrow\; A_k\cos 0 + B_k\sin 0 = A_k = 0$$

よって \(X(x) = B_k\sin kx\)

境界条件 \(u(L,t) = 0\) より：

$$B_k\sin(kL) = 0 \;\Rightarrow\; kL = n\pi \;\Rightarrow\; k_n = \frac{n\pi}{L} \quad (n = 1,2,3,\ldots)$$

固有関数展開：

$$u(x,t) = \sum_{n=1}^{\infty}B_n\sin\!\left(\frac{n\pi x}{L}\right)e^{-D\left(\frac{n\pi}{L}\right)^2\!t}$$

初期条件 \(u(x,0) = f(x)\) より：

$$f(x) = \sum_{n=1}^{\infty}B_n\sin\!\left(\frac{n\pi x}{L}\right)$$

フーリエ正弦級数の直交性を使い、両辺に \(\sin\!\left(\dfrac{m\pi x}{L}\right)\) をかけて \([0,L]\) で積分：

$$\int_0^L f(x)\sin\!\left(\frac{m\pi x}{L}\right)dx = B_m\int_0^L\sin^2\!\left(\frac{m\pi x}{L}\right)dx = B_m\cdot\frac{L}{2}$$ $$\therefore\; B_n = \frac{2}{L}\int_0^L f(x)\sin\!\left(\frac{n\pi x}{L}\right)dx$$
$$\boxed{u(x,t) = \sum_{n=1}^{\infty}B_n\sin\!\left(\frac{n\pi x}{L}\right)\exp\!\left[-D\!\left(\frac{n\pi}{L}\right)^{\!2}\!t\right]}$$ $$\text{ただし}\quad B_n = \frac{2}{L}\int_0^L f(x)\sin\!\left(\frac{n\pi x}{L}\right)dx$$
物理的意味：各モードは固有角周波数 \(k_n = n\pi/L\) を持ち、時定数 \(\tau_n = L^2/(D n^2\pi^2)\) で減衰する拡散（熱伝導）解。高次モードほど速く減衰し、長時間後は基本モード（\(n=1\)）が支配的になる。
