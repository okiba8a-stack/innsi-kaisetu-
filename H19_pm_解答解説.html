大問1：同軸線路の静電・磁気特性

構成：内半径 $a$（円柱状導体）、$a\le r\le b$ に誘電率 $\varepsilon$ の誘電体、$b\le r\le c$ が中空円筒導体（金属）、$r>c$ は真空（$\varepsilon_0$）。透磁率は全域 $\mu_0$。

問1-a）$a\le r\le b$ の電界 $E_r(r)$

単位長さあたり内導体の線電荷密度を $Q_L$ とする。ガウスの法則（半径 $r$，長さ $1$ の円筒面）： $$\varepsilon E_r\cdot 2\pi r = Q_L \quad \Rightarrow \quad E_r = \frac{Q_L}{2\pi\varepsilon r}$$ 電圧 $V = \int_a^b E_r\,dr = \frac{Q_L}{2\pi\varepsilon}\ln\frac{b}{a}$ より $Q_L = \frac{2\pi\varepsilon V}{\ln(b/a)}$。
$$E_r(r) = \frac{V}{r\ln(b/a)} \qquad (a\le r\le b)$$

問1-b）単位長さあたりの静電容量 $C$

$$C = \frac{Q_L}{V} = \frac{2\pi\varepsilon}{\ln(b/a)} \quad [\text{F/m}]$$

問1-c）誘電体中の電界の最大値 $E_m$

$E_r \propto 1/r$ は $r$ の減少関数 → $r=a$（最小値）で最大：
$$E_m = E_r(a) = \frac{V}{a\ln(b/a)}$$

問1-d）$E_m$ を最小化する $a$（$b$ 固定）

$E_m = V/[a\ln(b/a)]$ を最小化 ⟺ 分母 $f(a) = a\ln(b/a)$ を最大化。 $$f'(a) = \ln\frac{b}{a} + a\cdot\left(-\frac{1}{a}\right) = \ln\frac{b}{a} - 1 = 0$$
$$\ln\frac{b}{a} = 1 \implies a = \frac{b}{e} \approx 0.368\,b$$ このとき $E_m = V/(a\cdot1) = Ve/b$（最小値）
この $a=b/e$ は実際の同軸ケーブル設計における最適内径比であり，同じ外径 $b$ に対して絶縁破壊を起こしにくい設計を実現する。

大問2：一様磁束密度 $B$ 中の導体棒の運動（電磁誘導）

一様磁束密度 $\boldsymbol{B}$（$z$ 軸正方向）。質量 $m$ の導体棒，抵抗 $R$ 接続。

問1）領域1（平行レール，間隔 $l$）で速度 $v$

a）電流 $I$ の大きさ

誘起 EMF：$\varepsilon = Blv$（ファラデーの法則，単位時間の磁束変化 $= Bl\cdot v$）
$$I = \frac{Blv}{R}$$

b）一定速度を維持するための外力 $F$

電磁制動力（アンペア力）：$F_{brake} = BIl = B^2l^2v/R$（速度を下げる方向） 一定速度 → 外力が制動力を釣り合わせる：
$$F = \frac{B^2l^2v}{R}$$

c）$t=0$ で外力を除いた後の速度 $v(t)$

運動方程式（外力なし）：$m\dfrac{dv}{dt} = -BIl = -\dfrac{B^2l^2}{R}v$（$\alpha = B^2l^2/(mR)$） $\dfrac{dv}{v} = -\alpha\,dt$ → 積分（初期条件 $v(0)=v$）：
$$v(t) = v\,e^{-\alpha t} = v\exp\!\left(-\frac{B^2l^2}{mR}t\right)$$ 運動：指数関数的に減速し（電磁制動），$t\to\infty$ で $v\to 0$ に漸近して静止する。

d）$R$ で失われるエネルギーの総和 $= t=0$ の運動エネルギー

$P(t) = I^2 R = (B^2l^2v^2e^{-2\alpha t})/R$： $$W = \int_0^\infty P(t)\,dt = \frac{B^2l^2v^2}{R}\int_0^\infty e^{-2\alpha t}dt = \frac{B^2l^2v^2}{R}\cdot\frac{1}{2\alpha}$$ $\alpha = B^2l^2/(mR)$ を代入： $$W = \frac{B^2l^2v^2}{R}\cdot\frac{mR}{2B^2l^2} = \frac{1}{2}mv^2 \quad\blacksquare$$
エネルギー保存：$t=0$ の運動エネルギー $\frac{1}{2}mv^2$ がすべて抵抗のジュール熱として散逸する（電磁制動による完全減衰）。

問2）領域2（同心半円形レール，内半径 $r$，外半径 $r+l$）で角速度 $\omega$ 回転

a）電流 $I$ の大きさ

導体棒が角速度 $\omega$ で回転するとき，単位時間に掃く面積： $$\frac{dA}{dt} = \int_r^{r+l}\omega\rho\,d\rho = \frac{\omega}{2}\left[(r+l)^2-r^2\right] = \frac{\omega l(2r+l)}{2}$$ 誘起 EMF：$\varepsilon = B\dfrac{dA}{dt} = \dfrac{B\omega l(2r+l)}{2}$
$$I = \frac{\varepsilon}{R} = \frac{B\omega l(2r+l)}{2R}$$

b）$B$ の計算（数値）

$r=0.10\,\text{m}$，$l=0.20\,\text{m}$，$R=200\,\Omega$，$\omega=2\,\text{rad/s}$，$I=10^{-4}\,\text{A}$： $$I = \frac{B\cdot 2\cdot 0.20\cdot (0.20+0.20)}{2\times200} = \frac{B\cdot 2\cdot 0.20\cdot 0.40}{400} = \frac{0.16B}{400} = 4\times10^{-4}B$$ $10^{-4} = 4\times10^{-4}B$：
$$B = 0.25 \text{ T}$$
