大問1　平行平板コンデンサへの誘電体挿入

電極板一辺 $a$（面積 $a^2$），間隔 $b$，誘電体板（同寸法，厚さ $b$）を左端から挿入。

1-a)　誘電体の位置が $x$ のときの静電容量 $C$

誘電体が占める面積：$a \times x$（容量 $\varepsilon_f \times ax/b$） 真空が占める面積：$a \times (a-x)$（容量 $\varepsilon_0 \times a(a-x)/b$） これらは並列接続：
$$\boxed{C = \frac{a}{b}\bigl[\varepsilon_0 a + (\varepsilon_f - \varepsilon_0)x\bigr]}$$ $x=0$：$C = \varepsilon_0 a^2/b$（真空），$x=a$：$C=\varepsilon_f a^2/b$（全充填）

1-b)　電荷 $+Q, -Q$ を与えたときの誘電体に働く力

電荷一定（孤立系）での仮想変位法。静電エネルギー $U = Q^2/(2C)$： $$F = -\frac{\partial U}{\partial x}\bigg|_{Q=\text{const}} = -\frac{\partial}{\partial x}\!\left(\frac{Q^2}{2C}\right) = \frac{Q^2}{2C^2}\frac{dC}{dx}$$ $\dfrac{dC}{dx} = \dfrac{a(\varepsilon_f-\varepsilon_0)}{b} > 0$（$\varepsilon_f > \varepsilon_0$ より） $F > 0$ → 誘電体は $+x$ 方向（コンデンサ内部に引き込まれる方向）に引力を受ける。
$$\boxed{F = \frac{Q^2 a(\varepsilon_f-\varepsilon_0)}{2b\,C^2} = \frac{Q^2(\varepsilon_f-\varepsilon_0)b}{2a\bigl[\varepsilon_0 a+(\varepsilon_f-\varepsilon_0)x\bigr]^2}}$$ 方向：$+x$ 方向（誘電体がコンデンサ内に引き込まれる）
直感：$\varepsilon_f > \varepsilon_0$ なので誘電体を挿入するほど $C$ が増加 → エネルギーが減少（$Q$ 一定） → 自発的に挿入が進む。

2)　誘電率が $\varepsilon(x)=\varepsilon_1+\dfrac{\varepsilon_2-\varepsilon_1}{a}x$ のコンデンサの静電容量

誘電率が $x$ に依存するため，幅 $dx$ の無限小スライスを並列コンデンサとして積分： $$dC = \varepsilon(x)\frac{a\,dx}{b}, \quad C = \int_0^a \frac{a\,\varepsilon(x)}{b}\,dx$$ $$= \frac{a}{b}\int_0^a \left(\varepsilon_1 + \frac{\varepsilon_2-\varepsilon_1}{a}x\right)dx = \frac{a}{b}\left[\varepsilon_1 a + \frac{\varepsilon_2-\varepsilon_1}{a}\cdot\frac{a^2}{2}\right] = \frac{a^2}{b}\cdot\frac{\varepsilon_1+\varepsilon_2}{2}$$
$$\boxed{C = \frac{a^2(\varepsilon_1+\varepsilon_2)}{2b}}$$ （$\varepsilon_1$ と $\varepsilon_2$ の算術平均に対応する容量）

大問2　平行導体板間の体積電荷分布

導体 A（接地，$V=0$）：$x=-d/3$，導体 B（電荷 $=0$）：$x=2d/3$。 体積電荷：$-d/3

問1)　各導体板での電界境界条件（ガウスの法則による説明）

全電荷を確認： $$\rho_1\cdot\frac{d}{3} + \rho_2\cdot\frac{2d}{3} = (-2\rho_0)\cdot\frac{d}{3} + \rho_0\cdot\frac{2d}{3} = 0$$ 全体積電荷は零。ガウスの法則 $\displaystyle\oint_S \varepsilon_0\boldsymbol{E}\cdot d\boldsymbol{A} = Q_{\rm enc}$ を導体板 A から B を囲む閉曲面に適用： $$\varepsilon_0 E\!\left(\frac{2d}{3}\right) - \varepsilon_0 E\!\left(-\frac{d}{3}\right) = \frac{\text{全電荷}}{\text{面積}} = 0$$ 導体 B の電荷 $=0$ → ガウスの法則より $E(2d/3^-) = \sigma_B/\varepsilon_0 = 0$（表面電荷なし）。 上式に代入 → $E(-d/3^+) = 0$（導体 A も表面電荷なし，$\sigma_A=0$）。
$$x=-\frac{d}{3}:\quad E = 0 \quad \text{（導体 A の表面電荷はゼロ → 電界もゼロ）}$$ $$x=+\frac{2d}{3}:\quad E = 0 \quad \text{（導体 B の電荷がゼロ → 境界条件 } \varepsilon_0 E = \sigma_B = 0\text{）}$$

問2)　電位分布 $V(x)$，電界分布 $E(x)$，導体 B の電位 $V_B$

$\varepsilon_0 dE/dx = \rho(x)$ を積分（境界条件 $E(-d/3^+)=0$）： 領域 1（$-d/3 \le x \le 0$）： $$E_1(x) = \frac{1}{\varepsilon_0}\int_{-d/3}^{x}\rho_1\,dt = \frac{-2\rho_0}{\varepsilon_0}\left(x+\frac{d}{3}\right)$$ 領域 2（$0 \le x \le 2d/3$）： 連続条件 $E_2(0)=E_1(0)=-2\rho_0 d/(3\varepsilon_0)$ から： $$E_2(x) = \frac{\rho_0}{\varepsilon_0}(x-\frac{2d}{3})$$ 検証：$E_2(2d/3)=0$ ✓ 電位（$V(-d/3)=0$，$V=-\int E\,dx$）： 領域 1： $$V_1(x) = -\int_{-d/3}^{x}E_1\,dx = \frac{\rho_0}{\varepsilon_0}\left(x+\frac{d}{3}\right)^2$$ 領域 2（$V_2(0)=V_1(0)=\rho_0 d^2/(9\varepsilon_0)$）： $$V_2(x) = \frac{\rho_0 d^2}{9\varepsilon_0} - \frac{\rho_0}{\varepsilon_0}\left(\frac{x^2}{2}-\frac{2d}{3}x\right)$$ 導体 B の電位： $$V_B = V_2\!\left(\frac{2d}{3}\right) = \frac{\rho_0 d^2}{9\varepsilon_0} - \frac{\rho_0}{\varepsilon_0}\left(\frac{2d^2}{9}-\frac{4d^2}{9}\right) = \frac{\rho_0 d^2}{9\varepsilon_0} + \frac{2\rho_0 d^2}{9\varepsilon_0} = \frac{\rho_0 d^2}{3\varepsilon_0}$$
$$E(x) = \begin{cases} \dfrac{-2\rho_0}{\varepsilon_0}\!\left(x+\dfrac{d}{3}\right) & \left(-\dfrac{d}{3}\le x\le 0\right) \\[8pt] \dfrac{\rho_0}{\varepsilon_0}\!\left(x-\dfrac{2d}{3}\right) & \left(0\le x\le\dfrac{2d}{3}\right)\end{cases}$$ $$V(x) = \begin{cases} \dfrac{\rho_0}{\varepsilon_0}\!\left(x+\dfrac{d}{3}\right)^2 & \left(-\dfrac{d}{3}\le x\le 0\right) \\[8pt] \dfrac{\rho_0 d^2}{9\varepsilon_0} - \dfrac{\rho_0}{\varepsilon_0}\!\left(\dfrac{x^2}{2}-\dfrac{2d}{3}x\right) & \left(0\le x\le\dfrac{2d}{3}\right)\end{cases}$$ $$\boxed{V_B = \frac{\rho_0 d^2}{3\varepsilon_0}}$$

問3)　導体板 A の単位面積当たりの電荷密度

境界条件より $E(-d/3^+)=0$、したがって $\varepsilon_0 E(-d/3^+) = \sigma_A$： $$\boxed{\sigma_A = 0}$$ （両体積電荷の正負が完全に打ち消し合い，導体表面に電荷は誘起されない）

問4)　電荷境界が $x=a$（$-d/3 < a < 2d/3$）のときの条件

境界条件が成立（$E(-d/3^+)=0$ かつ $E(2d/3^-)=0$）するには，ガウスの法則より全体積電荷が零： $$\rho_1 \times\left(a-\left(-\frac{d}{3}\right)\right) + \rho_2 \times\left(\frac{2d}{3}-a\right) = 0$$
$$\boxed{\rho_1\!\left(a+\frac{d}{3}\right) + \rho_2\!\left(\frac{2d}{3}-a\right) = 0}$$ または：$\rho_1(3a+d) + \rho_2(2d-3a) = 0$

大問3　無限長直線状導線と矩形コイル

無限長直線状導線（$y$ 軸上，電流 $I_1$ 上向き）と矩形コイル（$2a \times b$，中心 $x=R$）が $x$-$y$ 平面上にある。$R > a$。

1-a)　距離 $x$ での磁束密度 $B(x)$

アンペールの法則（$\oint B\,dl = \mu_0 I_{\rm enc}$）で半径 $x$ の円周： $$B(x) \cdot 2\pi x = \mu_0 I_1$$
$$\boxed{B(x) = \frac{\mu_0 I_1}{2\pi x}}$$ （$I_1$ の向きに対して右ねじ方向、コイル面法線 = $-z$ 方向）

1-b)　相互インダクタンス $M$

コイルは $x = R-a$ から $x = R+a$ にわたり，幅 $dx$ のストリップを積分： $$\Phi = \int_{R-a}^{R+a}\! B(x)\cdot b\,dx = \frac{\mu_0 I_1 b}{2\pi}\int_{R-a}^{R+a}\!\frac{dx}{x} = \frac{\mu_0 I_1 b}{2\pi}\ln\frac{R+a}{R-a}$$
$$\boxed{M = \frac{\Phi}{I_1} = \frac{\mu_0 b}{2\pi}\ln\frac{R+a}{R-a}}$$

1-c)　導線とコイルの間に働く力 $F$

電流一定での仮想変位法（または各辺への力の直接計算）： $$F = I_1 I_2\frac{dM}{dR}$$ $$\frac{dM}{dR} = \frac{\mu_0 b}{2\pi}\left(\frac{1}{R+a}-\frac{1}{R-a}\right) = \frac{\mu_0 b}{2\pi}\cdot\frac{-2a}{R^2-a^2}$$ $$F = I_1 I_2 \cdot \frac{-\mu_0 ab}{\pi(R^2-a^2)}$$ $I_1 I_2 > 0$ なら $F < 0$（引力，コイルは導線に近づく向き）。
$$\boxed{F = \frac{\mu_0 a b I_1 I_2}{\pi(R^2-a^2)}}$$ 向き：図のような電流方向（$I_1 I_2 > 0$）なら引力（導線方向，$-x$ 方向）。

2-a)　コイルを角度 $\theta$ 回転させたときの鎖交磁束 $\Phi(\theta)$

回転軸：$x=R$ の $y$ 軸に平行な直線。コイルを $\theta$ 回転後： - 左辺（もとの $x=R-a$）の導線からの距離：$r_{\rm left} = \sqrt{R^2-2aR\cos\theta+a^2}$ - 右辺（もとの $x=R+a$）の距離：$r_{\rm right} = \sqrt{R^2+2aR\cos\theta+a^2}$ （余弦定理 $C^2=A^2+B^2-2AB\cos\gamma$ に対応：$A=R$，$B=a$，$\gamma=\theta$ および $\pi-\theta$） ベクトルポテンシャル法（$A_y = -\mu_0 I_1/(2\pi)\ln r$）による鎖交磁束： $$\Phi(\theta) = \oint \boldsymbol{A}\cdot d\boldsymbol{l}$$ 導線に平行な辺のみ寄与（上辺・下辺は $A_y$ に直交するので寄与なし）： $$\Phi(\theta) = A_y(r_{\rm left})\cdot b - A_y(r_{\rm right})\cdot b = \frac{\mu_0 I_1 b}{2\pi}\ln\frac{r_{\rm right}}{r_{\rm left}}$$
$$\boxed{\Phi(\theta) = \frac{\mu_0 I_1 b}{4\pi}\ln\frac{R^2+a^2+2aR\cos\theta}{R^2+a^2-2aR\cos\theta}}$$ 確認：$\theta=0$→$\Phi=(\mu_0 I_1 b/2\pi)\ln[(R+a)/(R-a)]$ ✓；$\theta=\pi/2$→$\Phi=0$（コイル面が $B$ と平行）✓

2-b)　角周波数 $\omega$ で回転したときの誘導起電力 $V$

$\theta = \omega t$ として $V = -d\Phi/dt = -\omega\,d\Phi/d\theta$： $$\frac{d\Phi}{d\theta} = \frac{\mu_0 I_1 b}{4\pi}\cdot\frac{d}{d\theta}\ln\frac{P}{Q}, \quad P=R^2+a^2+2aR\cos\theta,\quad Q=R^2+a^2-2aR\cos\theta$$ $$= \frac{\mu_0 I_1 b}{4\pi}\left(\frac{-2aR\sin\theta}{P} - \frac{2aR\sin\theta}{Q}\right) = \frac{\mu_0 I_1 b}{4\pi}\cdot(-2aR\sin\theta)\cdot\frac{P+Q}{PQ}$$ $P+Q = 2(R^2+a^2)$，$PQ=(R^2+a^2)^2-4a^2R^2\cos^2\theta$： $$\frac{d\Phi}{d\theta} = -\frac{\mu_0 I_1 b \cdot 2aR(R^2+a^2)\sin\theta}{\pi\bigl[(R^2+a^2)^2-4a^2R^2\cos^2\theta\bigr]}$$
$$\boxed{V = \frac{\mu_0 I_1 b\omega \cdot 2aR(R^2+a^2)\sin(\omega t)}{\pi\bigl[(R^2+a^2)^2-4a^2R^2\cos^2(\omega t)\bigr]}}$$ 最大値（$\theta=\pi/2$）：$V_{\max}=\dfrac{2\mu_0 I_1 b\omega aR}{\pi(R^2+a^2)}$
