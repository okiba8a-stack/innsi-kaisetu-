東京工業大学 大学院修士課程入試（平成22年8月17日実施）

大問1　回転式可変コンデンサ

2枚の同形電極板を向かい合わせた可変コンデンサを考える。各電極は 中心から半径 $a$ の円板に、半径 $a$〜$5a$・中心角 $\pi/2$ の扇形 を付加した形状（同一中心軸）を持つ。電極間距離は $d=a$、誘電率は $\varepsilon_0$ とする。 一方の電極を角度 $\theta$ だけ回転させたときの静電容量 $C(\theta)$ を求める。

問(1)　$\theta=0$ における静電容量 $C_0$

各電極の面積を分割して計算する。
  • 中心円板（半径 $a$）: $S_1 = \pi a^2$
  • 環状扇形（半径 $a$〜$5a$、角度 $\pi/2$）: $S_2 = \dfrac{\pi/2}{2}\bigl[(5a)^2 - a^2\bigr] = \dfrac{\pi}{4}\cdot 24a^2 = 6\pi a^2$
全面積 $S = S_1 + S_2 = 7\pi a^2$
$$\boxed{C_0 = \frac{\varepsilon_0 \cdot 7\pi a^2}{a} = 7\pi\varepsilon_0 a}$$

問(2)　静電容量 $C(\theta)$（$0 \le \theta \le \pi/2$）

角度 $\theta$ 回転させると：
  • 中心円板は常に重なる: 重なり面積 $= \pi a^2$（不変）
  • 扇形部分の重なり角度: $\pi/2 - \theta$
    重なり面積 $= \dfrac{\pi/2-\theta}{2}\cdot 24a^2 = 12a^2\!\left(\dfrac{\pi}{2}-\theta\right)$
全重なり面積 $= \pi a^2 + 12a^2\!\left(\dfrac{\pi}{2}-\theta\right) = a^2(7\pi - 12\theta)$
$$\boxed{C(\theta) = \varepsilon_0 a\,(7\pi - 12\theta)} \quad (0 \le \theta \le \tfrac{\pi}{2})$$ 扇形が完全にずれた後（$\pi/2 \le \theta \le 3\pi/2$）は、中心円板のみ重なるため： $$C_{\min} = \pi\varepsilon_0 a$$
$C(\theta)$ は $\theta$ に対して線形に減少する。最大値 $C_{\max}=C_0=7\pi\varepsilon_0 a$、最小値 $C_{\min}=\pi\varepsilon_0 a$。 比率 $C_{\max}/C_{\min} = 7$。

問(3)　電荷 $Q$ 一定のときのトルク $\tau$

電荷 $Q$ を固定した条件でのエネルギーは $U = \dfrac{Q^2}{2C(\theta)}$。
仮想仕事の原理より： $$\tau = -\left(\frac{\partial U}{\partial \theta}\right)_Q = -\frac{Q^2}{2}\cdot\frac{d}{d\theta}\!\left(\frac{1}{C}\right) = \frac{Q^2}{2C^2}\frac{dC}{d\theta}$$ $0 < \theta < \pi/2$ では $\dfrac{dC}{d\theta} = -12\varepsilon_0 a$ なので：
$$\boxed{\tau = \frac{Q^2}{2C^2}\cdot(-12\varepsilon_0 a) = -\frac{6Q^2\varepsilon_0 a}{C^2}}$$ 負号は $\theta$ が増加する向きに対して電極を引き戻す方向（$\theta$ 減少方向）にトルクが働くことを示す。

問(4)　電圧 $V(\theta)$

電荷 $Q$ が一定のとき： $$\boxed{V(\theta) = \frac{Q}{C(\theta)} = \frac{Q}{\varepsilon_0 a\,(7\pi - 12\theta)}}$$ $\theta$ が増加するにつれて $C$ が減少し、電圧 $V$ が増大する。$\theta=\pi/2$ で $V$ は最大 $Q/(\pi\varepsilon_0 a)$ となる。

問(5)　$\theta = 0$ から $\theta = \pi$ まで回転するときのエネルギー変化

$\theta=0$: $C_0 = 7\pi\varepsilon_0 a$、$\quad U_0 = \dfrac{Q^2}{2\cdot 7\pi\varepsilon_0 a} = \dfrac{Q^2}{14\pi\varepsilon_0 a}$

$\theta=\pi$: $C(\pi) = C_{\min} = \pi\varepsilon_0 a$（扇形は $\pi > \pi/2$ なので重ならない）、$\quad U_\pi = \dfrac{Q^2}{2\pi\varepsilon_0 a}$

差： $$\Delta U = U_\pi - U_0 = \frac{Q^2}{2\pi\varepsilon_0 a} - \frac{Q^2}{14\pi\varepsilon_0 a} = \frac{Q^2}{\pi\varepsilon_0 a}\!\left(\frac{1}{2}-\frac{1}{14}\right) = \frac{Q^2}{\pi\varepsilon_0 a}\cdot\frac{6}{14}$$
$$\boxed{\Delta U = \frac{3Q^2}{7\pi\varepsilon_0 a}}$$ エネルギーが増加する（回転させるために外力が仕事をする必要がある）。

問(6)〜(8)　チャージポンプ回路

以下の回路を考える：コンデンサ $C(\theta)$ の下側電極は電圧源 $V_1$ に接続、 上側電極（ノードP）はダイオード $D_1$（アノードをGND、カソードをPに接続）で初期充電され、 $D_2$（カソードをP、アノードを $-V_2$ に接続）とスイッチ $S$ を通じて $V_2$ 源に接続される。

問(6)　$\theta$ 増加時のノードP電圧 $V_p(\theta)$

初期状態（$\theta=0$, S開放, $D_1$ が導通）： $$V_p = 0,\quad Q = C_0 V_1 = 7\pi\varepsilon_0 a \cdot V_1$$ S を開放して $\theta$ を増加させると、電荷 $Q=C_0 V_1$ が保存される。
下側電極は $V_1$ に保たれたままなので： $$V_p = V_1 - \frac{Q}{C(\theta)} = V_1 - \frac{C_0 V_1}{C(\theta)} = V_1\!\left(1 - \frac{C_0}{C(\theta)}\right)$$
$$\boxed{V_p(\theta) = V_1\!\left(1 - \frac{C_0}{C(\theta)}\right)} \quad (0 \le \theta \le \tfrac{\pi}{2})$$ $\theta = 0$ で $V_p = 0$、$\theta$ 増加とともに $V_p$ は負になる。

問(7)　$V_p$ の最小値と $D_2$ が導通する条件

$\theta = \pi/2$ で $C = C_{\min} = \pi\varepsilon_0 a$： $$V_{p,\min} = V_1\!\left(1 - \frac{7\pi\varepsilon_0 a}{\pi\varepsilon_0 a}\right) = V_1(1-7) = -6V_1$$
$$\boxed{V_{p,\min} = -6V_1}$$ $D_2$ が導通する条件（$|V_{p,\min}| > V_2$）： $$\boxed{6V_1 > V_2}$$

問(8)　$V_2$ 源への移動電荷量

$\theta = \pi/2$ でスイッチ $S$ を閉じると、$D_2$ が導通して電荷が $V_2$ 源に移動する。
平衡状態では $V_p = -V_2$ となる（$D_2$ の順方向降下電圧を無視）。

移動後の容量はまだ $C_{\min} = \pi\varepsilon_0 a$（$\theta$ 固定）。
移動後のコンデンサ電荷： $$Q_f = C_{\min}(V_1 - V_{p,f}) = \pi\varepsilon_0 a\,(V_1 + V_2)$$ 移動電荷量： $$\Delta Q = Q - Q_f = 7\pi\varepsilon_0 a\cdot V_1 - \pi\varepsilon_0 a(V_1+V_2) = \pi\varepsilon_0 a\,(6V_1 - V_2)$$
$$\boxed{\Delta Q = \pi\varepsilon_0 a\,(6V_1 - V_2)}$$ $V_2$ 源に供給されるエネルギー：$W = \Delta Q \cdot V_2 = \pi\varepsilon_0 a\,V_2(6V_1-V_2)$
これはチャージポンプの原理：機械的な回転エネルギーを電気エネルギーに変換し、低電圧源 $V_1$ から高電圧 $6V_1$ で $V_2$ 源を充電する。最大効率の出力は $V_2 = 3V_1$（インピーダンス整合的な最大電力条件）。

大問2　半円弧電流の磁界と電磁誘導

$xy$ 平面上の半径 $a$ の半円弧（$y \ge 0$）と、$x$ 軸上の 2 本の半無限直線電流（$x \le -a$ と $x \ge a$）が接続されてD字形の電流ループを形成し、電流 $I$ が流れている（半円の向きは $x$ 軸正方向から半円に沿って $x$ 軸負方向へ流れる方向とする）。

問(1)　直線電流が原点に作る磁界

$x$ 軸上の電流素子 $I\,dl = I\,dx\,\hat{x}$、原点への変位ベクトルは $\vec{r} = -x\hat{x}$（$\hat{r} = -\hat{x}$）。
ビオ・サバールの法則： $$d\vec{B} = \frac{\mu_0 I}{4\pi}\frac{d\vec{l}\times\hat{r}}{r^2} = \frac{\mu_0 I}{4\pi}\frac{dx\,\hat{x}\times(-\hat{x})}{r^2} = 0$$ 電流素子と変位ベクトルが平行なため外積はゼロ。
原点での直線電流の寄与：$\vec{B}=0$

問(2)　原点での全磁束密度 $\vec{B}_0$（半円弧の寄与）

半円弧を $\vec{r}' = (a\cos\varphi,\, a\sin\varphi,\, 0)$（$0 \le \varphi \le \pi$）で parametrize。
電流素子: $d\vec{l} = a(-\sin\varphi,\, \cos\varphi,\, 0)\,d\varphi$
原点への変位: $\vec{r} = -\vec{r}' = (-a\cos\varphi,\,-a\sin\varphi,\,0)$、$|\vec{r}| = a$

$$d\vec{l}\times\hat{r} = a\,d\varphi\,(-\sin\varphi,\cos\varphi,0)\times(-\cos\varphi,-\sin\varphi,0)$$ $$= a\,d\varphi\left[\begin{vmatrix}\hat{y}&\hat{z}\\\cos\varphi&0\end{vmatrix}\hat{x} - \cdots\right] = a\,d\varphi\,(0,0,\sin^2\varphi+\cos^2\varphi) = a\,d\varphi\,\hat{z}$$ よって： $$\vec{B}_0 = \frac{\mu_0 I}{4\pi}\int_0^\pi \frac{a\,d\varphi}{a^2}\hat{z} = \frac{\mu_0 I}{4\pi a}\cdot\pi\,\hat{z}$$
$$\boxed{\vec{B}_0 = \frac{\mu_0 I}{4a}\,\hat{z}}$$ （全円の $\mu_0 I/(2a)$ の半分）

問(3)　$y$ 軸まわりに回転するコイルの起電力 $e_1$

面積 $S$ の平面コイルが $y$ 軸まわりに角速度 $\omega$ で回転（初期位置: コイル面が $xz$ 平面）。
角度 $\theta = \omega t$ として磁束： $$\Phi = B_0 S \cos\theta = \frac{\mu_0 I S}{4a}\cos(\omega t)$$ 誘導起電力： $$e_1 = -\frac{d\Phi}{dt} = \frac{\mu_0 I S \omega}{4a}\sin(\omega t)$$
$$\boxed{e_1(t) = \frac{\mu_0 I S \omega}{4a}\sin(\omega t), \quad E_m = \frac{\mu_0 I S \omega}{4a}}$$ $e_1 = 0$ となるのは $\omega t = 0, \pi, 2\pi, \ldots$（コイル面が $xz$ 平面、磁束が最大となる位置）。

問(4a)　点 $\mathrm{P}(0,0,b)$ での直線電流による磁束密度 $\vec{B}_\ell$

$x$ 軸上 $x \ge a$ の半無限直線電流（$+x$ 方向）が点 P に作る磁界。
電流素子: $d\vec{l} = dx\,\hat{x}$、P への変位: $\vec{r} = (0,0,b)-(x,0,0) = (-x,0,b)$、$|\vec{r}|=\sqrt{x^2+b^2}$

$$d\vec{l}\times\vec{r} = dx\,\hat{x}\times(-x,0,b) = dx\,(0\cdot b - 0,\;0\cdot(-x)-b,\;0) = dx\,(0,-b,0)$$ $$\vec{B}_1 = \frac{\mu_0 I}{4\pi}\int_a^\infty \frac{-b\,\hat{y}}{(x^2+b^2)^{3/2}}dx$$ 積分：$\displaystyle\int_a^\infty \frac{dx}{(x^2+b^2)^{3/2}} = \left[\frac{x}{b^2\sqrt{x^2+b^2}}\right]_a^\infty = \frac{1}{b^2}\!\left(1-\frac{a}{\sqrt{a^2+b^2}}\right)$

$x \le -a$ の半無限直線（$-x$ 方向）も対称的に同じ $-\hat{y}$ 方向の寄与を与える。合計：
$$\boxed{\vec{B}_\ell = \frac{\mu_0 I}{2\pi b}\!\left(1-\frac{a}{\sqrt{a^2+b^2}}\right)(-\hat{y})}$$ （両側の半無限線電流の合計）

問(4b)　点 $\mathrm{P}(0,0,b)$ での半円弧による磁束密度 $\vec{B}_r$

半円弧のパラメータ表示と同様に計算：
$\vec{r}' = (a\cos\varphi,a\sin\varphi,0)$、$d\vec{l} = a(-\sin\varphi,\cos\varphi,0)d\varphi$
P への変位: $\vec{r} = (-a\cos\varphi,-a\sin\varphi,b)$、$|\vec{r}| = \sqrt{a^2+b^2}$

$$d\vec{l}\times\vec{r} = a\,d\varphi\,(-\sin\varphi,\cos\varphi,0)\times(-a\cos\varphi,-a\sin\varphi,b)$$ $$= a\,d\varphi\,(b\cos\varphi,\; b\sin\varphi,\; a\sin^2\varphi+a\cos^2\varphi)$$ $$= a\,d\varphi\,(b\cos\varphi,\; b\sin\varphi,\; a)$$ 各成分を $0$ から $\pi$ で積分： $$\int_0^\pi\cos\varphi\,d\varphi = 0,\quad \int_0^\pi\sin\varphi\,d\varphi = 2,\quad \int_0^\pi d\varphi = \pi$$
$$\vec{B}_r = \frac{\mu_0 I}{4\pi}\cdot\frac{a}{(a^2+b^2)^{3/2}}\cdot(0,\;2ab,\;\pi a^2)\cdot\frac{1}{a}$$ $$\boxed{\vec{B}_r = \frac{\mu_0 I a}{4\pi(a^2+b^2)^{3/2}}\bigl(0,\;2b,\;\pi a\bigr)}$$
  • $y$ 成分: $B_{r,y} = \dfrac{\mu_0 I ab}{2\pi(a^2+b^2)^{3/2}}$（$+\hat{y}$ 方向）
  • $z$ 成分: $B_{r,z} = \dfrac{\mu_0 I a^2}{4(a^2+b^2)^{3/2}}$（$+\hat{z}$ 方向）
$b=0$ とすると $B_{r,z} = \mu_0 I/(4a)$ となり問(2)の結果と一致する。
$\vec{B}_\ell$ の $y$ 成分（負）と $\vec{B}_r$ の $y$ 成分（正）は部分的に打ち消し合う。これはP点が $x$ 軸の延長上にあるため、直線電流成分が半円の反作用方向に磁界を作るためである。

問(5)　$z$ 方向に移動するコイルの起電力 $e_2(t)$

コイル（面積 $S$、コイル面が $xy$ 平面と平行）が $z$ 方向に速度 $v$ で移動：$z(t) = vt$。
コイルを貫く磁束は $z$ 成分のみ（コイル法線 $= \hat{z}$）： $$\Phi(t) = B_{r,z}(z=vt)\cdot S = \frac{\mu_0 I a^2 S}{4\bigl(a^2 + v^2t^2\bigr)^{3/2}}$$ 誘導起電力： $$e_2 = -\frac{d\Phi}{dt} = -\frac{\mu_0 I a^2 S}{4}\cdot\frac{d}{dt}\bigl(a^2+v^2t^2\bigr)^{-3/2}$$ $$= -\frac{\mu_0 I a^2 S}{4}\cdot\left(-\frac{3}{2}\right)\bigl(a^2+v^2t^2\bigr)^{-5/2}\cdot 2v^2t$$
$$\boxed{e_2(t) = \frac{3\mu_0 I S a^2 v^2 t}{4(a^2+v^2t^2)^{5/2}}}$$ $t=0$（コイルが半円の中心位置）で $e_2=0$（磁束が最大で変化率ゼロ）。
$t \to \infty$ で $e_2 \to 0$（コイルが無限遠に離れると磁場は弱まる）。
