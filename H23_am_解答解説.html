H23年度 電気回路 午前 解答解説

平成22年8月17日実施 ／ 電気回路（単独科目）

大問1：リアクタンス2ポート回路と最大電力整合

問1）最大電力伝送の証明

回路：開放電圧 $E$、電源インピーダンス $R$（正の実数）の交流電源 + 負荷 $Z_L = R_L + jX_L$（$R_L > 0$）。
電流：$I = \dfrac{E}{R + Z_L} = \dfrac{E}{(R+R_L) + jX_L}$
負荷消費電力： $$P = |I|^2 R_L = \frac{|E|^2 R_L}{(R+R_L)^2 + X_L^2}$$ $X_L$ について最大化：分母を最小化するには $X_L = 0$： $$P = \frac{|E|^2 R_L}{(R+R_L)^2}$$ $R_L$ について最大化（AM-GM 不等式または微分）： $$\frac{dP}{dR_L} = |E|^2 \frac{(R+R_L)^2 - 2R_L(R+R_L)}{(R+R_L)^4} = 0 \implies R+R_L = 2R_L \implies R_L = R$$
$R_L = R$，$X_L = 0$ のとき $P$ 最大： $$P_{\max} = \frac{|E|^2 R}{(2R)^2} = \frac{|E|^2}{4R}, \quad I = \frac{E}{2R}$$

問2）対称2ポート回路 $[Z]$ の入力インピーダンス $Z_e,\,Z_o$

対称2ポートのインピーダンス行列：$[Z] = j\begin{pmatrix} X_{11} & X_{12} \\ X_{12} & X_{11} \end{pmatrix}$（純リアクタンス）。
対称励振（図1.2(b)→(d)）：両ポートから同方向に電流 $I_e$ が流入（$I_1 = I_2 = I_e$）。
$$V_1 = jX_{11}I_e + jX_{12}I_e = j(X_{11}+X_{12})I_e$$ 等価1ポートの入力インピーダンス（$[Z]$ の部分）： $$Z_e = jX_e = j(X_{11}+X_{12})$$ $$\boxed{X_e = X_{11}+X_{12}}$$
反対称励振（図1.2(c)→(e)）：$I_1 = I_o$，$I_2 = -I_o$（逆向き）。
$$V_1 = jX_{11}I_o + jX_{12}(-I_o) = j(X_{11}-X_{12})I_o$$ $$Z_o = jX_o = j(X_{11}-X_{12})$$ $$\boxed{X_o = X_{11}-X_{12}}$$

問3）最大電力条件 $Z_eZ_o = -X_eX_o = R^2$ の証明

重ね合わせ原理より：$I_1 = I_e + I_o$，$I_2 = I_e - I_o$（負荷電流 $I_2$ が最大のとき最大電力）。
各励振の電流（図1.2(d)(e)の等価回路から）： $$I_e = \frac{E/2}{R + jX_e}, \quad I_o = \frac{E/2}{R + jX_o}$$ 入力電流： $$I_1 = I_e + I_o = \frac{E}{2}\left[\frac{1}{R+jX_e} + \frac{1}{R+jX_o}\right] = \frac{E}{2}\cdot\frac{2R + j(X_e+X_o)}{(R+jX_e)(R+jX_o)}$$
問1）の結果：$|I_2|$ 最大（最大電力）の必要条件は $I_1 = E/(2R)$（1ポートで $R_L = R$ に相当）。よって： $$\frac{E}{2}\cdot\frac{2R + j(X_e+X_o)}{(R+jX_e)(R+jX_o)} = \frac{E}{2R}$$ 整理：$(R+jX_e)(R+jX_o) = R[2R + j(X_e+X_o)]$
左辺：$R^2 + jR(X_e+X_o) - X_eX_o$
右辺：$2R^2 + jR(X_e+X_o)$
虚部は自動的に一致。実部から： $$R^2 - X_eX_o = 2R^2 \implies -X_eX_o = R^2$$ $Z_e = jX_e$，$Z_o = jX_o$ なので： $$Z_eZ_o = (jX_e)(jX_o) = -X_eX_o = R^2 \qquad \blacksquare$$
$$\boxed{Z_eZ_o = -X_eX_o = R^2}$$

問4）図1.4の磁気結合コイル＋コンデンサ回路の $Z_e,\,Z_o$

図1.4の2ポート回路：各ポートにコンデンサ $C$ を直列，コイル $L_1 = L_2 = L$（相互インダクタンス $M = \sqrt{L_1 L_2} = L$）が中央に接続（完全結合）。

対称励振（$I_1 = I_2 = I_e$，同方向流入）：
磁気結合コイルの電圧：$V_{L} = j\omega L I_e + j\omega M I_e = j\omega(L+M)I_e$
コンデンサの電圧：$\dfrac{I_e}{j\omega C}$
各ポートの入力インピーダンス（$[Z]$ 部分）： $$Z_e = \frac{1}{j\omega C} + j\omega(L+M) = j\left[\omega(L+M) - \frac{1}{\omega C}\right]$$ $M = L$ を代入： $$\boxed{Z_e = j\!\left[2\omega L - \frac{1}{\omega C}\right]}$$
反対称励振（$I_1 = I_o$，$I_2 = -I_o$）：
磁気結合コイルの電圧：$V_{L} = j\omega L I_o + j\omega M(-I_o) = j\omega(L-M)I_o$
$M = L$ なので $j\omega(L-L) = 0$（コイルが等価的に消える）： $$Z_o = \frac{1}{j\omega C} + 0 = -\frac{j}{\omega C}$$ $$\boxed{Z_o = -\frac{j}{\omega C}}$$

問5）$C = 1/(\omega^2 M)$ で最大電力になることの証明

条件 $-X_eX_o = R^2$（ただし $R = \omega M = \omega L$）を検証。
$X_e = 2\omega L - \dfrac{1}{\omega C}$，$X_o = -\dfrac{1}{\omega C}$： $$-X_eX_o = \left(2\omega L - \frac{1}{\omega C}\right)\cdot\frac{1}{\omega C}$$ $C = 1/(\omega^2 M) = 1/(\omega^2 L)$ を代入 → $1/(\omega C) = \omega L$： $$-X_eX_o = \left(2\omega L - \omega L\right)\cdot\omega L = \omega L \cdot \omega L = \omega^2 L^2 = R^2 \qquad \blacksquare$$
別証（代数的）：$-X_eX_o = R^2$ を $C$ について解く。
$-\!\left(2\omega L - \frac{1}{\omega C}\right)\!\left(-\frac{1}{\omega C}\right) = \omega^2 L^2$
$\frac{1}{\omega C}\!\left(2\omega L - \frac{1}{\omega C}\right) = \omega^2 L^2$
$u = 1/(\omega C)$ とおくと：$u(2\omega L - u) = \omega^2 L^2$
$2\omega L u - u^2 = \omega^2 L^2$，すなわち $(u - \omega L)^2 = 0$ → $u = \omega L$
よって $C = 1/(\omega L) \cdot 1/\omega = 1/(\omega^2 L) = 1/(\omega^2 M)$。
$C = \dfrac{1}{\omega^2 M}$ のとき $-X_eX_o = R^2$ が成立し，$R$ への供給電力が最大となる。

大問2：無損失平行線路（分布定数回路）

パラメータ：単位長さ当たりインダクタンス $L$ [H/m]，静電容量 $C$ [F/m]。時間依存 $e^{j\omega t}$。特性インピーダンス $Z_0 = \sqrt{L/C}$，位相定数 $\beta = \omega\sqrt{LC}$。

問1）半無限長平行線路

a) $\Delta V$：微小区間 $\Delta x$ の直列インダクタンス $L\Delta x$ による電圧降下： $$\Delta V = V(x+\Delta x) - V(x) = -j\omega L\Delta x \cdot I(x)$$
b) $\Delta I$：並列コンデンサ $C\Delta x$ に流れ込む電流： $$\Delta I = I(x+\Delta x) - I(x) = -j\omega C\Delta x \cdot V(x)$$
c) $V(x)$ に関する微分方程式：$\Delta x \to 0$ で： $$\frac{dV}{dx} = -j\omega L I, \quad \frac{dI}{dx} = -j\omega C V$$ 第1式を $x$ で微分し第2式を代入： $$\frac{d^2V}{dx^2} = -j\omega L \frac{dI}{dx} = -j\omega L(-j\omega C V) = -\omega^2 LC \cdot V$$ $$\boxed{\frac{d^2V}{dx^2} + \beta^2 V = 0, \quad \beta = \omega\sqrt{LC}}$$
d) 一般解：特性根 $\pm j\beta$： $$V(x) = V^+ e^{-j\beta x} + V^- e^{+j\beta x}$$ 半無限長（$x \to \infty$ で有界）→ 右向き進行波のみ（$V^- = 0$）： $$V(x) = V^+ e^{-j\beta x}, \quad I(x) = \frac{V^+}{Z_0}e^{-j\beta x}$$
e) 波長：$\beta = 2\pi/\lambda$： $$\lambda = \frac{2\pi}{\beta} = \frac{2\pi}{\omega\sqrt{LC}}$$

問2）有限長 $\ell$，負荷 $Z_L$ で終端 — $Z|_{x=0}$ の導出

一般解：$V(x) = V^+e^{-j\beta x} + V^-e^{+j\beta x}$，$I(x) = \dfrac{1}{Z_0}(V^+e^{-j\beta x} - V^-e^{+j\beta x})$
境界条件 $x = \ell$：$V(\ell)/I(\ell) = Z_L$ $$\frac{V^+e^{-j\beta\ell} + V^-e^{j\beta\ell}}{V^+e^{-j\beta\ell} - V^-e^{j\beta\ell}} = \frac{Z_L}{Z_0}$$ 反射係数（$x=\ell$ で）：$\Gamma_L = \dfrac{Z_L - Z_0}{Z_L + Z_0}$，$V^- = V^+ e^{-j2\beta\ell}\Gamma_L$
$x=0$ でのインピーダンス（標準変換式）： $$Z|_{x=0} = Z_0\frac{Z_L + jZ_0\tan(\beta\ell)}{Z_0 + jZ_L\tan(\beta\ell)}$$
空欄を埋めると：$\text{ア}=Z_L$，$\text{イ}=\beta\ell$，$\text{ウ}=Z_0$，$\text{エ}=Z_L$，$\text{オ}=\beta\ell$ $$Z|_{x=0} = Z_0\cdot\frac{Z_L + jZ_0\tan(\beta\ell)}{Z_0 + jZ_L\tan(\beta\ell)}$$

問3）短絡終端 $Z_L = 0$

a) $Z|_{x=0}$（一般）：$Z_L = 0$ を代入： $$Z|_{x=0} = Z_0\cdot\frac{0 + jZ_0\tan(\beta\ell)}{Z_0 + 0} = jZ_0\tan(\beta\ell)$$ $\ell \ll \lambda$（$\beta\ell \ll \pi/2$，$\tan(\beta\ell) \approx \beta\ell$）： $$Z \approx jZ_0\beta\ell = j\sqrt{\frac{L}{C}}\cdot\omega\sqrt{LC}\cdot\ell = j\omega L\ell$$ → インダクタ，素子値 $L_{\rm eff} = L\ell$ [H]
c) $\ell = \lambda/4$：$\beta\ell = \pi/2$，$\tan(\pi/2) \to \infty$： $$Z|_{x=0} = \lim_{\tan(\beta\ell)\to\infty} Z_0\cdot\frac{jZ_0\tan(\beta\ell)}{jZ_0\cdot 0 + ... } \to \infty$$ → 開放（$Z = \infty$）

問4）開放終端 $Z_L = \infty$

a) 一般：$Z_L \to \infty$ で分子・分母を $Z_L$ で割る： $$Z|_{x=0} = Z_0\cdot\frac{1 + j(Z_0/Z_L)\tan(\beta\ell)}{Z_0/Z_L + j\tan(\beta\ell)} \xrightarrow{Z_L\to\infty} \frac{Z_0}{j\tan(\beta\ell)} = -jZ_0\cot(\beta\ell)$$ $\ell \ll \lambda$（$\cot(\beta\ell) \approx 1/(\beta\ell)$）： $$Z \approx -j\frac{Z_0}{\beta\ell} = -j\frac{\sqrt{L/C}}{\omega\sqrt{LC}\cdot\ell} = -\frac{j}{\omega C\ell} = \frac{1}{j\omega C\ell}$$ → コンデンサ，素子値 $C_{\rm eff} = C\ell$ [F]
b) $\ell = \lambda/4$：$\cot(\pi/2) = 0$： $$Z|_{x=0} = -jZ_0 \cdot 0 = 0$$ → 短絡（$Z = 0$）

問5）分岐スタブ（$\lambda_1/4$ 開放スタブ）の周波数特性

図2.3の構成：波長に対して十分長い主線路の途中に，問4)b)の「開放端 $\lambda_1/4$ 線路（スタブ）」を並列に分岐。$Z_L = Z_S = Z_0$（主線路の特性インピーダンスに等しい）。
スタブのインピーダンス：$Z_{\rm stub} = -jZ_0\cot(\beta\ell_1) = -jZ_0\cot\!\left(\dfrac{\omega}{\omega_1}\cdot\dfrac{\pi}{2}\right)$
（$\ell_1 = \lambda_1/4 = \pi/(2\beta_1)$，$\beta = \omega/u$）
  • $\omega = \omega_1$：$\beta\ell_1 = \pi/2$ → $\cot(\pi/2) = 0$ → $Z_{\rm stub} = 0$（短絡）
    スタブが分岐点を短絡 → $Z_L$ 両端の電圧がゼロ → 信号は遮断される
  • $\omega \ne \omega_1$：$Z_{\rm stub} \ne 0$（有限リアクタンス）→ 主線路に影響するが完全遮断ではない
分岐した平行線路は バンドリジェクションフィルタ（ノッチフィルタ） として働く。
特定の周波数 $\omega_1$ の信号成分を主線路から取り除き（$Z_L$ 両端電圧 = 0），他の周波数帯は通過させる。
この構成は「スタブ整合（stub tuner）」の原理と同じで，$\lambda/4$ 開放スタブは並列に置くと特定周波数で短絡接地を実現するフィルタ素子として機能する。
