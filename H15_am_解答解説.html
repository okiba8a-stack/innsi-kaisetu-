大問1【電気数学】3次元調和関数とフーリエ変換

$f(x,y,z) = \dfrac{1}{\sqrt{x^2+y^2+z^2}} = \dfrac{1}{r}$（$r > 0$）

問1）$\nabla f$ を求めよ

$\dfrac{\partial f}{\partial x} = -\dfrac{x}{(x^2+y^2+z^2)^{3/2}} = -\dfrac{x}{r^3}$，y, z 成分も同様。
$$\nabla f = -\frac{\mathbf{r}}{r^3} \quad \text{（$\mathbf{r} = x\mathbf{i}+y\mathbf{j}+z\mathbf{k}$，大きさ $r$）}$$

問2）$\nabla^2 f$ を求めよ

$\nabla^2 f = \nabla\cdot(\nabla f) = \nabla\cdot\!\left(-\dfrac{\mathbf{r}}{r^3}\right)$ $$\frac{\partial}{\partial x}\!\left(-\frac{x}{r^3}\right) = -\frac{1}{r^3} + \frac{3x^2}{r^5}$$ 3成分の和：$-\dfrac{3}{r^3}+\dfrac{3(x^2+y^2+z^2)}{r^5} = -\dfrac{3}{r^3}+\dfrac{3}{r^3}$
$$\nabla^2 f = 0 \quad (r \ne 0)$$ $f = 1/r$ は原点を除く空間でラプラス方程式を満たす（調和関数）。

問3）$z = z_0 > 0$ 面での2次元フーリエ変換の証明

$F(k_x,k_y,z_0) = \dfrac{2\pi}{k}\exp(-kz_0)$（$k = \sqrt{k_x^2+k_y^2}$）を証明する。

極座標変換：$x = r\cos\varphi$，$y = r\sin\varphi$，$dx\,dy = r\,dr\,d\varphi$ $$F = \int_0^\infty\!\!\int_0^{2\pi} \frac{1}{\sqrt{r^2+z_0^2}}\exp\!\bigl(-j(k_x r\cos\varphi + k_y r\sin\varphi)\bigr)\,r\,d\varphi\,dr$$ $k_x r\cos\varphi + k_y r\sin\varphi = rk\cos(\varphi - \alpha)$（$\alpha = \arctan(k_y/k_x)$）なので： $$F = \int_0^\infty \frac{r}{\sqrt{r^2+z_0^2}}\left[\int_0^{2\pi}\exp(-jkr\cos(\varphi-\alpha))\,d\varphi\right]dr$$ 内側の積分（変数変換 $\theta = \varphi - \alpha$）： $$\int_0^{2\pi}\exp(-jkr\cos\theta)\,d\theta = 2\pi J_0(kr)$$ （0次ベッセル関数の定義 $J_0(r) = \frac{1}{2\pi}\int_0^{2\pi}\exp(-jr\cos\theta)d\theta$ を利用）
したがって： $$F = \int_0^\infty \frac{r}{\sqrt{r^2+z_0^2}}\cdot 2\pi J_0(kr)\,dr = 2\pi\cdot\frac{1}{k}\exp(-kz_0)$$ （与えられた公式 $\int_0^\infty \frac{r}{\sqrt{r^2+a^2}}J_0(kr)dr = \frac{1}{k}e^{-ka}$ に $a = z_0 > 0$ を代入）
$$F(k_x,k_y,z_0) = \frac{2\pi}{k}\exp(-kz_0) \qquad \text{（証明された）}$$

問4）$\phi(x,y,z)$ のフーリエ変換

$\phi(x,y,z) = \displaystyle\sum_{i=1}^n w_i f(x-x_i, y-y_i, z-z_i)$，$z_i < 0$

シフト定理と問3の結果を適用。$z_0 - z_i > 0$ なので各項のフーリエ変換： $$\mathcal{F}[f(x-x_i, y-y_i, z_0-z_i)] = e^{-j(k_x x_i + k_y y_i)} \cdot \frac{2\pi}{k}\exp(-k(z_0-z_i))$$
$$\Phi(k_x,k_y,z_0) = \frac{2\pi}{k}\exp(-kz_0)\sum_{i=1}^n w_i \exp\!\bigl(-j(k_x x_i+k_y y_i)+kz_i\bigr)$$ $G(k_x,k_y) = \displaystyle\sum_{i=1}^n w_i e^{-j(k_xx_i+k_yy_i)+kz_i}$ とおくと $\Phi = \dfrac{2\pi}{k}G\,e^{-kz_0}$

問5）$b_x$ の分布から $b_y$, $b_z$ を求める方法

$b_x = \partial\phi/\partial x$, $b_y = \partial\phi/\partial y$, $b_z = \partial\phi/\partial z$ のフーリエ変換（2次元）： $$B_x = jk_x\Phi, \quad B_y = jk_y\Phi, \quad B_z = \frac{\partial\Phi}{\partial z_0} = -k\Phi$$
$B_x(k_x,k_y,z_0)$ が既知なら $\Phi = B_x/(jk_x)$ より： $$B_y = \frac{k_y}{k_x}B_x, \quad B_z = \frac{jk}{k_x}B_x$$ 逆2次元フーリエ変換で $b_y(x,y,z_0)$，$b_z(x,y,z_0)$ を復元できる。
この手法は磁気異常から地下の磁化分布を推定する地球物理学的インバース問題に対応する。

問6）$z = \zeta > z_0$ 面での $b_x, b_y, b_z$ の推定

問4の結果から：$\Phi(k_x,k_y,\zeta) = \Phi(k_x,k_y,z_0)\cdot e^{-k(\zeta - z_0)}$ （上方継続式：higher altitude では高波数成分が指数的に減衰）
$b_x(x,y,z_0)$ のフーリエ変換 $B_x(k_x,k_y,z_0) = jk_x\Phi(z_0)$ から： $$B_x(k_x,k_y,\zeta) = jk_x\Phi(z_0)e^{-k(\zeta-z_0)} = B_x(k_x,k_y,z_0)\,e^{-k(\zeta-z_0)}$$ 同様に $B_y,B_z$ も求まる。逆変換で実空間の分布を得る。 指数因子 $e^{-k(\zeta-z_0)}$ は高波数（高空間周波数）成分を強く減衰させる（低域通過フィルタ）。

大問2【電気回路】四端子回路・ラプラス変換・インピーダンス整合

問1）四端子回路（ステップ入力 $e_1(t) = Eu(t)$）

$V(s) = \dfrac{E}{s(1+s\tau)}$（$\tau > 0$）

a) $v(t)$ の初期値・最終値

初期値定理：$v(0^+) = \lim_{s\to\infty}sV(s) = \lim_{s\to\infty}\dfrac{E}{1+s\tau} = 0$
最終値定理：$v(\infty) = \lim_{s\to 0}sV(s) = \lim_{s\to 0}\dfrac{E}{1+s\tau} = E$
$v(0^+) = 0$，$v(\infty) = E$

b) コンデンサ電流 $i(t)$ の初期値・最終値

$I(s) = CsV(s) = \dfrac{CE}{1+s\tau}$
初期値：$i(0^+) = \lim_{s\to\infty}sI(s) = \dfrac{CE}{\tau}$
最終値：$i(\infty) = \lim_{s\to 0}sI(s) = 0$
$i(0^+) = CE/\tau = E/R$（$\tau = RC$），$i(\infty) = 0$

c) 四端子回路の構造

$V(s)/E_1(s) = 1/(1+s\tau)$ は1次 RC ローパスフィルタの伝達関数。
四端子回路内部は抵抗 $R$ のみ（内部抵抗）で，出力に $C$ を接続した構成。$\tau = RC$。

d) 妥当性の吟味

・$v(0^+) = 0$：コンデンサ電圧は急変不可 ✓
・$v(\infty) = E$：直流定常でコンデンサは開放 → 電源電圧と等しい ✓
・$i(0^+) = E/R$：直後はCが短絡，RのみでE/R ✓
・$i(\infty) = 0$：定常でコンデンサに電流流れず ✓

e) パルス電圧 $e_2(t)$ のラプラス変換

$e_2(t) = E[u(t) - u(t-T)]$（幅Tのパルス）
$$E_2(s) = \frac{E}{s}(1 - e^{-sT})$$

f) コンデンサ電流の時間応答

$I_2(s) = CsV_2(s) = Cs\cdot\dfrac{E_2(s)}{1+s\tau} = \dfrac{CE(1-e^{-sT})}{1+s\tau}$
$\mathcal{L}^{-1}\!\left[\dfrac{CE}{1+s\tau}\right] = \dfrac{CE}{\tau}e^{-t/\tau}u(t)$，遅延定理を利用。
$$i_2(t) = \frac{CE}{\tau}\!\left[e^{-t/\tau}u(t) - e^{-(t-T)/\tau}u(t-T)\right]$$ $0 < t < T$：$i_2 = \dfrac{CE}{\tau}e^{-t/\tau}$（正，減衰）
$t > T$：$i_2 = \dfrac{CE}{\tau}e^{-t/\tau}(1 - e^{T/\tau}) < 0$（電流逆転）

問2）インピーダンス整合

a) 最大有能電力 $P_{AV}$

$P = V_0^2 R_L/(R_S+R_L)^2$，$R_L = R_S$ で最大。
$$P_{AV} = \frac{V_0^2}{4R_S}$$

b) 整合回路使用時の $R_L$ の電圧 $V_2$（無損失）

$P_2 = P_{AV} = V_2^2/R_L$ より $V_2 = \sqrt{P_{AV}\cdot R_L}$
$$V_2 = \frac{V_0}{2}\sqrt{\frac{R_L}{R_S}}$$

c) 理想変成器による整合（巻線比 $n_2/n_1$）

理想変成器の入力インピーダンス：$Z_{in} = (n_1/n_2)^2 R_L = R_S$
$$\frac{n_2}{n_1} = \sqrt{\frac{R_L}{R_S}}$$

d) LC整合回路（$R_L > R_S$，図2.5：L直列・Cシャント）

入力インピーダンス：$Z_{in} = j\omega L + \dfrac{R_L}{1+j\omega CR_L}$

実部 $= R_S$：$\dfrac{R_L}{1+\omega^2C^2R_L^2} = R_S$ → $1+\omega^2C^2R_L^2 = \dfrac{R_L}{R_S}$

虚部 $= 0$：$\omega L = \dfrac{\omega CR_L^2}{1+\omega^2C^2R_L^2} = \omega CR_LR_S$ → $L = CR_LR_S$
i) $\omega_0$： $\omega_0^2C^2R_L^2 = (R_L-R_S)/R_S$ より $$\omega_0 = \frac{1}{CR_L}\sqrt{\frac{R_L - R_S}{R_S}}$$ ii) L, C, $R_S$, $R_L$ の関係： $$L = CR_LR_S, \quad \omega_0^2C^2R_L^2 = \frac{R_L - R_S}{R_S}$$ iii) $R_L$ 側からみたインピーダンス（$\omega = \omega_0$）： 整合条件の逆方向から：$Y_{out} = j\omega_0 C + \dfrac{1}{j\omega_0 L + R_S}$ を計算すると
虚部がキャンセル（整合の相反性）し，$Z_{out} = R_L$

e) $|V_2/V_0|$ の周波数特性（グラフの概略）

$\omega=0$：Cは開放，Lはインピーダンスなし → 分圧 $|V_2/V_0| = R_L/(R_S+R_L)$
$\omega=\omega_0$：整合 → 最大電力 → $|V_2/V_0| = \frac{1}{2}\sqrt{R_L/R_S} > R_L/(R_S+R_L)$ for $R_L>R_S$
$\omega\to\infty$：Cが短絡（$R_L$を短絡），Lが開放 → $|V_2/V_0| \to 0$

バンドパス的特性（$\omega_0$ 付近で最大）

大問3【電磁気学】静電界と磁束密度

問1) 真空中の電荷

a) 接地導体平面と点電荷 +Q（鏡像法）

鏡像電荷 $-Q$ を導体面の対称位置（A から導体を挟んで距離 a）に置く。

i) +Q に働く力：
大きさ：$F = \dfrac{Q^2}{16\pi\varepsilon_0 a^2}$，方向：導体面に向かう（引力）
ii) 電気力線の様子：
+Q を中心に放射状に出た電気力線はすべて導体表面に直角に入射し，表面に終端する。 +Q と鏡像 -Q の場合と類似した双極子状分布（導体面が対称面）。
iii) 導体表面上の点 B ($x$ 軸座標 $x$) での電界：
座標系：導体面を $y=0$，+Q は $(0, a, 0)$，鏡像 $-Q$ は $(0, -a, 0)$，B は $(x, 0, 0)$。
+Q，-Q からの電界の x成分は打ち消し，y成分のみ残る： $$E_x = 0, \quad E_y = -\frac{Qa}{2\pi\varepsilon_0(x^2+a^2)^{3/2}}$$
電界は導体面に垂直（導体内向き），大きさ： $$E = \frac{Qa}{2\pi\varepsilon_0(x^2+a^2)^{3/2}}$$
iv) 表面電荷密度 $\sigma(x)$：
$$\sigma(x) = \varepsilon_0 E_{n}\big|_{\rm outward} = -\frac{Qa}{2\pi(x^2+a^2)^{3/2}}$$ （負：導体に負電荷が誘起）$x=0$ で最大，$|x|\to\infty$ で $\to 0$。

b) 半径 $a$ の導体球（電荷 $+Q$）

球対称性よりガウスの法則から：
領域$E$$V$（基準：無限遠）
$r \le a$（導体内）$0$$\dfrac{Q}{4\pi\varepsilon_0 a}$（一定）
$r > a$（外部）$\dfrac{Q}{4\pi\varepsilon_0 r^2}$$\dfrac{Q}{4\pi\varepsilon_0 r}$

c) 同心導体球（+2Q）と球殻（−Q）

ガウスの法則を半径 $r$ の球面に適用（各領域での囲まれる電荷を確認）：
  • $r < a$：導体内 → $E = 0$
  • $a < r < 2a$：囲む電荷 $= +2Q$ → $E = \dfrac{2Q}{4\pi\varepsilon_0 r^2}$
  • $r > 2a$：囲む電荷 $= 2Q + (-Q) = +Q$ → $E = \dfrac{Q}{4\pi\varepsilon_0 r^2}$
i) 電界 $E(r)$：
$$E(r) = \begin{cases} 0 & 0 \le r < a \\ \dfrac{Q}{2\pi\varepsilon_0 r^2} & a < r < 2a \\[6pt] \dfrac{Q}{4\pi\varepsilon_0 r^2} & r > 2a \end{cases}$$
ii) 導体球と球殻を導線接続した時の電位 $V(r)$：
接続により両者は同電位（1つの導体）。全電荷 $= 2Q + (-Q) = +Q$ が外表面（$r = 2a$）に集中。
$r < 2a$ 内部：$E = 0$ → 電位一定 $= Q/(4\pi\varepsilon_0 \cdot 2a)$
$$V(r) = \begin{cases} \dfrac{Q}{8\pi\varepsilon_0 a} & 0 \le r \le 2a \\[6pt] \dfrac{Q}{4\pi\varepsilon_0 r} & r > 2a \end{cases}$$

問2) 電流と磁束密度

a) 無限長円筒導体（半径 $a$，内部空洞，電流 $I$）

アンペールの法則 $\oint\mathbf{B}\cdot d\mathbf{l} = \mu_0 I_{enc}$：
$r < a$（空洞内）：$I_{enc} = 0$ → $B = 0$
$r > a$（外部）：$B \cdot 2\pi r = \mu_0 I$
$$B(r) = \begin{cases} 0 & 0 \le r < a \\ \dfrac{\mu_0 I}{2\pi r} & r > a \end{cases}$$ 方向：電流の向きに対して右ねじの方向（円周方向）

b) 二本の平行円筒導体（間隔 $d$，往復電流 $I$）の $x$ 軸上の磁束密度

左円筒（$x = 0$）を電流 $+I$（$+z$方向），右円筒（$x = d$）を電流 $-I$（$-z$方向）とする。
各円筒からの寄与をアンペールの法則と重ね合わせで計算：
領域$B$ の大きさ方向
$|x| < a$（左筒内）$\dfrac{\mu_0 I}{2\pi(d-x)}$（右筒のみ）$+y$
$a < x < d-a$（導体間）$\dfrac{\mu_0 I}{2\pi}\!\!\left(\dfrac{1}{x}+\dfrac{1}{d-x}\right)$$+y$（同方向で加算）
$|x-d| < a$（右筒内）$\dfrac{\mu_0 I}{2\pi x}$（左筒のみ）$+y$
$x > d+a$（右側外部）$\dfrac{\mu_0 I}{2\pi}\!\!\left(\dfrac{1}{x}-\dfrac{1}{x-d}\right)$反方向（減算）

導体間では両円筒の磁界が同方向に加算されて最大，外側では打ち消し合う。

c) 往復二導体の単位長さあたりの自己インダクタンス

$d \gg a$ の近似。両導体間の鎖交磁束（単位長さ）： $$\frac{\Phi}{l} \approx 2\times\frac{\mu_0 I}{2\pi}\ln\frac{d}{a} = \frac{\mu_0 I}{\pi}\ln\frac{d}{a}$$
$$\frac{L}{l} = \frac{\mu_0}{\pi}\ln\frac{d}{a}$$

d) 実体（中実）円柱導体に変えた場合のインダクタンスの変化

中実導体では導体内部にも磁界が存在する（内部インダクタンス）。単位長さの内部インダクタンス：各導体 $\mu_0/(8\pi)$（均一電流分布の場合），往復で $\mu_0/(4\pi)$。
単位長あたりの自己インダクタンス： $$\frac{L}{l} = \frac{\mu_0}{\pi}\left(\ln\frac{d}{a} + \frac{1}{4}\right)$$ 増加する（内部インダクタンスの分だけ）。

e) 高周波交流電流での電流密度分布（表皮効果）

高周波では表皮効果（skin effect）により，電流は導体表面に集中し不均一分布となる。
表皮深さ $\delta = \sqrt{2/(\omega\mu_0\sigma)}$ が小さくなり，電流は外表面の薄い層に集中。
理由：交流磁場が導体内に浸透するにつれて誘導起電力が逆向きに生じ，内部電流を打ち消す。
結果：内部磁束（内部インダクタンス）が減少し，有効インダクタンスは空洞円筒の値に近づく。
