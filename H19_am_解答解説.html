H19年度 午前 電気回路 解答解説

平成18年8月17日実施 ／ 専門科目：電気回路（午前）

大問1：変圧器の解析

一次インダクタンス \(L_1\)、二次インダクタンス \(L_2\)、相互インダクタンス \(M\)（\(L_1L_2 \neq M^2\)）。磁気飽和・巻線抵抗無視。

問1-1) 二次開放時の二次電圧 \(V_2\)

二次開放 → \(I_2=0\)。一次方程式：\(V_1 = j\omega L_1 I_1\) → \(I_1 = V_1/(j\omega L_1)\)
二次誘起起電力：\(V_2 = j\omega M I_1 = j\omega M \cdot V_1/(j\omega L_1)\)
\[V_2 = \frac{M}{L_1} V_1\] （変圧比は巻数比ではなく \(M/L_1\) で表される）

問1-2) 二次にコンデンサ \(C\) 接続時の共振周波数（すべて）

連立方程式（フェーザー）： \[V_1 = j\omega L_1 I_1 - j\omega M I_2 \quad (1)\] \[0 = -j\omega M I_1 + \left(j\omega L_2 + \frac{1}{j\omega C}\right) I_2 \quad (2)\] (2)より：\(I_2 = j\omega M I_1 / \left(j\omega L_2 + \frac{1}{j\omega C}\right)\)
入力インピーダンスを求めると（\(\Delta = L_1L_2 - M^2\)）： \[Z_{in} = j\omega\,\frac{L_1 - \omega^2 C\Delta}{1 - \omega^2 L_2 C}\]
直列共振（\(Z_{in}=0\)）：\(L_1 - \omega^2 C\Delta = 0\) \[\omega_s = \sqrt{\frac{L_1}{C(L_1L_2-M^2)}}\]
並列共振（\(Z_{in}\to\infty\)）：\(1 - \omega^2 L_2 C = 0\) \[\omega_p = \frac{1}{\sqrt{L_2 C}}\]
\(L_1L_2 > M^2\) の場合、\(\Delta = L_1L_2-M^2 > 0\)。\(\omega_s\) と \(\omega_p\) の大小関係：
\(\omega_s^2/\omega_p^2 = L_1L_2/\Delta = L_1L_2/(L_1L_2-M^2) > 1\) → \(\omega_s > \omega_p\)

問1-3) 二次に抵抗 \(R\) 接続時の平均消費電力・力率・最大電力

二次側テブナン等価（一次短絡時）： \[V_{oc} = \frac{M}{L_1}V, \qquad Z_{th} = j\omega\frac{\Delta}{L_1} \quad (\text{純虚数！})\] 一次から見た入力インピーダンス（反射インピーダンス利用）： \[Z_{in} = j\omega L_1 + \frac{(\omega M)^2}{R + j\omega L_2} = \underbrace{\frac{\omega^2 M^2 R}{R^2+\omega^2L_2^2}}_{\text{Re}} + j\omega\underbrace{\left[L_1 - \frac{\omega^2 M^2 L_2}{R^2+\omega^2L_2^2}\right]}_{\text{Im}/\omega}\]
平均消費電力： \[P = \frac{1}{2}\cdot\frac{|V|^2\,\mathrm{Re}(Z_{in})}{|Z_{in}|^2} = \frac{\omega^2 M^2 R\,|V|^2}{2\left[(\omega^2M^2R)^2/(R^2+\omega^2L_2^2)^2 + \text{...}\right]|Z_{in}|^2/(R^2+...)}\] （簡潔に書くと） \[P = \frac{|V|^2\,\omega^2 M^2 R}{2\bigl[(R^2+\omega^2L_2^2)\,|Z_{in}|^2/(R^2+\omega^2L_2^2)\bigr]}\]
力率：\(\cos\phi = \mathrm{Re}(Z_{in})/|Z_{in}|\)

最大消費電力条件：
二次側テブナン \(Z_{th}=j\omega\Delta/L_1\)（純虚数）なので最大電力条件 \(R = |Z_{th}|\)： \[R_{opt} = \frac{\omega\Delta}{L_1} = \frac{\omega(L_1L_2-M^2)}{L_1}\] 最大平均電力： \[P_{max} = \frac{|V_{oc}|^2}{4R_{opt}} = \frac{M^2V^2}{4\omega L_1(L_1L_2-M^2)}\]

大問2：ブリッジ回路と全域通過条件

図2.1：\(L\)–\(R_1\) アーム（上：\(L\)、下：\(R_1\)）と \(C\)–\(R_2\) アーム（上：\(C\)、下：\(R_2\)）からなるブリッジ回路。\(V_0\) は両アームの中点間電圧。

各中点電位： \[V_A = V\cdot\frac{R_1}{R_1+j\omega L}, \qquad V_B = V\cdot\frac{j\omega C R_2}{1+j\omega C R_2}\] \[V_0 = V_A - V_B = V\cdot\frac{R_1+\omega^2 LCR_2}{(R_1+j\omega L)(1+j\omega CR_2)}\]

問2-1) \(\omega=0\) のとき

\(\omega\to 0\)：\(L\to\) 短絡、\(C\to\) 開放 → \(V_A\to V,\ V_B\to 0\)
\(V_0 = V\)（電源電圧と等振幅・同位相）

問2-2) \(\omega\to\infty\) のとき

\(\omega\to\infty\)：\(L\to\) 開放、\(C\to\) 短絡 → \(V_A\to 0,\ V_B\to V\)
\(|V_0| = V\)（振幅は同じ、位相は \(-180°\) 反転）

問2-3) \(|V_0|\) が \(\omega\) に依らず一定となる条件

\[|V_0|^2 = V^2\,\frac{(R_1+\omega^2LCR_2)^2}{(R_1^2+\omega^2L^2)(1+\omega^2C^2R_2^2)}\] これが \(\omega\) に依らず一定 → 分子と分母が同比：

分子 \(= R_1^2 + 2\omega^2R_1LCR_2 + \omega^4L^2C^2R_2^2\)
分母 \(= R_1^2 + \omega^2(R_1^2C^2R_2^2+L^2) + \omega^4L^2C^2R_2^2\)

\(\omega^2\) の係数を一致させる： \[2R_1LCR_2 = R_1^2C^2R_2^2 + L^2 \Rightarrow (CR_1R_2-L)^2 = 0\]
\[\boxed{R_1 R_2 = \frac{L}{C}}\] このとき \(|V_0| = V\)（全域通過：位相のみ変化する全域通過フィルタ）

大問3：RC 並列回路とフーリエ解析

問3-1) 正弦波電圧 \(v(t)=V\sin\omega t\) に対する \(i(t)\)（図3.1）

\[i_R = \frac{v}{R} = \frac{V}{R}\sin\omega t, \qquad i_C = C\frac{dv}{dt} = CV\omega\cos\omega t\] \[i(t) = i_R + i_C = \frac{V}{R}\sin\omega t + V\omega C\cos\omega t\]
\[i(t) = V\sqrt{\frac{1}{R^2}+\omega^2C^2}\,\sin(\omega t + \phi), \qquad \phi = \arctan(\omega CR)\]

問3-2) 2周波数電源の場合（図3.2）と重ね合わせの理

重ね合わせの理（線形性原理）：
線形回路では、複数の独立電源が同時に存在する場合の応答は、各電源が単独に存在する場合の応答の和に等しい。

\(v_1(t)=V_1\sin\omega_1 t\) の単独応答：\(i_1(t) = V_1\sqrt{1/R^2+\omega_1^2C^2}\sin(\omega_1 t+\phi_1)\) \(v_2(t)=V_2\sin\omega_2 t\) の単独応答：\(i_2(t) = V_2\sqrt{1/R^2+\omega_2^2C^2}\sin(\omega_2 t+\phi_2)\)
\[i(t) = i_1(t)+i_2(t), \qquad \phi_k = \arctan(\omega_k CR)\]

問3-3) 三角波 \(f(t)\) のフーリエ級数（周期 \(T\)、振幅 \(T/4\)）（図3.3）

基本角周波数 \(\omega_0 = 2\pi/T\)。奇関数の三角波（\(f(0)=0\), \(f(T/4)=T/4\)）： \[f(t) = \frac{8}{\pi^2}\cdot\frac{T}{4}\sum_{n=1,3,5,...}\frac{(-1)^{(n-1)/2}}{n^2}\sin(n\omega_0 t) = \frac{2T}{\pi^2}\sum_{n=1,3,5,...}\frac{(-1)^{(n-1)/2}}{n^2}\sin\!\left(\frac{2\pi n t}{T}\right)\]
\[f(t) = \frac{2T}{\pi^2}\!\left[\sin\frac{2\pi t}{T} - \frac{1}{9}\sin\frac{6\pi t}{T} + \frac{1}{25}\sin\frac{10\pi t}{T} - \cdots\right]\]

問3-4) 三角波電圧を RC 並列回路に印加（図3.4、定常状態）

図3.4の三角波：周期 \(T=\pi/\omega\)（基本角周波数 \(\omega_0=2\omega\)）、振幅 \(A=\pi/(2\omega)\)。

フーリエ展開（問3-3と同形）： \[v(t) = \frac{2\cdot\pi/(2\omega)}{\pi^2/\pi}\cdot\frac{2T}{\pi^2}\cdots = \frac{4}{\pi\omega}\!\left[\sin 2\omega t - \frac{\sin 6\omega t}{9} + \frac{\sin 10\omega t}{25} - \cdots\right]\]
各調波成分 \(V_n\sin(n\omega_0 t)\)（\(n\)=1,3,5,...; \(n\omega_0=2n\omega\)）に対して：
\[i(t) = \sum_{n=1,3,5,...} V_n\sqrt{\frac{1}{R^2}+(2n\omega)^2C^2}\;\sin(2n\omega t + \phi_n)\] ただし \(V_n = \dfrac{4(-1)^{(n-1)/2}}{n\pi\omega}\)、\(\phi_n = \arctan(2n\omega CR)\)

または直接微分法：\(i(t) = v(t)/R + C\,dv/dt\)（\(dv/dt\) は方形波）で計算可。

大問4：RLC 回路とラプラス変換

問4-1) 図4.1の直列 RLC 回路（\(t=0\) でスイッチ投入、初期値 0）

a) 微分方程式

\(v_0(t)\)：コンデンサ電圧、\(i = C\,dv_0/dt\)： \[E = Ri + L\frac{di}{dt} + v_0 \Longrightarrow LC\frac{d^2v_0}{dt^2} + RC\frac{dv_0}{dt} + v_0 = E\]

b) \(V_0(s)\) をラプラス変換で求める

初期値ゼロで変換： \[LCs^2V_0(s) + RCsV_0(s) + V_0(s) = \frac{E}{s}\]
\[V_0(s) = \frac{E/LC}{s(s^2 + \frac{R}{L}s + \frac{1}{LC})} = \frac{E\omega_0^2}{s(s^2+2\alpha s+\omega_0^2)}\] ただし \(\omega_0^2 = 1/(LC),\ 2\alpha = R/L\)

c) \(L=50\text{ H},\ C=10^{-2}\text{ F},\ R=150\,\Omega\) のとき \(v_0(t)\)

\[\omega_0^2 = \frac{1}{50\times10^{-2}} = 2,\quad \frac{R}{L} = \frac{150}{50} = 3\] 特性方程式：\(s^2+3s+2=(s+1)(s+2)=0\) → \(s=-1,\,-2\)（過減衰）

\[V_0(s) = \frac{2E}{s(s+1)(s+2)} = E\left(\frac{1}{s} - \frac{2}{s+1} + \frac{1}{s+2}\right)\] （部分分数展開：\(A=2E/2=E,\ B=2E/((-1)\cdot 1)=-2E,\ C=2E/(2\cdot 1)=E\)）
\[v_0(t) = E\bigl(1 - 2e^{-t} + e^{-2t}\bigr)\,U(t)\] 過減衰応答（振動なし）。\(v_0(0)=0,\ v_0'(0)=0,\ v_0(\infty)=E\)。単調増加。

d) \(v_0(t)\) の概形

\(v_0'(t) = E(2e^{-t}-2e^{-2t}) = 2E(e^{-t}-e^{-2t}) \geq 0\) for \(t>0\)：単調増加。
\(t=0\) では \(v_0=0\)（初期値）、\(t\to\infty\) では \(v_0\to E\)（定常値）。
最大傾きは \(t\)→ 小で生じ、徐々に傾きが \(0\) に近づく S 字形（凸型）。

問4-2) 図4.2の矩形パルスのラプラス変換

\(v_i(t) = E\,U(t) - E\,U(t-\tau)\)（0 から \(\tau\) まで高さ \(E\) のパルス）
\[V_i(s) = \frac{E}{s}(1 - e^{-\tau s})\]

問4-3) パルス入力に対する \(v_0(t)\)（図4.3、同じ \(R,L,C\) 値）

伝達関数：\(H(s) = \dfrac{1/LC}{s^2+(R/L)s+1/LC} = \dfrac{2}{(s+1)(s+2)}\)

\(V_0(s) = H(s)\cdot V_i(s) = \dfrac{2E}{(s+1)(s+2)}\cdot\dfrac{1-e^{-\tau s}}{s}\)

ステップ応答 \(g(t) = E(1-2e^{-t}+e^{-2t})U(t)\)（問4-1c の結果）を利用し、
遅延定理（\(e^{-\tau s}\) → \(t-\tau\) の遅延）：
\[v_0(t) = g(t) - g(t-\tau)\] \[= E\bigl[(1-2e^{-t}+e^{-2t})U(t) - (1-2e^{-(t-\tau)}+e^{-2(t-\tau)})U(t-\tau)\bigr]\]

解答まとめ

問主な結果
1-1\(V_2 = (M/L_1)V_1\)
1-2並列共振 \(\omega_p=1/\sqrt{L_2C}\)、直列共振 \(\omega_s=\sqrt{L_1/(C\Delta)}\)（\(\Delta=L_1L_2-M^2\)）
1-3\(R_{opt}=\omega\Delta/L_1\)、\(P_{max}=M^2V^2/(4\omega L_1\Delta)\)
2-1\(\omega=0\): \(V_0=V\)
2-2\(\omega\to\infty\): \(|V_0|=V\)（位相 \(-180°\)）
2-3条件：\(R_1R_2 = L/C\)（全域通過フィルタ）
3-1\(i(t)=V\sqrt{1/R^2+\omega^2C^2}\sin(\omega t+\arctan\omega CR)\)
3-3\(f(t)=\frac{2T}{\pi^2}\sum_{n\text{ odd}}\frac{(-1)^{(n-1)/2}}{n^2}\sin\frac{2\pi nt}{T}\)
4-1\(v_0(t)=E(1-2e^{-t}+e^{-2t})U(t)\)（過減衰）
4-2\(V_i(s)=(E/s)(1-e^{-\tau s})\)
4-3\(v_0(t)=g(t)-g(t-\tau)\)（ステップ応答の差）
