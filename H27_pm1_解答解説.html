大問1　変圧器（結合インダクタ）の行列表現

問1)　図1.1 の変圧器の行列（$V_1, I_1, V_2, I_2$ の関係）

ドット規則の確認（図1.1）：
  • $L_1$ の黒丸は上側、$L_2$ の黒丸も上側（両方同じ側）
  • $I_1$ は右向き（→）：$L_1$ の黒丸端（上）から流入
  • $I_2$ は左向き（←）：$L_2$ の黒丸端（上）から流入
  • 両電流とも黒丸端に流入 → 磁束が強め合う → 加算結合（+M）
ファラデーの法則により： $$V_1 = j\omega L_1 I_1 + j\omega M I_2 \quad \text{（}I_2\text{が}L_2\text{の黒丸端に入り}L_1\text{に加算的に寄与）}$$ $$V_2 = j\omega M I_1 + j\omega L_2 I_2$$
$$\boxed{\begin{pmatrix}V_1\\V_2\end{pmatrix} = j\omega\begin{pmatrix}L_1 & M\\ M & L_2\end{pmatrix}\begin{pmatrix}I_1\\I_2\end{pmatrix}}$$

問2)　電流制御電圧源（CCVS）を用いた等価回路

$V_1 = j\omega L_1 I_1 + j\omega M I_2$：$j\omega L_1$ と CCVS $j\omega M I_2$（$I_2$ で制御）の直列 $V_2 = j\omega M I_1 + j\omega L_2 I_2$：CCVS $j\omega M I_1$（$I_1$ で制御）と $j\omega L_2$ の直列
等価回路（テキスト表記）： 
ポート1側：
  I₁ →  [jωL₁]  [jωMI₂（CCVS）]  → ポート1（V₁）

ポート2側：
  I₂ ←  [jωMI₁（CCVS）]  [jωL₂]  → ポート2（V₂）
CCVS $j\omega MI_2$ は $I_2$ で制御（ポート1側に直列挿入）、CCVS $j\omega MI_1$ は $I_1$ で制御（ポート2側に直列挿入）。

問3)　図1.2 の閉路方程式（行列形式）

図1.2 のメッシュ構成（図より）：
  • メッシュ A（左ループ，$V_S, Z_A, L_1$）：電流 $I_A$
  • メッシュ C（中央上ループ，$Z_C, L_1$ 共有，$L_2$ 共有）：電流 $I_C$
  • メッシュ B（右ループ，$L_2, Z_B$）：電流 $I_B$
$L_1$ を流れる電流：$I_A - I_C$（黒丸端から $I_A$ が，反対方向から $I_C$ が流れる）。
$L_2$ を流れる電流：$I_C - I_B$（同様）。
メッシュ A の $L_1$ は黒丸端から $I_A$（正方向），メッシュ B の $L_2$ は黒丸端から $I_B$ が出る方向 → 相互結合の符号が正（加算）。 KVL 各メッシュ： $$V_S = (Z_A + j\omega L_1)I_A - (j\omega L_1 + j\omega M)I_C + j\omega M I_B$$ $$0 = -(j\omega L_1 + j\omega M)I_A + (j\omega L_1 + Z_C + j\omega L_2)I_C -(j\omega L_2 + j\omega M)I_B$$ $$0 = j\omega M I_A - (j\omega L_2 + j\omega M)I_C + (Z_B + j\omega L_2)I_B$$
$$\boxed{\begin{pmatrix}Z_A+j\omega L_1 & -(j\omega L_1+j\omega M) & j\omega M\\ -(j\omega L_1+j\omega M) & j\omega L_1+Z_C+j\omega L_2 & -(j\omega L_2+j\omega M)\\ j\omega M & -(j\omega L_2+j\omega M) & Z_B+j\omega L_2\end{pmatrix}\begin{pmatrix}I_A\\I_C\\I_B\end{pmatrix} = \begin{pmatrix}V_S\\0\\0\end{pmatrix}}$$

問4)　図1.3 の変圧器の行列（黒丸位置注意）

図1.3 のドット規則：
  • $L_1$ の黒丸は上側（図1.1 と同じ）
  • $L_2$ の黒丸は下側（図1.1 と異なる）
  • $I_1$ →：$L_1$ の黒丸端（上）から流入 ✓
  • $I_2$ ←：$L_2$ の上端から流入（非黒丸端）→ 減算結合（−M）
$I_1$ は $L_1$ の黒丸端に入り、$I_2$ は $L_2$ の非黒丸端に入る → 磁束が打ち消し合う： $$V_1 = j\omega L_1 I_1 - j\omega M I_2$$ $$V_2 = -j\omega M I_1 + j\omega L_2 I_2$$
$$\boxed{\begin{pmatrix}V_1\\V_2\end{pmatrix} = j\omega\begin{pmatrix}L_1 & -M\\ -M & L_2\end{pmatrix}\begin{pmatrix}I_1\\I_2\end{pmatrix}}$$ （図1.1 と比較して $M$ の符号が反転）

大問2　無損失分布定数線路の過渡応答

特性インピーダンス $Z_0$，長さ $\ell$，伝搬速度 $v$。電圧源 $V_0$，内部抵抗 $Z_0/2$。終端抵抗 $2Z_0$。$t=0$ で接続。

問1)　各端での反射係数

入力端（電源側）：ソースインピーダンス $Z_0/2$： $$\Gamma_{\rm in} = \frac{Z_0/2 - Z_0}{Z_0/2 + Z_0} = \frac{-Z_0/2}{3Z_0/2} = -\frac{1}{3}$$ 出力端（負荷側）：負荷インピーダンス $2Z_0$： $$\Gamma_{\rm out} = \frac{2Z_0 - Z_0}{2Z_0 + Z_0} = \frac{Z_0}{3Z_0} = +\frac{1}{3}$$
$$\boxed{\Gamma_{\rm in} = -\frac{1}{3},\quad \Gamma_{\rm out} = +\frac{1}{3}}$$

問2)　$0 < t < \ell/v$ における入力端電圧

$t=0$ で電源接続。初期進行波電圧（電圧分割）： $$V_+ = V_0 \times \frac{Z_0}{Z_0/2 + Z_0} = V_0 \times \frac{Z_0}{3Z_0/2} = \frac{2V_0}{3}$$ $t < \ell/v$ では出力端からの反射波がまだ入力端に到達していない： $$V_{\rm in} = V_+ = \frac{2V_0}{3}$$
$$\boxed{V_{\rm in} = \frac{2V_0}{3}}$$

問3)　$\ell/v < t < 3\ell/v$ における出力端電圧

$t = \ell/v$：進行波 $V_+ = 2V_0/3$ が出力端に到達。 $$V_{\rm out} = V_+(1+\Gamma_{\rm out}) = \frac{2V_0}{3} \times \left(1+\frac{1}{3}\right) = \frac{2V_0}{3} \times \frac{4}{3} = \frac{8V_0}{9}$$ 反射波 $V_- = \Gamma_{\rm out} \times V_+ = \frac{1}{3}\times\frac{2V_0}{3} = \frac{2V_0}{9}$（左向き）が入力端に戻るのは $t=2\ell/v$，再度出力端に来るのは $t=3\ell/v$ なのでこの期間は変化なし。
$$\boxed{V_{\rm out} = \frac{8V_0}{9} \quad (\ell/v < t < 3\ell/v)}$$

問4)　出力端電圧の時間変化（$0 < t < 6\ell/v$）

格子線図で反射波を追跡： | 時刻 | 事象 | 出力端電圧 | |---|---|---| | $t=0$ | 初期進行波 $2V_0/3$ 発射 | 0 | | $t=\ell/v$ | 出力端到達：$+8V_0/9$，反射波 $+2V_0/9$ 左向き | $8V_0/9$ | | $t=2\ell/v$ | 入力端で反射：$\Gamma_{\rm in}\times 2V_0/9 = -2V_0/27$（右向き） | （変化なし） | | $t=3\ell/v$ | 出力端に $-2V_0/27$ 到達，$\Delta V = (-2V_0/27)(4/3) = -8V_0/81$ | $8V_0/9-8V_0/81=64V_0/81$ | | $t=4\ell/v$ | 入力端反射：$2V_0/243$（右向き） | （変化なし） | | $t=5\ell/v$ | 出力端に $+2V_0/243$ 到達，$\Delta V = +8V_0/729$ | $64V_0/81+8V_0/729=584V_0/729$ |
電圧 8V₀/9 ──────────────────────┐ │ 64V₀/81 64V₀/81 └──────────┐ │ 584V₀/729 584V₀/729 └────── ↓（4V₀/5 に収束） 0 ℓ/v 2ℓ/v 3ℓ/v 4ℓ/v 5ℓ/v 6ℓ/v → t
変化時刻と電圧： $$t=\frac{\ell}{v}:\;V=\frac{8V_0}{9}\approx 0.889V_0,\quad t=\frac{3\ell}{v}:\;V=\frac{64V_0}{81}\approx 0.790V_0,\quad t=\frac{5\ell}{v}:\;V=\frac{584V_0}{729}\approx 0.801V_0$$

問5)　十分時間経過後の出力端電圧（定常値）

直流定常状態では線路は単なる導線。 $$V_{\rm out,DC} = V_0 \times \frac{2Z_0}{Z_0/2 + 2Z_0} = V_0 \times \frac{2Z_0}{5Z_0/2} = V_0 \times \frac{4}{5}$$ 等比級数での確認：各ステップの比 = $\Gamma_{\rm out} \times \Gamma_{\rm in} = (1/3)\times(-1/3) = -1/9$： $$V_\infty = \frac{8V_0}{9}\times\frac{1}{1+1/9} = \frac{8V_0}{9}\times\frac{9}{10} = \frac{4V_0}{5} \checkmark$$
$$\boxed{V_{\rm out,\infty} = \frac{4V_0}{5}}$$

大問3　ゲート接地 MOSFET 回路

$I_{DS} \propto V_{\rm eff}^2$，$V_{\rm eff}=0.2$V のとき $I_{DS}=1\,\text{mA}$。$V_{th}=0.3$V，$V_{DD}=5$V。比例係数：$k = I_{DS}/V_{\rm eff}^2 = 1/(0.04) = 25\,\text{mA/V}^2$。

問1)　$I_{DS}=4\,\text{mA}$，$V_{OUT}=3\,\text{V}$ のバイアス設計

$V_{\rm eff}$：$I_{DS} = k V_{\rm eff}^2$ より： $$V_{\rm eff}^2 = \frac{4\,\text{mA}}{25\,\text{mA/V}^2} = 0.16\,\text{V}^2 \Rightarrow V_{\rm eff} = 0.4\,\text{V}$$ $R$：ドレイン電流による電圧降下： $$R = \frac{V_{DD} - V_{OUT}}{I_{DS}} = \frac{5-3}{4\times 10^{-3}} = \frac{2}{0.004} = 500\,\Omega$$ $V_G$：ゲート接地回路では $V_S = 0$（小信号源の直流成分 = 0）なので： $$V_{GS} = V_G - V_S = V_G$$ $$V_{\rm eff} = V_{GS} - V_{th} = V_G - V_{th} = 0.4\,\text{V}$$ $$V_G = V_{th} + V_{\rm eff} = 0.3 + 0.4 = 0.7\,\text{V}$$
$$\boxed{V_{\rm eff} = 0.4\,\text{V},\quad R = 500\,\Omega,\quad V_G = 0.7\,\text{V}}$$

問2)　伝達コンダクタンス $g_m$

$$g_m = \frac{\partial I_{DS}}{\partial V_{GS}} = 2k V_{\rm eff} = 2 \times 25\,\text{mA/V}^2 \times 0.4\,\text{V} = 20\,\text{mA/V}$$ 別表現：$g_m = 2I_{DS}/V_{\rm eff} = 2\times4/0.4 = 20\,\text{mA/V}$ ✓
$$\boxed{g_m = 20\,\text{mA/V}}$$

問3)　小信号等価回路

ゲート接地（コモンゲート）小信号等価回路： 
       v_in (入力)
         │
         ├──── ソース (s)
         │        │
         │      v_gs（−端子：ゲートAC接地 = 0，+端子：ソース = v_in）
         │         v_gs = 0 − v_in = −v_in
         │        │
         │      [g_m × v_gs]  ↓電流源（ドレインへ）
         │        │
    AC   ├──── ドレイン (d) ─── R ─── VDD(AC接地)
   接地  │                       │
  (ゲート)│                     v_out
         └── グラウンド
ゲート（g）：VGから DC バイアス、小信号では AC グラウンド（$v_g = 0$）
$v_{gs} = v_g - v_s = 0 - v_{in} = -v_{in}$
電流源：$g_m v_{gs} = -g_m v_{in}$（ドレイン→ソース方向）

問4)　電圧増幅率 $A_v = v_{out}/v_{in}$

ドレインノードの KCL（小信号、VDD は AC グラウンド）： $R$ を通る電流（VDD→ドレイン方向）= MOSFET ドレイン電流（ドレイン→ソース方向）: $$\frac{0 - v_{out}}{R} + g_m v_{gs} = 0$$ $$-\frac{v_{out}}{R} + g_m \times(-v_{in}) = 0... \text{（電流方向を確認）}$$ 正確に：ドレインノードへ流入電流の和 = 0： - $R$ から流入：$(0-v_{out})/R = -v_{out}/R$ - MOSFET 電流源（ソースからドレインに向かう電流 = ドレインに流入）：$g_m v_{gs} = -g_m v_{in}$ $$-\frac{v_{out}}{R} + (-g_m v_{in}) = 0 \quad ... \text{ここで符号を整理}$$ コモンゲート増幅器の標準結果：ドレイン電流増加 → 出力電圧増加（非反転）： $$v_{out} = g_m v_{in} R$$ $$A_v = \frac{v_{out}}{v_{in}} = g_m R$$
$$\boxed{A_v = g_m R = 20\,\text{mA/V} \times 500\,\Omega = 10}$$ （コモンゲート増幅器は非反転、利得 $= g_m R_D$）
