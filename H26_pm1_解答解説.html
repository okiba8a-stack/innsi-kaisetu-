大問1　無損失分布定数線路の過渡応答

特性インピーダンス $Z_0$、長さ $2\ell$ の無損失分布定数線路。BB'（$x=\ell$）にシャント抵抗 $R=Z_0$ を接続。電圧源 $E$、直列抵抗 $R=Z_0$ を介して $t=0$ にスイッチ S オン。CC' は開放。伝搬速度 $c$。

問1)　S行列

シャント抵抗 $R=Z_0$ を2端子対回路（図1.2）として見る。 入射波電圧 $a_k$、反射波電圧 $b_k$（規格化：$V_k = a_k+b_k$，$Z_0 I_k = a_k - b_k$）。 ジャンクション電圧の連続性： $$a_1 + b_1 = a_2 + b_2 = V \quad \cdots (1)$$ KCL（左からの電流 + 右からの電流 = $R$ への電流）： $$\frac{a_1-b_1}{Z_0} + \frac{a_2-b_2}{Z_0} = \frac{V}{R} = \frac{a_1+b_1}{Z_0}$$ $$\Rightarrow (a_2 - b_2) = 2b_1 \quad \cdots (2)$$ (1)(2) 連立して $b_1, b_2$ を $a_1, a_2$ で表す： (2) より $b_2 = a_2 - 2b_1$、(1) へ代入：$a_1 + b_1 = 2a_2 - 2b_1$ $$b_1 = \frac{2a_2 - a_1}{3}, \quad b_2 = \frac{2a_1 - a_2}{3}$$
$$\begin{pmatrix}b_1 \\ b_2\end{pmatrix} = \begin{pmatrix}S_{11} & S_{12} \\ S_{21} & S_{22}\end{pmatrix}\begin{pmatrix}a_1 \\ a_2\end{pmatrix}$$ $$\boxed{\begin{pmatrix}S_{11} & S_{12} \\ S_{21} & S_{22}\end{pmatrix} = \begin{pmatrix}-\dfrac{1}{3} & \dfrac{2}{3} \\[6pt] \dfrac{2}{3} & -\dfrac{1}{3}\end{pmatrix}}$$ （一般化：$R$ 一般のとき $S_{11}=S_{22}=-1/(2z+1)$, $S_{12}=S_{21}=2z/(2z+1)$，$z=R/Z_0$）

問2)　各端子対の電圧反射係数

電圧反射係数 $\Gamma = (Z_L - Z_0)/(Z_L + Z_0)$： AA'（電源側）：直列抵抗 $R=Z_0$ が電源インピーダンス。線路から見たソースインピーダンス $= Z_0$。 $$\Gamma_{AA'} = \frac{Z_0 - Z_0}{Z_0 + Z_0} = 0 \quad (\text{整合電源})$$ BB'（シャント抵抗）：S行列より $S_{11}=S_{22}$： $$\Gamma_{BB'} = S_{11} = -\frac{1}{3}$$ CC'（開放端）：$Z_L = \infty$： $$\Gamma_{CC'} = \frac{\infty - Z_0}{\infty + Z_0} = +1$$
$$\Gamma_{AA'} = 0, \quad \Gamma_{BB'} = -\frac{1}{3}, \quad \Gamma_{CC'} = +1$$

問3)　$x=\ell$（BB'）における電圧の時間変化（$0\le t\le 4\ell/c$）

格子線図による波追跡： $t=0$：スイッチ S オン。電源 $E$ から初期進行波（整合分圧）： $$V_+ = E \times \frac{Z_0}{R+Z_0} = \frac{E}{2} \quad (\text{右向き})$$ $t = \ell/c$：$V_+ = E/2$ が BB' に到達（$a_1 = E/2$，$a_2=0$）： $$b_1 = S_{11}\cdot a_1 = -\frac{E}{6} \quad (\text{左向き}), \quad b_2 = S_{21}\cdot a_1 = \frac{E}{3} \quad (\text{右向き})$$ $$V_{BB'} = a_1 + b_1 = \frac{E}{2} - \frac{E}{6} = \frac{E}{3}$$ 反射波 $-E/6$ は AA' へ戻るが $\Gamma_{AA'}=0$ で吸収。透過波 $E/3$ は CC' へ。 $t = 2\ell/c$：$E/3$ が CC' に到達 $\rightarrow$ $\Gamma_{CC'}=+1$ で反射：$+E/3$（左向き）。 $t = 3\ell/c$：$E/3$（左向き）が BB' に到達（$a_2 = E/3$，$a_1=0$）： $$b_1 = S_{12}\cdot a_2 = \frac{2}{3}\cdot\frac{E}{3} = \frac{2E}{9} \quad (\text{左向き})$$ $$b_2 = S_{22}\cdot a_2 = -\frac{1}{3}\cdot\frac{E}{3} = -\frac{E}{9} \quad (\text{右向き})$$ $$\Delta V_{BB'} = a_2 + b_2 = \frac{E}{3} - \frac{E}{9} = \frac{2E}{9}$$ $$V_{BB'} = \frac{E}{3} + \frac{2E}{9} = \frac{3E}{9} + \frac{2E}{9} = \frac{5E}{9}$$ $2E/9$ は AA' へ吸収。$-E/9$ は CC' へ（$t=4\ell/c$ 到達→BB'に戻るのは $5\ell/c$、範囲外）。
電圧グラフ（$x=\ell$, BB'）：
電圧 V 5E/9 ───────────────────────────── │ E/3 ─────────────────────┤ │ │ 0 ────────┤ │ 0 ℓ/c 3ℓ/c 4ℓ/c → t
変化時刻と電圧値： $$t = \frac{\ell}{c}:\ V = \frac{E}{3} \qquad t = \frac{3\ell}{c}:\ V = \frac{5E}{9}$$ （直流定常値は $V=E/2$：電源 $E$ を $R+R_{BB'}$ で分圧）

大問2　LC 共振回路の過渡応答

$C_1, C_2, L$ の閉ループ（$t=0$ でスイッチ S オン）。$v_1(0)=E$，$v_2(0)=0$，$i(0)=0$。

問1)　振動角周波数 $\omega$

KVL: $v_1 - L\dfrac{di}{dt} - v_2 = 0$ 素子方程式：$i = -C_1\dfrac{dv_1}{dt}$，$i = C_2\dfrac{dv_2}{dt}$ 電荷保存 $\dfrac{d}{dt}(C_1v_1+C_2v_2)=0$ より：$C_1v_1+C_2v_2 = C_1E$（一定） よって $v_1 = E - \dfrac{C_2}{C_1}v_2$。KVL に代入して微分方程式化： $$\frac{di}{dt} = \frac{v_1-v_2}{L} = \frac{E - v_2(1+C_2/C_1)}{L} = \frac{C_1 E - v_2(C_1+C_2)}{LC_1}$$ $i = C_2 \dfrac{dv_2}{dt}$ で $\dfrac{d^2 i}{dt^2} + \dfrac{C_1+C_2}{LC_1C_2}\,i = 0$
$$\boxed{\omega = \sqrt{\frac{C_1+C_2}{LC_1C_2}} = \frac{1}{\sqrt{L C_{\rm eq}}}, \quad C_{\rm eq} = \frac{C_1C_2}{C_1+C_2} \quad (\text{直列合成容量})}$$

問2)　電流 $i(t)$

$i(0)=0$ より $i(t) = A\sin(\omega t)$。 初期条件 $\dfrac{di}{dt}\big|_{t=0} = \dfrac{v_1(0)-v_2(0)}{L} = \dfrac{E}{L}$： $$A\omega = \frac{E}{L} \Rightarrow A = \frac{E}{\omega L}$$
$$\boxed{i(t) = \frac{E}{\omega L}\sin(\omega t) = E\sqrt{\frac{C_1C_2}{L(C_1+C_2)}}\sin(\omega t)}$$

問3)　$v_2(t)$ の最大値

$v_2(t) = \dfrac{1}{C_2}\displaystyle\int_0^t i\,dt' = \dfrac{E}{\omega L C_2}(1-\cos\omega t)$ 最大値は $\cos(\omega t) = -1$（$t = \pi/\omega$）のとき。 エネルギーで確認：$t=\pi/\omega$ で $i=0$。電荷保存 $C_1v_1 + C_2v_2 = C_1E$ と エネルギー保存 $\dfrac{1}{2}C_1E^2 = \dfrac{1}{2}C_1v_1^2 + \dfrac{1}{2}C_2v_2^2$ より $v_{2,\max} = \dfrac{2C_1 E}{C_1+C_2}$。
$$\boxed{v_{2,\max} = \frac{2C_1 E}{C_1+C_2}}$$ （$C_1 = C_2$ のとき $v_{2,\max} = E$：全エネルギーが $C_2$ に移行）

大問3　三相対称 RLC 回路（Y結線）

$f=50\,\mathrm{Hz}$、線間電圧 $V_L=200\,\mathrm{V}$、$R=10\,\Omega$、$X_L = 2\pi fL = 10\,\Omega$、$X_C = 1/(2\pi fC) = 20\,\Omega$。RLC 並列，Y結線。有効数字2桁。

問1)　相電圧

$$V_{ph} = \frac{V_L}{\sqrt{3}} = \frac{200}{\sqrt{3}} \approx \boxed{1.2 \times 10^2 \text{ V} = 120\text{ V}}$$

問2)　各素子の電流（実効値）

RLC 並列なので各素子に相電圧 $V_{ph}$ がかかる： $$I_R = \frac{V_{ph}}{R} = \frac{200/\sqrt{3}}{10} = \frac{20}{\sqrt{3}} \approx 12\text{ A}$$ $$I_L = \frac{V_{ph}}{X_L} = \frac{200/\sqrt{3}}{10} \approx 12\text{ A}$$ $$I_C = \frac{V_{ph}}{X_C} = \frac{200/\sqrt{3}}{20} = \frac{10}{\sqrt{3}} \approx 5.8\text{ A}$$
$$\boxed{I_R \approx 12\text{ A},\quad I_L \approx 12\text{ A},\quad I_C \approx 5.8\text{ A}}$$

問3)　線電流 $I$

並列アドミタンス： $$\dot{Y} = \frac{1}{R} + j\left(\frac{1}{X_C} - \frac{1}{X_L}\right) = \frac{1}{10} + j\left(\frac{1}{20}-\frac{1}{10}\right) = 0.1 - j\,0.05 \text{ [S]}$$ $$|Y| = \sqrt{0.1^2+0.05^2} = \sqrt{0.0125} = \frac{1}{4\sqrt{5}} \approx 0.112\text{ S}$$ $$I = V_{ph}\cdot|Y| = \frac{200}{\sqrt{3}}\cdot\frac{1}{4\sqrt{5}} = \frac{200}{4\sqrt{15}} = \frac{50}{\sqrt{15}} \approx 12.9\text{ A}$$
$$\boxed{I \approx 13\text{ A}}$$ （位相角：$\phi = \arctan(-0.05/0.1) = \arctan(-0.5) \approx -26.6°$，遅れ無効電力あり）

問4)　$\Delta$結線コンデンサ $C_0$ 追加：$I$ 最小化

$\Delta$ 結線容量のY等価アドミタンス：$B_{Y} = 3/X_{C_0}$（$\Delta$-Y変換で容量 3 倍）。 $I$ 最小 ↔ 合成アドミタンスの虚部 = 0（力率1）： $$\mathrm{Im}[\dot{Y}] + \frac{3}{X_{C_0}} = 0 \Rightarrow -0.05 + \frac{3}{X_{C_0}} = 0$$ $$X_{C_0} = \frac{3}{0.05} = 60\,\Omega \quad (\text{各}\Delta\text{素子のリアクタンス})$$ このとき合成アドミタンス = $0.1\text{ S}$（純抵抗）： $$I_{\min} = V_{ph}\times 0.1 = \frac{200}{\sqrt{3}}\times 0.1 = \frac{20}{\sqrt{3}} \approx 11.5\text{ A} \approx 12\text{ A}$$
$$\boxed{X_{C_0}=60\,\Omega \quad (\Delta \text{各素子}),\quad I_{\min} = \frac{20}{\sqrt{3}} \approx 12\text{ A}}$$ （力率1（=1）に改善。問3の 13 A から削減）

大問4　演算増幅器回路

問1)　図4.1 の利得 $V_2/V_1$

入力インピーダンス（直列 $R_1$, $C_1$）：$Z_{in} = R_1 + \dfrac{1}{j\omega C_1} = \dfrac{1+j\omega R_1C_1}{j\omega C_1}$ 帰還インピーダンス（並列 $R_2 \parallel C_2$）：$Z_f = \dfrac{R_2}{1+j\omega R_2C_2}$ 反転増幅の利得：$V_2/V_1 = -Z_f/Z_{in}$： $$\frac{V_2}{V_1} = -\frac{R_2/(1+j\omega R_2C_2)}{(1+j\omega R_1C_1)/(j\omega C_1)} = -\frac{j\omega R_2C_1}{(1+j\omega R_1C_1)(1+j\omega R_2C_2)}$$
$$\boxed{\frac{V_2}{V_1} = \frac{-j\omega R_2 C_1}{(1+j\omega R_1C_1)(1+j\omega R_2C_2)}}$$ バンドパスフィルタ（低域遮断：$\omega R_1C_1=1$、高域遮断：$\omega R_2C_2=1$）

問2)　素子値の設計（$f_L=1\,\mathrm{kHz}$, $f_H=100\,\mathrm{kHz}$, 最大利得40 dB, $R_1=1\,\mathrm{k\Omega}$）

中域利得（$f_L \ll f \ll f_H$）：$j\omega R_1C_1\gg1$, $j\omega R_2C_2 \ll 1$ で： $$\left|\frac{V_2}{V_1}\right|_{\rm mid} \approx \frac{R_2}{R_1} = 40\text{ dB} = 100$$ $$R_2 = 100R_1 = 100\times 1\text{ k}\Omega = 100\text{ k}\Omega$$ 低域遮断 $\omega_L R_1C_1 = 1$（$f_L = 1\text{ kHz}$）： $$C_1 = \frac{1}{\omega_L R_1} = \frac{1}{2\pi\times 10^3\times 10^3} = \frac{1}{2\pi\times 10^6} \approx 160\text{ nF}$$ 高域遮断 $\omega_H R_2C_2 = 1$（$f_H = 100\text{ kHz}$）： $$C_2 = \frac{1}{\omega_H R_2} = \frac{1}{2\pi\times 10^5\times 10^5} = \frac{1}{2\pi\times 10^{10}} \approx 16\text{ pF}$$
$$\boxed{R_2 = 100\text{ k}\Omega,\quad C_1 \approx 160\text{ nF},\quad C_2 \approx 16\text{ pF}}$$

問3)　10 Hz 三角波（波高値 0.1 V）の入力・出力波形概形

$f = 10\text{ Hz} \ll f_L = 1\text{ kHz}$：低周波域では $|\omega R_1C_1| \ll 1$ のため $Z_{in} \approx 1/(j\omega C_1)$（容量性）、$Z_f \approx R_2$。 → 回路が微分器として動作。 三角波の微分 = 方形波（スロープの正負が交互入れ替わり）。
  • 入力：三角波（周期 0.1 s、振幅 $\pm0.1\text{ V}$）
  • 出力：方形波（符号反転）。振幅 $|V_2| = R_2C_1\left|\dfrac{dV_1}{dt}\right| = R_2C_1\times 4\times 0.1\times 10 \approx 64\text{ mV}$
（出力は入力三角波の微分なので、入力が正の傾きのとき出力は負の一定値、負の傾きのとき正の一定値）

問4)　図4.2 の出力電流 $i_0$（$R_1 \gg R_2, R_3$）

Howland 型電流源回路。4つの $R_1$（うち2つは電圧帰還路）、$R_2$ が電流出力段、$R_3$ は負荷。 仮想短絡 $V_- = V_+$。$V_+$ は $R_1$ 分圧（$R_1$：出力ノード $N$ から $V_+$、$R_1$：$V_+$ からグラウンド）： $$V_+ = \frac{N}{2}$$ $V_-$ での KCL（$R_1 \gg R_2$ 近似で $R_1$入力電流は無視）： $$\frac{v_1 - V_-}{R_1} + \frac{V_{\rm out}-V_-}{R_1} = 0 \Rightarrow v_1 + V_{\rm out} = 2V_- = N$$ 出力ノード $N$：$V_{\rm out} = N - v_1$（KVL より）。$R_2$ 電圧：$V_{\rm out}-N = -v_1 = R_2\times i_0$... → KCL at $N$：$(V_{\rm out}-N)/R_2 = N/R_3 + i_{R_1}$（$i_{R_1}\approx 0$ で $R_1\gg$）： $$\frac{-v_1}{R_2} = \frac{N}{R_3} \Rightarrow N = -\frac{v_1 R_3}{R_2}$$ $$i_0 = \frac{N}{R_3} = -\frac{v_1}{R_2}$$
$$\boxed{i_0 = -\frac{v_1}{R_2}}$$ $i_0 = -\dfrac{v_1}{R}$（$R=1\,\Omega$）となる条件： $$\boxed{R_2 = R = 1\,\Omega \quad (\text{かつ } R_1 \gg R_2, R_3 = 1\,\Omega\text{ を満たすこと})}$$ （Howland 電流ポンプ：$R_2$ の値のみで出力電流を決定、$R_3$ 負荷に依存しない）
