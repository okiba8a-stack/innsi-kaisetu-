電磁気学　解答解説

東京工業大学 大学院修士課程入試問題（平成26年8月19日実施）
専門科目 電気電子工学・電子物理工学（午後2）

大問1：平行平板コンデンサと誘電体

【問題設定】　一辺 \(a\)、間隔 \(b\) の平行平板コンデンサに、同サイズの誘電体板を挿入する。

1-1a）容量 \(C\) を求めよ

考え方：誘電体（誘電率 \(\varepsilon_f\)）領域と真空（\(\varepsilon_0\)）領域を並列コンデンサとみなす。

  • 誘電体部分：面積 \(ax\)、容量 \(C_1 = \dfrac{\varepsilon_f \cdot ax}{b}\)
  • 真空部分：面積 \(a(a-x)\)、容量 \(C_2 = \dfrac{\varepsilon_0 \cdot a(a-x)}{b}\)
$$C = C_1 + C_2 = \frac{a}{b}\bigl[\varepsilon_f x + \varepsilon_0(a-x)\bigr] = \frac{a}{b}\bigl[\varepsilon_0 a + (\varepsilon_f - \varepsilon_0)x\bigr]$$

1-1b）誘電体板に働く力 \(F\)

考え方：電荷 \(Q\) が一定条件で仮想仕事の原理を適用する。

静電エネルギー \(U = \dfrac{Q^2}{2C}\) として

$$F = -\left.\frac{\partial U}{\partial x}\right|_Q = \frac{Q^2}{2C^2}\cdot\frac{\partial C}{\partial x}$$

\(\dfrac{\partial C}{\partial x} = \dfrac{a(\varepsilon_f - \varepsilon_0)}{b} > 0\)（\(\varepsilon_f > \varepsilon_0\) より）

$$\boxed{F = \frac{Q^2(\varepsilon_f - \varepsilon_0)\,b}{2a\left[\varepsilon_0 a + (\varepsilon_f-\varepsilon_0)x\right]^2}}$$

方向：+x 方向（誘電体をコンデンサ内部に引き込む向き）

　誘電体が入るほど容量が増えエネルギーが下がる（\(\partial U/\partial x < 0\)）ため、引き込まれる。

1-2）誘電率が \(x\) の関数 \(\varepsilon(x) = \varepsilon_1 + \dfrac{\varepsilon_2-\varepsilon_1}{a}x\) の場合

考え方：誘電率は電界方向（\(y\) 方向）ではなく板面内（\(x\) 方向）で変化する。 微小幅 \(dx\) のストリップを並列コンデンサとして積分する。

$$dC = \frac{\varepsilon(x)\cdot a\,dx}{b}$$ $$C = \int_0^a \frac{a}{b}\,\varepsilon(x)\,dx = \frac{a}{b}\int_0^a\!\left[\varepsilon_1 + \frac{\varepsilon_2-\varepsilon_1}{a}x\right]dx$$ $$= \frac{a}{b}\left[\varepsilon_1 a + \frac{(\varepsilon_2-\varepsilon_1)a}{2}\right]$$
$$\boxed{C = \frac{a^2(\varepsilon_1+\varepsilon_2)}{2b}}$$

（誘電率を \(x\) で平均した値 \(\dfrac{\varepsilon_1+\varepsilon_2}{2}\) に対応）

大問2：平行導体板間の電荷分布

【問題設定】　導体 A（\(x=-d/3\)、接地）と導体 B（\(x=2d/3\)、電荷 0）の間に体積電荷が分布。
\(-d/3 < x \le 0\)：\(\rho_1 = -2\rho_0\)、\(0 \le x < 2d/3\)：\(\rho_2 = \rho_0\)

2-1）境界条件

まず板間の総電荷を確認する：

$$Q_{total} \propto \rho_1\cdot\frac{d}{3} + \rho_2\cdot\frac{2d}{3} = -2\rho_0\cdot\frac{d}{3} + \rho_0\cdot\frac{2d}{3} = 0$$

総電荷 \(= 0\)、導体 B 電荷 \(= 0\)　⟹　導体 A の電荷も \(= 0\)

\(x = -d/3\)（導体 A）：　外部（左側）に電界なし、A の表面電荷 \(= 0\)

$$E\!\left(-\frac{d}{3}\right) = 0$$

\(x = 2d/3\)（導体 B）：　導体 B 電荷 \(= 0\)、右外部に電界なし

$$E\!\left(\frac{2d}{3}\right) = 0$$
ガウスの法則による説明：導体板を囲む閉曲面を取ると、内部の自由電荷量 = ε₀ × 流出する電束　の関係が成立。導体 B の電荷 = 0 かつ右側に電界なし → 左側の電界も 0。同様に総電荷 = 0 より A でも電界 = 0。

2-2）電位・電界分布と \(V_B\)

ガウスの法則 \(\dfrac{dE}{dx} = \dfrac{\rho}{\varepsilon_0}\) を各領域で積分し、境界条件 + 連続条件を適用する。

■ 領域1（\(-d/3 \le x \le 0\)）

$$\frac{dE}{dx} = \frac{-2\rho_0}{\varepsilon_0} \Rightarrow E(x) = -\frac{2\rho_0}{\varepsilon_0}x + C_1$$

境界条件 \(E(-d/3)=0\)：

$$0 = \frac{2\rho_0 d}{3\varepsilon_0} + C_1 \Rightarrow C_1 = -\frac{2\rho_0 d}{3\varepsilon_0}$$
$$E(x) = -\frac{2\rho_0}{\varepsilon_0}\!\left(x + \frac{d}{3}\right) \qquad \left(-\frac{d}{3} \le x \le 0\right)$$ $$V(x) = \frac{\rho_0}{\varepsilon_0}\!\left(x + \frac{d}{3}\right)^2 \qquad \bigl[V(-d/3) = 0 \text{ を満足}\bigr]$$

■ 領域2（\(0 \le x \le 2d/3\)）

$$\frac{dE}{dx} = \frac{\rho_0}{\varepsilon_0} \Rightarrow E(x) = \frac{\rho_0}{\varepsilon_0}x + C_2$$

\(x=0\) で連続：\(C_2 = E(0) = -\dfrac{2\rho_0 d}{3\varepsilon_0}\)

$$E(x) = \frac{\rho_0}{\varepsilon_0}\!\left(x - \frac{2d}{3}\right) \qquad \left(0 \le x \le \frac{2d}{3}\right)$$ $$V(x) = \frac{\rho_0}{\varepsilon_0}\!\left(-\frac{x^2}{2} + \frac{2dx}{3} + \frac{d^2}{9}\right)$$

確認：\(E(2d/3) = 0\) ✓

■ 導体 B の電位

$$V_B = V\!\left(\frac{2d}{3}\right) = \frac{\rho_0}{\varepsilon_0}\!\left(-\frac{2d^2}{9} + \frac{4d^2}{9} + \frac{d^2}{9}\right) = \boxed{\frac{\rho_0 d^2}{3\varepsilon_0}}$$

2-3）導体 A の単位面積当たり電荷密度

$$\sigma_A = \varepsilon_0 \cdot E\!\left(-\frac{d}{3}^+\right) = \varepsilon_0 \times 0 = \boxed{0}$$

（板間総電荷 \(= 0\) のため、導体に電荷は誘起されない）

2-4）正負の境界が \(x = a\) の場合の条件

境界条件 \(E(-d/3) = E(2d/3) = 0\) が成立するには板間の総電荷 = 0 が必要：

$$\boxed{\rho_1\!\left(a + \frac{d}{3}\right) + \rho_2\!\left(\frac{2d}{3} - a\right) = 0}$$

確認（\(a=0\)）：\(-2\rho_0\cdot\dfrac{d}{3} + \rho_0\cdot\dfrac{2d}{3} = 0\) ✓

大問3：相互インダクタンスと回転コイル

【問題設定】　無限長直線導線（\(y\) 軸上）と矩形コイル（幅 \(2a\)、高さ \(b\)、中心 \(x=R\)）。

3-1a）距離 \(x\) での磁束密度 \(B(x)\)

$$\boxed{B(x) = \frac{\mu_0 I_1}{2\pi x}}$$

方向：\(z\) 方向（アンペールの法則・右ねじ則より）

3-1b）相互インダクタンス \(M\)

$$\Phi = \int_{R-a}^{R+a} \frac{\mu_0 I_1}{2\pi x}\cdot b\,dx = \frac{\mu_0 I_1 b}{2\pi}\Bigl[\ln x\Bigr]_{R-a}^{R+a}$$
$$\boxed{M = \frac{\Phi}{I_1} = \frac{\mu_0 b}{2\pi}\ln\frac{R+a}{R-a}}$$

3-1c）導線とコイル間の力 \(F\)

左辺 AB（\(x = R-a\)、電流 \(I_2\) が \(I_1\) と同方向）→ 引力

右辺 CD（\(x = R+a\)、電流 \(I_2\) が逆方向）→ 斥力

$$F = \frac{\mu_0 I_1 I_2 b}{2\pi}\!\left(\frac{1}{R-a} - \frac{1}{R+a}\right) = \frac{\mu_0 I_1 I_2 b}{2\pi}\cdot\frac{2a}{R^2-a^2}$$
$$\boxed{F = \frac{\mu_0 I_1 I_2 ab}{\pi(R^2 - a^2)}}$$

方向：導線側（\(-x\) 方向）（引力が斥力を上回る）

3-2a）回転後の磁束 \(\Phi(\theta)\)

重要：コイルが傾くと、\(y\) 軸上の導線による磁場は \(z\) 成分だけでなく \(x\) 成分も持つ。

$$\boldsymbol{B} = \frac{\mu_0 I_1}{2\pi\rho^2}(-z\,\hat{x} + x\,\hat{z}), \quad \rho^2 = x^2 + z^2$$

回転後のコイル上の点（パラメータ \(r \in [-a,a]\)）：\((R+r\cos\theta,\ y,\ r\sin\theta)\)

面素ベクトル：\(d\boldsymbol{A} = (-\sin\theta,\ 0,\ \cos\theta)\,dr\,dy\)

内積を計算：

$$\boldsymbol{B}\cdot d\boldsymbol{A} = \frac{\mu_0 I_1}{2\pi}\cdot\frac{r + R\cos\theta}{R^2 + 2Rr\cos\theta + r^2}\,dr\,dy$$

\(u = R^2 + 2Rr\cos\theta + r^2\) と置換すると \((r + R\cos\theta)\,dr = du/2\)

$$\Phi = \frac{\mu_0 I_1 b}{2\pi}\int_{-a}^{a}\frac{r+R\cos\theta}{R^2+2Rr\cos\theta+r^2}\,dr = \frac{\mu_0 I_1 b}{4\pi}\Bigl[\ln u\Bigr]_{r=-a}^{r=a}$$
$$\boxed{\Phi(\theta) = \frac{\mu_0 I_1 b}{4\pi}\ln\frac{R^2 + a^2 + 2Ra\cos\theta}{R^2 + a^2 - 2Ra\cos\theta}}$$
余弦定理との対応：
\(R^2 + a^2 + 2Ra\cos\theta\) = コイル遠端から \(y\) 軸までの距離の 2 乗
\(R^2 + a^2 - 2Ra\cos\theta\) = コイル近端から \(y\) 軸までの距離の 2 乗
（余弦定理 \(C^2 = R^2 + a^2 - 2Ra\cos\gamma\) に相当）

確認：\(\theta=0\) で \(\Phi = \dfrac{\mu_0 I_1 b}{2\pi}\ln\dfrac{R+a}{R-a} = MI_1\) ✓　　\(\theta=\pi/2\) で \(\Phi = 0\) ✓

3-2b）誘導起電力 \(V\)（\(\theta = \omega t\)）

$$\frac{d\Phi}{d\theta} = \frac{\mu_0 I_1 b}{4\pi}\cdot\frac{-2Ra\sin\theta \cdot 2(R^2+a^2)}{(R^2+a^2)^2 - 4R^2a^2\cos^2\theta}$$ $$= -\frac{\mu_0 I_1 b Ra\sin\theta(R^2+a^2)}{\pi\bigl[(R^2+a^2)^2 - 4R^2a^2\cos^2\theta\bigr]}$$ $$V = -\frac{d\Phi}{dt} = -\omega\frac{d\Phi}{d\theta}\bigg|_{\theta=\omega t}$$
$$\boxed{V = \frac{\mu_0 I_1 abR\,(R^2+a^2)\,\omega\sin(\omega t)}{\pi\bigl[(R^2+a^2)^2 - 4R^2a^2\cos^2(\omega t)\bigr]}}$$

分母の因数分解：\((R^2+a^2)^2 - 4R^2a^2\cos^2\theta = (R^2+a^2+2Ra\cos\theta)(R^2+a^2-2Ra\cos\theta)\)

解答まとめ

問キーポイント主な答え
1-1a並列コンデンサとして合成\(\dfrac{a}{b}[\varepsilon_0 a + (\varepsilon_f-\varepsilon_0)x]\)
1-1b定電荷・仮想仕事の原理\(\dfrac{Q^2(\varepsilon_f-\varepsilon_0)b}{2a[\varepsilon_0 a+(\varepsilon_f-\varepsilon_0)x]^2}\)（+x方向）
1-2微小ストリップを並列積分\(\dfrac{a^2(\varepsilon_1+\varepsilon_2)}{2b}\)
2-1板間総電荷 = 0 → 両端 E = 0\(E(\pm\text{端点}) = 0\)
2-2Gauss 法則を区分積分\(V_B = \dfrac{\rho_0 d^2}{3\varepsilon_0}\)
2-3表面電荷 = ε₀ × E（境界）\(\sigma_A = 0\)
2-4総電荷 = 0 の条件\(\rho_1(a+d/3)+\rho_2(2d/3-a)=0\)
3-1aAmpere の法則\(\dfrac{\mu_0 I_1}{2\pi x}\)
3-1b磁束を積分して M 導出\(\dfrac{\mu_0 b}{2\pi}\ln\dfrac{R+a}{R-a}\)
3-1c両辺への力の合成（引力優位）\(\dfrac{\mu_0 I_1 I_2 ab}{\pi(R^2-a^2)}\)（−x方向）
3-2aB 場の全成分 + 置換積分（余弦定理対応）\(\dfrac{\mu_0 I_1 b}{4\pi}\ln\dfrac{R^2+a^2+2Ra\cos\theta}{R^2+a^2-2Ra\cos\theta}\)
3-2bΦ(θ) を t 微分\(\dfrac{\mu_0 I_1 abR(R^2+a^2)\omega\sin\omega t}{\pi[(R^2+a^2)^2-4R^2a^2\cos^2\omega t]}\)
