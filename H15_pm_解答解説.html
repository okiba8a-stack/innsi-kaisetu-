問題1：電力・エネルギー

設問1-a）基準値の表現

インピーダンス基準値（相電圧 $E_B$、1相容量 $S_B$ を用いる）： $$Z_B = \frac{E_B^2}{S_B}$$ 電流基準値（線間電圧 $V_B$、三相容量 $S_B'$ を用いる）： $$I_B = \frac{S_B'}{\sqrt{3}\,V_B}$$
三相系では $S_B' = 3S_B$。線間電圧 $V_B = \sqrt{3}E_B$。これを使うと $Z_B = V_B^2/S_B'$ とも書ける。

設問1-b）単位法電流（基準：66 kV, 100 MVA）

複素電力（潮流）：$\dot{S} = 40 + j5$ [MVA]（遅れ無効電力は正）
基準で正規化：$\dot{S}_{\rm pu} = (40+j5)/100 = 0.40 + j0.05$ [pu]
電圧：$\dot{V}_{\rm pu} = 65/66 = 0.9848$ [pu]（位相角 $0°$ を参照）
電流 $\dot{I} = \dot{S}^*/\dot{V}^*$： $$\dot{I}_{\rm pu} = \frac{0.40 - j0.05}{0.9848} = 0.406 - j0.0508 \text{ [pu]}$$
$$|\dot{I}_{\rm pu}| = \frac{\sqrt{0.40^2+0.05^2}}{0.9848} \approx \boxed{0.41 \text{ [pu]}}$$ 位相角 $\approx -7.1°$（遅れ）

設問1-c）系統全体の％インピーダンス（基準：1000 MVA）

基準容量変換：$Z_{\rm new} = Z_{\rm old} \times (S_{\rm base,new}/S_{\rm base,old})$

設問1-d）変圧器-1 二次側三相地絡故障電流

地絡点（T1二次側）から見た電源側の合計インピーダンス（発電機 + T1）： $$Z = j45 + j40 = j85\% = j0.85 \text{ [pu]}$$ 発電機は定格電圧で運転 → $E = 1.0$ [pu]
地絡電流： $$I = \frac{E}{Z} = \frac{1.0}{j0.85} \approx \boxed{1.176 \text{ [pu]}}$$
実際の電流値への換算（基準：1000 MVA）：
T1 一次側（15 kV）の基準電流：$I_{B1} = \dfrac{1000 \times 10^6}{\sqrt{3}\times 15\times 10^3} \approx 38{,}490$ [A]

$$I_1 = 1.176 \times 38{,}490 \approx \boxed{4.5 \times 10^4 \text{ [A]}}$$ $$I_2 = 1.176 \times 2{,}099 \approx \boxed{2.5 \times 10^3 \text{ [A]}}$$
パーユニット法では変圧比は吸収されるため、一次・二次の pu 電流は等しい（1.176 pu）。実電流は各電圧レベルの基準電流で換算する。

設問1-f）2ノード系統の電力方程式

アドミタンス $\dot{Y} = G + jB$、電圧 $\dot{E}_1 = V_1 e^{j\theta_1}$, $\dot{E}_2 = V_2 e^{j\theta_2}$、$\delta = \theta_1 - \theta_2$。
電流：$\dot{I}_1 = \dot{Y}(\dot{E}_1 - \dot{E}_2)$
複素電力：$\dot{S}_1 = P_1 + jQ_1 = \dot{E}_1 \dot{I}_1^*$ $$= (G-jB)(V_1^2 - V_1 V_2 e^{-j\delta})$$
ノード1（送電端）： $$P_1 = G V_1^2 - V_1 V_2(G\cos\delta + B\sin\delta)$$ $$Q_1 = -B V_1^2 - V_1 V_2(G\sin\delta - B\cos\delta)$$ ノード2（受電端）：$\dot{S}_2 = \dot{E}_2(-\dot{I}_1)^*$ $$P_2 = G V_2^2 - V_1 V_2(G\cos\delta - B\sin\delta)$$ $$Q_2 = -B V_2^2 + V_1 V_2(G\sin\delta + B\cos\delta)$$
$G=0$（純リアクタンス線路）のとき：$P = V_1 V_2 B\sin\delta$（電力 ∝ $\sin\delta$）が得られる。

設問2-a）同期電動機の定格回転数とトルク

4極（極対数 $p=2$）、50 Hz： $$n_s = \frac{60f}{p} = \frac{60\times50}{2} = \boxed{1500 \text{ [rpm]}}$$ $$T = \frac{P}{\omega_s} = \frac{10000}{2\pi\times1500/60} = \frac{10000}{157.1} \approx \boxed{63.7 \text{ [N·m]}}$$

設問2-b）力率1のフェーザ図と無負荷誘導起電力

相電圧（Y接続）：$V_a = 200/\sqrt{3} = 115.5$ V。力率1 → 電機子電流 $I_a$ は $V_a$ と同位相。 $$I_a = \frac{P}{3V_a} = \frac{10000}{3\times115.5} = 28.87 \text{ [A]}$$ 誘導起電力（電動機では $\dot{E}_a = \dot{V}_a - jX\dot{I}_a$）： $$\dot{E}_a = 115.5 - j \times 3 \times 28.87 = 115.5 - j86.6 \text{ [V]}$$
$$|\dot{E}_a| = \sqrt{115.5^2 + 86.6^2} \approx \boxed{144 \text{ [V]（相電圧）}}$$ フェーザ図：$V_a$（実軸）→ $jXI_a$（上向き）→ $E_a$（右下方向）

設問2-c）界磁電流増加時の挙動

力率1の状態から界磁電流を増加させると：

  • 無負荷誘導起電力 $E_a$：界磁電流 $I_f$ に比例して増加
  • 電機子電流 $I_a$：増加し、位相は進み（進相）になる
過励磁の同期電動機は電力系統に対して進相電流を供給→コンデンサと等価に機能し、力率改善・電圧支持に利用される。

設問2-d）脱調直前の電源電圧

$E_a = 200$ V（相）一定。定格負荷 $P = 10$ kW。電源電圧を $V_a$ として低下させる。
脱調（プルアウト）は負荷角 $\delta = 90°$ のとき（最大出力点）： $$P_{\max} = \frac{3V_a E_a}{X}$$ 定格出力と等しくなる条件： $$\frac{3\times V_a \times 200}{3} = 10000$$ $$200 V_a = 10000 \implies V_a = 50 \text{ [V]}（相電圧）$$
線間電圧 $= 50\sqrt{3} \approx \boxed{86.6 \text{ [V]}}$

設問3-a）チョッパ回路の電流変化率

SW オン（$D$ が逆バイアス）： $$\frac{di_L}{dt} = \frac{E - V_O}{L}$$ SW オフ（$D$ が導通）： $$\frac{di_L}{dt} = -\frac{V_O}{L}$$

設問3-b）$V_O = E/4$ のときの $T_{on}/T_{off}$

定常状態ではインダクタの平均電圧 = 0： $$(E-V_O) T_{on} = V_O T_{off}$$ $$\left(E - \frac{E}{4}\right) T_{on} = \frac{E}{4} T_{off}$$ $$\frac{3E}{4} T_{on} = \frac{E}{4} T_{off}$$
$$\frac{T_{on}}{T_{off}} = \frac{1}{3}$$ デューティ比 $D = T_{on}/(T_{on}+T_{off}) = 1/4$，$V_O = D\cdot E = E/4$ と一致。

設問3-c）ダイオードDの役割

役割：SW オフ時にインダクタに蓄えられたエネルギーによる電流の還流路（フリーホイール）を提供する。

D がない場合：SW オフ時に電流の流路が断たれ、インダクタに過大なサージ電圧が発生してスイッチ素子を破壊する恐れがある。また、出力電流が不連続となり制御が困難になる。

問題2：波動・通信

設問1-a）1周期の放射エネルギー $\delta h$

振動：$y = a\sin(2\pi ft)$，加速度：$\ddot{y} = -a(2\pi f)^2\sin(2\pi ft)$
加速度の2乗の時間平均：$\langle b^2 \rangle = \dfrac{a^2(2\pi f)^4}{2}$
平均放射パワー：$\overline{W} = \dfrac{d q^2 \langle b^2 \rangle}{c^3}$
1周期 $T = 1/f$：
$$\delta h = \overline{W} \cdot \frac{1}{f} = \frac{d q^2 a^2 (2\pi f)^4}{2 c^3 f} = \frac{8\pi^4 d q^2 a^2 f^3}{c^3}$$

設問1-b）$\delta h / h_0$ の計算

力学的エネルギー（振幅 $a$，角周波数 $\omega = 2\pi f$）： $$h_0 = \frac{1}{2}m(a\omega)^2 = \frac{1}{2}m(2\pi fa)^2 = 2\pi^2 m f^2 a^2$$ $$\frac{\delta h}{h_0} = \frac{8\pi^4 dq^2 a^2 f^3/c^3}{2\pi^2 m f^2 a^2} = \frac{4\pi^2 dq^2 f}{m c^3}$$
数値代入（$\pi\approx3$）： $$\frac{\delta h}{h_0} = \frac{4\times9\times6\times10^9\times(10^{-4})^2\times10^7}{10^{-5}\times(3\times10^8)^3} = \frac{4\times9\times6\times10^9\times10^{-8}\times10^7}{10^{-5}\times2.7\times10^{25}}$$ $$= \frac{216\times10^8}{2.7\times10^{20}} \approx 8\times10^{-11}$$
$$\frac{\delta h}{h_0} \approx \boxed{10^{-10}}$$

設問1-c）エネルギーが $e^{-1}$ 倍になる時間 $\Delta t$

エネルギー減衰率 $\gamma = f \cdot \dfrac{\delta h}{h_0} = \dfrac{4\pi^2 dq^2 f^2}{mc^3}$
（1秒間に $f$ 回の振動があり，各振動で $\delta h/h_0$ のエネルギーを失う）
$h = h_0 e^{-\gamma t}$ より，$h = e^{-1}h_0$ のとき $\gamma \Delta t = 1$： $$\Delta t = \frac{1}{\gamma} = \frac{mc^3}{4\pi^2 dq^2 f^2}$$
数値代入： $$\Delta t = \frac{10^{-5}\times(3\times10^8)^3}{4\times9\times6\times10^9\times(10^{-4})^2\times(10^7)^2} = \frac{10^{-5}\times2.7\times10^{25}}{216\times10^9\times10^{-8}\times10^{14}} = \frac{2.7\times10^{20}}{2.16\times10^{16}} \approx 10^4 \text{ s}$$
$$\Delta t \approx \boxed{10^4 \text{ [s]}}$$

設問1-d）M に働くクーロン力

C$(0,\ell)$ からの力（斥力，下向き）＋ D$(0,-\ell)$ からの力（斥力，上向き）： $$F = \frac{qq_0}{4\pi\varepsilon_0}\left[\frac{1}{(y+\ell)^2} - \frac{1}{(\ell-y)^2}\right]$$ $y \ll \ell$ で展開：$\dfrac{1}{(\ell\pm y)^2} \approx \dfrac{1}{\ell^2}\left(1 \mp \dfrac{2y}{\ell}\right)$
$$F \approx \frac{qq_0}{4\pi\varepsilon_0}\cdot\frac{-4y}{\ell^3} = -\frac{qq_0 y}{\pi\varepsilon_0 \ell^3}$$ （$q, q_0$ 同符号のとき復元力方向，$y>0$ で $F<0$）

設問1-e）運動方程式

$$m\ddot{y} = -ky - \frac{qq_0}{\pi\varepsilon_0\ell^3}y = -\left(k + \frac{qq_0}{\pi\varepsilon_0\ell^3}\right)y$$

設問1-f）周波数変化 $\delta f$ と $q_0$ の計算

新しい角周波数：$\omega'^2 = (k + qq_0/(\pi\varepsilon_0\ell^3))/m$，元の $\omega^2 = k/m = (2\pi f)^2$
$$f' = f\sqrt{1+\frac{qq_0}{\pi\varepsilon_0\ell^3 k}} \approx f\left(1+\frac{qq_0}{2\pi\varepsilon_0\ell^3 k}\right)$$ $$\delta f = f' - f = \frac{f qq_0}{2\pi\varepsilon_0\ell^3 k} = \frac{qq_0}{2\pi\varepsilon_0\ell^3 m(2\pi f)^2} \cdot f = \frac{qq_0}{8\pi^3\varepsilon_0 m f^2 \ell^3} \cdot f$$
$q_0$ を求める：$q_0 = \dfrac{\delta f}{f}\cdot\dfrac{8\pi^3\varepsilon_0 m f^2 \ell^3}{q}$
$= 10^{-3}\times\dfrac{8\times27\times9\times10^{-12}\times10^{-5}\times(10^7)^2\times(10^{-2})^3}{10^{-4}}$
$= 10^{-3}\times\dfrac{216\times9\times10^{-12-5+14-6}}{10^{-4}}$ $= 10^{-3}\times1944\times10^{-5} = 1944\times10^{-8} \approx 2\times10^{-5}$ C
$$q_0 \approx \boxed{2\times10^{-5} \text{ [C]}}$$

設問2-a）最適受信フィルタ出力

フィルタのインパルス応答を $h(t)$ とすると： $$y_s(t) = s(t)*h(t) = \int_{-\infty}^{\infty} s(\tau)h(t-\tau)\,d\tau$$ $$y_n(t) = n(t)*h(t) = \int_{-\infty}^{\infty} n(\tau)h(t-\tau)\,d\tau$$

設問2-b）最適フィルタ $h(t)=s(t_0-t)$ での SNR

$h(t) = s(t_0-t)$ を代入，$t=t_0$： $$y_s(t_0) = \int_0^{t_0}s(\tau)h(t_0-\tau)\,d\tau = \int_0^{t_0}s(\tau)s(\tau)\,d\tau = E$$ 分母の積分：$\int_0^{t_0}h^2(t_0-\tau)\,d\tau = \int_0^{t_0}s^2(\tau)\,d\tau = E$
$$\mathrm{SNR} = \frac{E^2}{\frac{1}{2}N_0 E} = \boxed{\frac{2E}{N_0}}$$

設問2-c）矩形パルスの場合の出力波形

$s(t) = A$（$0\le t\le t_0$），$h(t) = s(t_0-t) = A$（$0\le t\le t_0$）のとき，出力は三角波：

  • $0\le t\le t_0$：$y_s(t) = A^2 t$（線形増加）
  • $t_0\le t\le 2t_0$：$y_s(t) = A^2(2t_0-t)$（線形減少）
  • $t=t_0$ でピーク $y_s(t_0) = A^2 t_0$
出力は $t=0$ で 0 から立ち上がり，$t=t_0$ でピーク，$t=2t_0$ で 0 に戻る三角形状。

設問2-d）SNR最大サンプル時刻

$t = t_0$（三角波の頂点，フィルタ出力が最大）でSNRも最大。

設問2-e）シンボル間干渉を生じない条件

ナイキスト（ISI）条件（Nyquist ISI criterion）：$s(t)*h(t)$ が $t = nT$（$n\ne0$）でゼロになること。

設問2-f）ナイキスト条件を満たす $s(t)*h(t)$ の例

代表例：raised cosine（ロールオフフィルタ）の出力波形。

  • $t=0$ でピーク（$=1$）
  • $t = \pm T, \pm 2T, \ldots$ ですべてゼロ
  • $\mathrm{sinc}(t/T)$ 形状（あるいはその変形）

設問2-g）Cauchy-Schwarz 不等式による最適フィルタの証明

SNR 最大化は $y_s^2(t_0) / \left[\frac{1}{2}N_0\int h^2(t_0-\tau)\,d\tau\right]$ の最大化に等しい。
分子を Cauchy-Schwarz 不等式で評価（$g_1(\tau)=s(\tau)$，$g_2(\tau)=h(t_0-\tau)$）： $$y_s^2(t_0) = \left(\int s(\tau)h(t_0-\tau)\,d\tau\right)^2 \le \int s^2(\tau)\,d\tau \cdot \int h^2(t_0-\tau)\,d\tau = E\cdot\int h^2\,d\tau$$
したがって： $$\mathrm{SNR} \le \frac{E\cdot\int h^2\,d\tau}{\frac{1}{2}N_0\int h^2\,d\tau} = \frac{2E}{N_0}$$ 等号成立条件：$g_1(\tau) \propto g_2(\tau)$，すなわち $s(\tau) \propto h(t_0-\tau)$。
最適フィルタ：$h(t) = s(t_0-t)$（時間反転シフト＝整合フィルタ）でSNR $= 2E/N_0$ 達成。

問題3：電気物性

設問1-a）電子移動度と平均自由行程

マシーセンの法則：$\dfrac{1}{\mu} = \dfrac{1}{\mu_{\rm ph}} + \dfrac{1}{\mu_{\rm imp}} = \dfrac{1}{1000} + \dfrac{1}{500} = \dfrac{3}{1000}$ $$\mu = \frac{1000}{3} \approx 333 \text{ cm}^2/\text{Vs}$$ 緩和時間：$\tau = \dfrac{\mu m^*}{e} = \dfrac{333\times10^{-4}\times0.25\times9.1\times10^{-31}}{1.6\times10^{-19}} \approx 4.7\times10^{-13}$ s
平均自由行程：$\lambda = v_{\rm th}\tau = 10^7\times10^{-2}\times4.7\times10^{-13} \approx \boxed{47 \text{ nm}}$

設問1-b）有効質量

i）自由電子：$E = p^2/(2m_0)$

ii）$E = 5p^2/m_0 = p^2/(2m^*)$ と比較 → $m^* = m_0/10$

iii）一般的な表現（有効質量の定義）：

$$m^* = \hbar^2\left(\frac{d^2E}{dp^2}\right)^{-1}$$

設問1-c）P ドープ Si（$N_D = 10^{17}$ cm$^{-3}$）

i） P はドナー → n形。多数キャリア濃度 $n = 10^{17}$ cm$^{-3}$

ii）フェルミ準位（$N_c = 10^{19}$ cm$^{-3}$，$kT = 25$ meV，$\log e = 0.43$）：

$n = N_c\exp\!\left(-\dfrac{E_c-E_F}{kT}\right)$ より $$E_c - E_F = kT\ln\frac{N_c}{N_D} = 25\times\frac{\log(10^{19}/10^{17})}{0.43} = 25\times\frac{2}{0.43} \approx 116 \text{ meV}$$
$E_F$ は $E_c$ より 116 meV 下（禁制帯上部）

iii）B（アクセプタ）を $5\times10^{17}$ cm$^{-3}$ まで増加：

  • B = 0 ～ $10^{17}$ cm$^{-3}$：補償により実効ドナー減少，$E_F$ が $E_c$ から禁制帯中央に向かって低下
  • B $= 10^{17}$ cm$^{-3}$：完全補償，真性半導体的，$E_F$ ≒ 禁制帯中央
  • B $> 10^{17}$ cm$^{-3}$：p形に転換，$E_F$ が $E_v$ 側に移動

設問2-a）全分子が $x$ 方向に整列（電界なし/あり）

i）分極：$P_x = Nm$（単位体積あたり $N$ 個×双極子モーメント $m$）

ii）電界 $E$ を加えても，分子は既に完全に整列しているので $P_x = Nm$（変化なし）

設問2-b）無秩序配向の場合

i）角度 $\theta$ の分子の $x$ 成分：$m_x = m\cos\theta$

ii）三次元等方的ランダム配向の立体角比：

$$\text{割合} = \frac{\int_0^{\theta_A}2\pi\sin\theta\,d\theta}{4\pi} = \frac{1-\cos\theta_A}{2}$$

iii）等方配向では $\langle\cos\theta\rangle=0$ → $P_x = 0$

設問2-c）電界印加・ボルツマン分布

i）$\theta = \pi/2$ を基準とするポテンシャルエネルギー：$W(\theta) = -mE\cos\theta$

ii）双極子モーメント $x$ 成分の平均値（$f(\theta) = A\exp(mE\cos\theta/kT)$）：

$$\langle m\cos\theta\rangle = \frac{\int_0^\pi m\cos\theta\cdot f(\theta)\sin\theta\,d\theta}{\int_0^\pi f(\theta)\sin\theta\,d\theta}$$

iii）$W(\theta) \ll kT$ の近似（$f(\theta)\approx A(1+mE\cos\theta/kT)$）：

分子 $= m\int_0^\pi\cos\theta\sin\theta\,d\theta + \frac{m^2E}{kT}\int_0^\pi\cos^2\theta\sin\theta\,d\theta = 0 + \frac{m^2E}{kT}\cdot\frac{2}{3} = \frac{2m^2E}{3kT}$
分母 $= \int_0^\pi\sin\theta\,d\theta = 2$
$$P_x = N\langle m\cos\theta\rangle = \frac{Nm^2 E}{3kT} \quad \text{（ランジュバン分極：Curie則）}$$

分子の状態：熱揺らぎが支配的（$kT \gg mE$）で，ほぼランダム配向。

iv）$P_x \propto N$（あ：比例する）…単純な式では。

実際には $N$ が増大するにつれて隣接分子間の電場（局所場補正）が無視できなくなり，外部電場だけでなく分極による場も各分子に作用する。これにより局所場が増強され，有効電場が外部電場より大きくなる（クラウジウス－モソッティ補正）。十分大きい $N$ では正のフィードバックが生じ，自発分極（強誘電相転移）が起こりうる。→ 比例しなくなる。

問題4：電子デバイス・回路システム

設問1-a）同期式カウンタの特徴

同期式カウンタ：全フリップフロップが共通クロックで同時にトリガされる回路。

非同期式との比較：

項目同期式非同期式（リップルカウンタ）
クロック全FF共通前段の出力を次段のCLKに使用
速度高速（並列遷移）低速（段数分の遅延が累積）
グリッチ少ない多い（過渡的な誤り状態が生じる）
回路規模組合せ論理が必要で大きい小さい

設問1-b）A1, A2 の論理式と NOR 実装

真理値表から次状態論理式を導出する：
  • $E=0$：保持（$D_1 = Q_1$，$D_0 = Q_0$）
  • $E=1, U=1$（アップ）：$00\to01\to10\to11\to00$
  • $E=1, U=0$（ダウン）：$00\to11\to10\to01\to00$
整理すると： $$D_0 = \bar{E}\cdot Q_0 + E\cdot\bar{Q_0} = E \oplus Q_0$$ $$D_1 = \bar{E}\cdot Q_1 + E\cdot U\cdot(Q_0\oplus Q_1) + E\cdot\bar{U}\cdot(\bar{Q_0}\oplus Q_1)$$
A1（D0 の XOR 実装）：$D_0 = E\oplus Q_0$
NOR のみでの XOR 実現（XOR = 4 NAND = 4 NOR + 変換）：
$A = \overline{E+Q_0}$（NOR），$B = \overline{E+A} = \overline{E+\overline{E+Q_0}}$，$C = \overline{Q_0+A}$，$D_0 = \overline{B+C}$
（4個の 2入力 NOR で XOR が構成可能）
XOR を NOR 4 つで構成：$E\oplus Q_0 = \overline{\overline{E+\overline{E+Q_0}}+\overline{Q_0+\overline{E+Q_0}}}$

設問1-c）A3 の CMOS NOR + インバータ回路

3入力 NOR（入力 $a, b, c$）の CMOS 構成：

  • pMOS（プルアップ）：$a, b, c$ を各ゲートとする pMOS 3個を直列接続（$V_{DD}$ ～出力）
  • nMOS（プルダウン）：$a, b, c$ を各ゲートとする nMOS 3個を並列接続（出力～GND）
  • 出力後にインバータ接続（pMOS 1個 + nMOS 1個）
CMOS NOR では pMOS が直列・nMOS が並列（NAND と逆）。入力数が増えると pMOS の直列接続数が増え，動作速度が低下する。

設問1-d）A1 部分の伝搬遅延（Q0 の遷移）

A1 で $D_0 = E\oplus Q_0$ を NOR 4個で実装した場合のクリティカルパス：
NOR 1段目 → NOR 2段目 → NOR 3段目 → NOR 4段目 = 4段
各 NOR の遅延 = 5 ns
伝搬遅延 $= 4\times5 = \boxed{20 \text{ [ns]}}$
XOR を NOR 4段（直列最大 4段）で実装した場合。インバータのみで閉じる経路は 5 ns 短縮可能な場合もあるが，NOR とインバータが同じ 5 ns なら同様。

設問2-a）MOS ダイオードの C-V 特性

i） $V<0$ で $C$ が高い（蓄積），$V>V_T$ で $C$ が低い（反転） → 半導体は p形（正バイアスで空乏・反転が生じる）

ii）点 T（閾値電圧 $V_T$）のバンド図：

  • Si/SiO₂ 界面で強反転（$E_i$ が $E_F$ を下回る）
  • バンド曲がり量 $= 2\phi_F$（$\phi_F = (kT/e)\ln(N_A/n_i)$）
  • バイアス $V_T$ は金属側 Fermi 準位と SiO₂ 内の電位降下＋空乏層電位降下の合計として現れる

iii）領域Ⅰ（蓄積，$V\ll0$）：正孔が酸化膜直下に蓄積 → 半導体が導体的，容量 $= C_{ox}$（一定）。
領域Ⅱ（強反転，$V\gg V_T$，高周波）：少数キャリア（電子）が高周波に追従できず，空乏層幅が最大値 $W_{d,\max}$ に固定 → $C = C_{ox}$ と $C_{dep,\min}$ の直列（一定）。

iv）不純物濃度 $N_A$ 増大 → $W_{d,\max} = \sqrt{4\varepsilon_s\phi_F/(eN_A)}$ が減少 → $C_{dep,\min}$ 増大 → $C_{\min}$ が上昇。また $V_T$ が増大する。

v）低周波 → 反転層電荷（少数キャリア）が交流に追従可能 → 強反転領域でも $C = C_{ox}$（最大値）に戻る。領域Ⅱの $C$ が立ち上がる形に変化。

設問2-b）MOS トランジスタの電流特性

図4.6より（端子1=ソース接地，端子2=ゲート，端子3=ドレイン，端子4=バルク）：
$V_T \approx 1$ V（$V_{DS}=1$ mV で $V_{GS}=1$ V からオン）
